# 비전 트랜스포머: 이미지에 트랜스포머 적용

## 소개

비전 [트랜스포머](/llm-kor.html)(ViT)는 NLP의 트랜스포머 아키텍처를 [컴퓨터 비전](/cv-kor.html)에 적용했으며 이미지 분류 및 관련 작업에서 놀라운 결과를 달성했습니다. 공간 구조에 대한 귀납적 편향을 가진 CNN과 달리 ViT는 순수하게 데이터에서 관계를 학습하여 시각적 추론에 대한 다른 관점을 제공합니다.

## 패치 임베딩

ViT는 이미지를 일반적으로 16×16 또는 32×32 픽셀의 겹치지 않는 패치로 나누는 것부터 시작합니다. 각 패치는 평탄화되어 고정 차원 벡터로 선형 임베딩됩니다. 이는 패치 임베딩 수열을 생성하여 [트랜스포머](/llm-kor.html) 처리를 가능하게 합니다.

**핵심 통찰:** 패치는 NLP의 단어와 유사하게 토큰이 됩니다. [트랜스포머](/llm-kor.html)는 명시적 공간 구조 지식 없이 이 수열을 처리합니다.

**위치 인코딩:** 패치 위치 정보는 학습 가능한 위치 임베딩을 통해 추가되어 모델이 공간 관계를 이해할 수 있습니다.

## ViT 아키텍처

표준 ViT는 [트랜스포머](/llm-kor.html) 설계를 따릅니다:
1. **입력:** 이미지 패치 + 특수 [CLS] 토큰
2. **임베딩:** 선형 프로젝션 + [위치 인코딩](/positional-encoding-kor.html)
3. **트랜스포머 인코더:** 다중 헤드 자기 주의와 피드포워드 네트워크의 여러 계층
4. **분류:** [CLS] 토큰 출력에 적용된 MLP 헤드

CNN이 계층적으로 다운샘플링하는 것과 달리 ViT는 해상도를 유지하여 모든 패치를 동일한 규모로 처리합니다.

## DeiT (데이터 효율적 이미지 트랜스포머)

ViT는 초기에 강력한 성능을 위해 대규모 데이터셋(JFT-300M)이 필요했습니다. DeiT는 다음을 통해 데이터 효율성을 개선했습니다:

- **지식 증류:** [CNN](/cnn-deep-dive-kor.html) 교사로부터 학습하여 데이터 필요성 감소
- **정규화 기법:** 확률적 깊이, 레이블 스무딩, 증강 전략
- **경쟁력 있는 결과 달성:** ImageNet 전용 학습으로 ViT가 실용적

## Swin 트랜스포머

이동된 윈도우 주의를 통해 ViT에 계층적 구조를 도입합니다:

- **계층적 설계:** CNN처럼 점진적 다운샘플링으로 특징 피라미드 생성
- **이동된 윈도우:** 로컬 윈도우 내에서 주의 계산, 윈도우 간 연결을 위해 이동
- **효율성:** 전역 주의 대비 계산 복잡도 대폭 감소
- **다재다능성:** 탐지, 분할, 분류에 효과적

## 하이브리드 접근법

[트랜스포머](/llm-kor.html)의 강점과 [CNN](/cnn-deep-dive-kor.html) 귀납적 편향을 결합합니다:

- **CNN → ViT:** 패치 생성을 위해 CNN을 백본으로 사용
- **ViT → CNN:** 초기 [트랜스포머](/llm-kor.html) 계층 후 합성곱 헤드
- **혼합:** 주의 및 합성곱 계층이 교대로 배열

하이브리드는 순수 접근법보다 더 적은 데이터로 더 나은 성능을 자주 달성합니다.

## ViT vs CNN 논쟁

**ViT 장점:**
- 데이터와 모델 크기에 따른 더 나은 확장성
- 명시적 아키텍처 제약 없이 전역 맥락 학습
- 더 해석 가능한 주의 패턴
- 다양한 입력 유형(이미지, 비디오, 3D)에 유연함

**CNN 장점:**
- 우수한 효율성 (적은 매개변수, 빠른 추론)
- 공간 국소성에 대한 더 강한 귀납적 편향
- 제한된 데이터 시 더 나은 성능
- 확립된 최적화 및 배포 인프라

**수렴:** 현대 관행은 둘 다 채택합니다—ViT는 대규모 설정에서 우위; CNN은 효율성 제약 시나리오에서 최적입니다. 하이브리드는 종종 최고의 트레이드오프를 제공합니다.

## 최근 발전

- **효율적 ViT 변형:** 증류, 가지치기, [양자화](/quantization-kor.html)로 모델 크기 감소
- **마스크된 이미지 모델링:** [BERT](/bert-kor.html) 유사 자기 지도 사전 학습
- **비디오 ViT:** 비디오 이해를 위해 시간 차원으로 ViT 확장
- **다중모드 ViT:** CLIP 유사 이미지-텍스트 쌍 학습 모델

## 실무 고려사항

ViT를 선택하세요:
- 대규모 학습 데이터셋 사용 가능
- 계산 리소스([GPU](/gpu-hardware-kor.html)/TPU) 접근 가능
- 최첨단 정확도 필요
- 모델 크기/추론 속도 덜 중요

CNN을 선택하세요:
- 제한된 학습 데이터
- 엣지 배포 또는 실시간 요구사항
- 계산 효율성 필수
- 확립된 도구 및 라이브러리 선호

## 결론

비전 [트랜스포머](/llm-kor.html)는 CNN에서의 패러다임 전환을 나타내며 명시적 귀납적 편향이 시각 작업에 불필요함을 보여줍니다. CNN은 효율성을 위해 관련성을 유지하지만 ViT의 확장성과 유연성으로 인해 대규모 비전 모델의 지배적 접근법입니다. 이 분야는 ViT 적용성을 확대하기 위해 계속 하이브리드 아키텍처와 효율적 변형을 탐색합니다.

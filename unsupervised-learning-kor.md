# 비지도학습: 레이블 없이 패턴 발견하기

비지도학습은 미리 정의된 레이블 없이 데이터를 탐색하며, 숨겨진 구조, 패턴, 관계를 발견합니다. 정답으로 모델을 안내하는 지도학습과 달리, 비지도학습은 데이터 자체가 이야기하게 합니다. 새로운 데이터셋을 이해하는 첫 걸음이 됩니다.

## 비지도학습이란?

비지도학습은 레이블이 없는 데이터로 작동합니다. 입력 X는 있지만 대응하는 출력 y는 없습니다. 알고리즘은 내재된 패턴, 그룹화, 표현을 발견합니다. 본질적으로 탐색적이며 인간이 예상하지 못한 데이터 특성을 드러냅니다. 공통 작업은 비슷한 예시들을 클러스터링하고, 차원을 축소하고, 이상을 찾는 것입니다.

### 핵심 동기
레이블을 얻기는 비용이 크고 시간이 많이 걸립니다. 비지도학습은 쉽게 구할 수 있는 대량의 레이블 없는 데이터에서 가치를 추출합니다. 또한 후속 지도학습 작업의 전처리 역할을 합니다.

## 클러스터링: 유사한 데이터 그룹화

클러스터링은 데이터를 그룹으로 분할하여 비슷한 예시들이 함께 모입니다.

### K-평균 클러스터링
K-평균은 데이터를 k개 클러스터로 나누어 클러스터 내 분산을 최소화합니다. 반복적으로 각 점을 가장 가까운 중심점으로 할당하고, 중심점을 재계산합니다. 단순하고 확장성이 좋지만 구 모양 클러스터에만 잘 작동하며 볼록하지 않은 모양이나 다양한 크기의 클러스터에는 어려움을 겪습니다.

**과정:** k개의 중심점을 무작위로 초기화, 각 점을 가장 가까운 중심점에 할당, 중심점을 클러스터 점들의 평균으로 업데이트, 수렴할 때까지 반복.

**한계:** 사전에 k를 지정해야 합니다. 팔꿈치 방법이나 실루엣 분석이 k 선택을 돕습니다.

### DBSCAN (밀도 기반 공간 클러스터링)
DBSCAN은 밀도를 기반으로 점들을 그룹화하며, 자연스럽게 임의 모양과 크기의 클러스터를 처리합니다. 핵심 점(충분한 이웃을 가진), 경계 점, 노이즈를 식별합니다. K-평균과 달리 사전에 클러스터 개수를 지정할 필요가 없습니다.

**장점:** 클러스터 구조를 자동으로 발견하고 이상을 노이즈로 식별합니다.

**도전:** 엡실론(이웃 반경)과 최소 점수 파라미터에 민감합니다.

### 계층적 클러스터링
계층적 클러스터링은 중첩된 클러스터의 트리(수형도)를 만듭니다. 두 가지 접근 방식:

**응집형(상향식):** 각 점을 자신의 클러스터로 시작, 반복적으로 가장 가까운 클러스터 병합. 계산 비용은 더 크지만 해석이 더 쉬운 경우가 많습니다.

**분열형(하향식):** 모든 데이터를 하나의 클러스터로 시작, 재귀적으로 클러스터 분할.

**장점:** 클러스터 개수를 지정할 필요가 없고, 수형도는 직관적 시각화를 제공합니다.

**도전:** 연결 방법(단일, 완전, 평균, Ward)에 민감하고 큰 데이터셋에는 계산 비용이 많습니다.

## 차원 축소

실제 데이터는 종종 높은 차원(수천 개의 특성)입니다. 차원 축소는 중요한 정보를 보존하면서 데이터를 압축하여 시각화와 계산 효율을 개선합니다.

### 주성분 분석 (PCA)
PCA는 주성분—최대 분산을 포착하는 직교 방향—을 식별합니다. 처음 몇 개의 주성분으로 데이터를 투영하여 대부분의 정보를 보존하면서 차원을 축소합니다.

**이점:** 해석 가능하고 비지도이며 효율적입니다. 시각화와 전처리에 널리 사용됩니다.

**한계:** 선형 관계를 가정하고, 성분이 직교한다고 가정합니다.

### t-SNE (t-분포 확률적 이웃 임베딩)
t-SNE는 낮은 차원에서 로컬 데이터 구조를 보존하여 시각화에 우수합니다. 고차원에서 쌍별 유사성을 계산하고 이를 낮은 차원으로 매핑하여 클러스터를 시각적으로 명확하게 합니다.

**강점:** 복잡한 비선형 구조 드러냄, 탐색 분석에 최고.

**약점:** 계산 비용이 크고, 비결정적이며, 매우 큰 데이터셋에 확장 성이 좋지 않습니다.

### 오토인코더
병목 구조를 가진 신경망이 압축된 표현을 학습합니다. 인코더가 입력을 잠재 벡터로 압축하고, 디코더가 이 벡터에서 복원합니다. 병목은 네트워크가 필수 특성을 포착하도록 강제합니다.

**유연성:** 비선형 변환 학습 가능, 이미지, 텍스트 등에 적용 가능.

## 밀도 추정

밀도 추정은 데이터의 확률 분포를 모델링합니다.

### 가우시안 혼합 모델 (GMM)
GMM은 가우시안 분포의 혼합에서 데이터가 생성된다고 가정합니다. 각 성분은 자신의 평균과 공분산을 가진 잠재적 클러스터입니다. EM(기대 최대화) 알고리즘이 성분 파라미터를 학습합니다.

**사용:** 소프트 클러스터링(확률적 할당), 밀도 추정.

**장점:** 불확실성 추정 제공, K-평균보다 유연함.

### 커널 밀도 추정 (KDE)
KDE는 각 데이터 점에 커널(부드러운 함수)을 놓고 합산하여 밀도를 추정합니다. 비모수적이며 특정 분포 형태를 가정하지 않습니다.

**적용:** 확률 분포 추정, 이상 탐지.

## 연관 규칙과 장바구니 분석

연관 규칙은 거래 데이터에서 항목 간의 관계를 발견합니다. 고전적 예: 매장 거래를 분석하여 함께 자주 구매되는 제품 찾기.

**지표:**
- **지지도:** 모든 거래에 대한 항목집합의 빈도
- **확신도:** 항목 A를 샀을 때 항목 B를 살 확률
- **향상도:** 관찰된 확신도와 예상 확신도의 비율

**알고리즘:** Apriori 알고리즘이 빈번한 항목집합을 효율적으로 찾고 규칙 생성.

**적용:** 추천 시스템, 매장 배치 최적화, 교차 판매 전략.

## 이상 탐지

이상 탐지는 정상 행동과 다른 드물고 비정상적인 의심스러운 인스턴스를 식별합니다. 사기 탐지, 시스템 모니터링, 품질 관리에 중요합니다.

### 고립 포레스트
고립 포레스트는 무작위 특성 선택과 분할로 이상을 고립시킵니다. 이상은 정상 점보다 빠르게 고립됩니다(짧은 경로), 이를 식별 가능하게 합니다.

**장점:** 확장 가능, 데이터 분포 가정 안 함, 고차원 데이터를 잘 처리.

### 로컬 이상 인자 (LOF)
LOF는 점의 밀도와 이웃의 밀도를 비교합니다. 이웃 대비 저밀도 지역의 점들은 이상입니다.

**강점:** 로컬 이상 효과적 감지, 비균등 분포 데이터에 작동.

### 원클래스 SVM
원클래스 SVM은 정상 데이터 주위 경계를 학습합니다. 경계 밖의 점들은 이상입니다.

## 자기조직화 지도 (SOM)

자기조직화 지도는 입력 공간을 낮은 차원 지도로 표현하도록 학습하는 신경망으로, 위상 특성을 보존합니다. 각 뉴런은 지도의 위치에 대응되고, 인접 뉴런은 유사한 가중치 벡터를 가집니다.

**과정:** 지도 뉴런을 무작위로 초기화, 반복적으로 데이터 제시, 입력을 기반으로 우승 뉴런과 이웃 업데이트, 시간에 따라 이웃 축소.

**적용:** 시각화, 클러스터링, 특성 추출.

**강점:** 위상 보존, 고차원 데이터 관계 시각화에 유용.

## 산업 응용

**마케팅:** 고객 세분화는 유사한 구매 행동을 가진 그룹을 드러내 타겟 캠페인 가능.

**생물학:** 유전자 발현 클러스터링은 질병 소형을 드러내고, 단백질 구조 분석이 패턴 발견.

**이미지 처리:** 압축, 엣지 탐지, 후속 작업의 특성 추출.

**추천 시스템:** 협업 필터링으로 유사 사용자나 항목 발견.

**사기 탐지:** 정상 패턴에서 벗어난 의심스러운 거래 식별.

**천문학:** 은하 분류, 천체 객체 패턴 발견.

## 비지도학습의 도전

**정답 부재:** 레이블이 없어 결과를 객관적으로 검증하기 어렵습니다. 실루엣 점수, Davies-Bouldin 지수, Calinski-Harabasz 지수가 내부 검증을 제공하지만 외부 레이블이 이상적입니다.

**파라미터 민감성:** 많은 알고리즘이 명확한 지침 없이 파라미터(K-평균의 k, DBSCAN의 엡실론) 선택을 필요로 합니다.

**확장성:** 계층적 클러스터링 같은 일부 알고리즘은 큰 데이터셋에서 계산상 불가능합니다.

**해석성:** 발견된 패턴이 인간 직관이나 사업 필요와 맞지 않을 수 있습니다.

비지도학습은 탐색, 특성 공학, 전처리에 매우 귀중합니다. 지도학습과 결합하면 실제 문제를 위한 완전한 ML 툴킷을 형성합니다.

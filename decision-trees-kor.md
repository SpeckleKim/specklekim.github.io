# 의사결정나무와 랜덤 포레스트

## 소개

의사결정나무는 특성 공간을 재귀적으로 분할하여 예측하는 기본적인 [지도 학습](/supervised-learning-kor.html) 알고리즘입니다. 피처에 대한 간단한 예/아니오 질문을 통해 트리 형태의 모델을 만듭니다. 랜덤 포레스트는 여러 트리를 결합하여 과적합을 줄이고 일반화 성능을 향상시킵니다.

## 의사결정나무 구축

### 성장 과정

의사결정나무는 위에서 아래로 성장합니다(하향식 귀납법). 각 노드에서 데이터를 가장 잘 분할하는 피처와 임계값을 선택합니다. 알고리즘은 중단 기준에 도달할 때까지 이 과정을 재귀적으로 반복합니다.

**핵심 요소:**
- **루트 노드**: 전체 데이터에 대한 초기 분할
- **내부 노드**: 피처와 임계값 조합
- **리프 노드**: 최종 예측값(클래스 레이블 또는 회귀값)
- **브랜치**: 의사결정 결과 표현

### 정보 이득(Information Gain)

정보 이득은 분할이 불확실성을 얼마나 감소시키는지 측정합니다. 부모 노드 [엔트로피](/information-theory-kor.html)와 자식 노드 엔트로피의 가중 평균 차이로 계산됩니다.

**공식:**
```
정보 이득 = H(부모) - Σ(|자식|/|부모|) × H(자식)
```

여기서 H는 [엔트로피](/information-theory-kor.html): `H = -Σ p_i × log₂(p_i)`

정보 이득이 높은 분할이 선호됩니다. 더 순수한 자식 노드를 만들기 때문입니다.

### 지니 불순도(Gini Impurity)

지니 불순도는 [엔트로피](/information-theory-kor.html)의 대안입니다. 클래스 분포에 따라 무작위로 레이블을 할당할 때 잘못 분류할 확률을 측정합니다.

**공식:**
```
Gini = 1 - Σ p_i²
```

여기서 p_i는 클래스 i의 비율입니다. 지니 불순도는 0(순수 노드)에서 0.5(이진 분류에서 최대 불순도)까지입니다.

**엔트로피보다의 장점:**
- 계산 속도 빠름(로그 함수 불필요)
- 유사한 결과를 더 나은 성능으로 생성
- 균형잡힌 분할에 약간 편향

## 가지치기(Pruning)

가지치기를 하지 않은 트리는 과적합 경향이 있으며 패턴 외 노이즈도 포착합니다. 가지치기는 검증 성능을 개선하지 않는 브랜치를 제거합니다.

### 가지치기 방법

**조기 정지(Pre-pruning):**
- 기준 충족 시 성장 중단(최소 샘플 수, 최대 깊이)
- 간단하지만 과소 적합 위험

**사후 가지치기(Post-pruning):**
- 완전한 트리를 성장시킨 후 재귀적으로 브랜치 제거
- 계산 비용이 더 크지만 일반적으로 더 효과적
- 공통 접근법: 검증 세트를 사용한 비용-복잡도 가지치기

## 랜덤 포레스트

랜덤 포레스트는 여러 의사결정나무를 결합하여 강력한 [앙상블](/ensemble-methods-kor.html) 분류기를 만듭니다. 다양성과 평균화를 통해 뛰어난 정확도를 달성합니다.

### 배깅(부트스트랩 집계)

랜덤 포레스트는 부트스트랩 샘플링을 사용합니다. 각 트리에 대해 복원 추출로 훈련 데이터를 무작위로 샘플링하여 서로 다른 부분집합을 만듭니다. 각 트리는 자신의 부트스트랩 샘플에서 훈련되어 최종 예측의 분산을 증가시킵니다.

**과정:**
1. B개의 부트스트랩 샘플 생성
2. 각 샘플에서 완전한 의사결정나무 성장
3. 예측 평균화(분류: 다수결 투표, 회귀: 평균)

### 피처 무작위성

각 노드에서 모든 피처를 검색하는 대신, 랜덤 포레스트는 전체 p개 피처 중에서 무작위로 m개 피처를 선택합니다. 이는 트리 간 다양성을 더욱 증가시키고 예측을 비상관화합니다.

**일반적인 선택:**
- 분류: m = √p
- 회귀: m = p/3

## 피처 중요도

랜덤 포레스트는 모든 트리에서 각 피처가 불순도를 감소시키는 정도를 측정하여 피처 중요도를 계산합니다.

**중요도 계산:**
```
중요도(피처) = Σ(불순도 감소) / 총 감소량
```

자주 큰 정보 이득을 제공하는 피처가 더 높게 순위가 매겨집니다. 이는 어떤 변수가 예측을 주도하는지 파악하는 데 도움이 됩니다.

## 장점

- **비선형성 처리**: 선형 가정 불필요
- **피처 상호작용**: 자동으로 상호작용 포착
- **강건성**: 배깅이 분산과 과적합 감소
- **피처 중요도**: 내장된 피처 선택
- **속도**: 트리 간 병렬 처리 가능
- **혼합 피처**: 범주형과 수치형 데이터 모두 처리
- **결측값**: 결측 데이터 처리 가능

## 단점

- **해석 가능성**: 개별 트리 예측 설명 어려움
- **계산 비용**: 많은 트리 훈련이 단일 트리보다 느림
- **메모리**: 여러 모델이 더 많은 저장 공간 필요
- **편향**: 기본 학습기가 편향되면 개선 불가능
- **클래스 불균형**: 불균형 데이터셋에서 어려움
- **외삽**: 훈련 데이터 범위 밖 예측 불가능(회귀)

## 실무 고려사항

**하이퍼파라미터:**
- 트리 수(B): 더 많은 트리가 안정성을 개선하지만 수확 체감
- 최대 트리 깊이: 복잡도 제어
- 리프당 최소 샘플 수: 과적합 방지
- 피처 무작위성 파라미터(m)

**사용 시기:**
- 테이블 데이터의 분류 및 회귀
- 해석 가능성이 성능보다 덜 중요할 때
- 병렬화가 도움이 되는 대규모 데이터셋
- 혼합 데이터 유형

**피해야 할 때:**
- 확률적 신뢰 구간이 필요할 때
- 모델 해석 가능성이 중요할 때
- 매우 고차원 데이터(차원의 저주)

## 요약

의사결정나무는 데이터 이해를 위한 직관적이고 해석 가능한 기초를 제공합니다. 랜덤 포레스트는 [앙상블 학습](/ensemble-methods-kor.html), 비상관화, 부트스트랩 샘플링을 통해 성능을 크게 향상시킵니다. 이들은 여전히 산업 응용 분야 전반에 걸쳐 테이블 데이터에 가장 실용적이고 효과적인 알고리즘 중 하나입니다.

# 셀프 어텐션 심층 분석

셀프 어텐션은 현대의 트랜스포머 기반 언어 모델을 구동하는 핵심 메커니즘입니다. 외부 문맥에 의존하는 기존 어텐션 메커니즘과 달리, 셀프 어텐션은 토큰들이 시퀀스 내 다른 모든 토큰에 주목하도록 허용하여 장거리 의존성을 효율적으로 포착합니다.

## 쿼리, 키, 밸류 프레임워크

셀프 어텐션은 입력의 세 가지 선형 투영(linear projection)을 기반으로 작동합니다:

- **쿼리(Query, Q)**: "내가 무엇을 찾고 있는가?"를 나타냅니다
- **키(Key, K)**: "내가 어떤 정보를 가지고 있는가?"를 나타냅니다
- **밸류(Value, V)**: "내가 어떤 정보를 전달할 것인가?"를 나타냅니다

입력 시퀀스 X (크기: batch_size, seq_len, d_model)에서:
```
Q = X * W_Q
K = X * W_K
V = X * W_V
```

여기서 W_Q, W_K, W_V는 학습 가능한 가중치 행렬입니다.

## 닷 프로덕트 어텐션 메커니즘

어텐션 메커니즘은 쿼리-키 유사도를 기반으로 밸류의 가중 합을 계산합니다:

```
Attention(Q, K, V) = softmax(Q * K^T / sqrt(d_k)) * V
```

이 수식은 세 가지 핵심 요소로 구성됩니다:

1. **Q * K^T**: 각 쿼리와 모든 키 간 쌍별 유사도 점수를 계산
2. **sqrt(d_k) 스케일링**: 어텐션 점수가 극단적으로 치우치지 않도록 방지
3. **소프트맥스**: 점수를 확률 분포로 변환
4. **밸류와의 행렬 곱셈**: 가중 합산 수행

## sqrt(d_k)로 나누는 이유

스케일링 계수 sqrt(d_k)는 학습 안정성을 위해 매우 중요합니다. 이 없다면 닷 프로덕트의 분산이 d_k와 함께 증가하여 소프트맥스가 거의 원-핫(one-hot) 분포를 생성합니다. 역전파 시 매우 작은 기울기를 초래합니다.

예를 들어, d_k = 64일 때 닷 프로덕트는 -∞에서 +∞까지 범위를 가질 수 있습니다. sqrt(64) = 8로 나누면 이 값들을 평균 0, 분산 1로 정규화하여 안정적인 기울기 흐름을 보장합니다.

## 소프트맥스와 정보 흐름

소프트맥스 연산은 원본 어텐션 점수를 확률 분포로 변환합니다:

```
softmax(x_i) = exp(x_i) / sum_j(exp(x_j))
```

각 출력은 어텐션 확률로 가중치 지어진 밸류들의 볼록 조합입니다. 이는 다음을 보장합니다:
- 확률 가중치의 합이 1
- 역전파를 통한 부드러운 기울기 흐름
- 해석 가능성: 모델이 어떤 토큰에 주목하는지 시각화 가능

## 계산 복잡도

셀프 어텐션의 계산 복잡도는 시퀀스 길이 n에 대해 **O(n²)** 입니다:

- Q, K, V 투영 계산: O(n * d_model²)
- 행렬 곱셈 Q * K^T: O(n² * d_k)
- 소프트맥스 및 V 곱셈: O(n² * d_v)

**n²** 항이 지배적이므로 매우 긴 시퀀스에서 계산이 비쌉니다. 이것이 희소 어텐션(sparse attention), 선형 어텐션(linear attention), KV 캐시가 실무에서 중요한 이유입니다.

## 장거리 의존성 포착

셀프 어텐션이 장거리 의존성을 자연스럽게 포착하는 이유:

1. **직접 연결성**: 모든 토큰이 한 단계에서 다른 모든 토큰에 직접 주목 가능
2. **적응형 가중치**: 어텐션 가중치는 토큰 쌍별로 학습되어 유연한 패턴 매칭 가능
3. **기울기 흐름**: 기울기가 먼 토큰 간 직접 흐를 수 있어 기울기 소실 문제 회피
4. **멀티헤드 중복성**: 여러 어텐션 헤드가 다양한 의존성 유형 학습

## 실전 예시

문장 "The cat sat on the mat"를 생각해봅시다. 위치 2(단어 "cat")에서 셀프 어텐션은:

- "The"에 높은 가중치 (관사-명사 관계)
- "sat"에 주목 (주어-동사 관계)
- "mat"에 주목 (의미론적 관계)
- "on"에 주목 (합성적 이해)

셀프 어텐션은 고정된 윈도우나 순환 연결에 의존하지 않고 작업별 어텐션 패턴을 학습합니다.

## 수학적 직관

어텐션 메커니즘은 다음으로 해석할 수 있습니다:

1. **검색(Retrieval)**: Q는 키-밸류 저장소로의 쿼리 역할
2. **매칭(Matching)**: 유사도 점수가 관련성 결정
3. **집계(Aggregation)**: 관련 밸류들이 결합됨
4. **출력**: 관련 정보를 요약하는 문맥 벡터

이 검색-집계 프레임워크는 시퀀스를 넘어 일반화되며 검색 증강 생성, 교차 어텐션, 메모리 아키텍처에 나타납니다.

## 제한사항 및 확장

순수 셀프 어텐션에는 여러 제한이 있습니다:

- **이차 복잡도**: 100k 이상 토큰 시퀀스에서 비실용적
- **동일 어텐션 용량**: 모든 토큰 쌍이 동일 계산 예산 할당
- **귀납적 편향 부재**: 위치 인코딩을 통해 위치 정보 학습 필요
- **밀집 어텐션**: 모든 토큰에 동등하게 주목 가능 (낭비적)

이러한 제한은 희소 어텐션 패턴, 상대 위치 인코딩, ALiBi, 회전 임베딩, 효율적 어텐션 변형으로 이어졌습니다.

## 결론

셀프 어텐션은 RNN의 순차적 병목현상이나 CNN의 제한된 수용 영역 없이 모델이 장거리 의존성을 직접 학습하도록 함으로써 NLP에 혁명을 일으켰습니다. 우아한 닷 프로덕트 공식과 소프트맥스 정규화의 결합은 현대 언어 모델을 구동하는 해석 가능하고 미분 가능한 어텐션 메커니즘을 만듭니다.

셀프 어텐션의 메커니즘을 이해하는 것은 트랜스포머로 작업하고, 모델 동작을 디버깅하며, 특수 응용 분야를 위한 효율적 변형을 설계하는 데 필수적입니다.

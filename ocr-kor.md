# 광학 문자 인식: OCR과 현대적 접근법

## 소개

광학 문자 인식(OCR)은 텍스트를 포함하는 이미지를 기계가 읽을 수 있는 텍스트로 변환합니다. 스캔된 문서에서 자연 이미지의 장면 텍스트까지 OCR은 디지털화와 접근성에 중요합니다. 현대 심층 학습 접근법은 전통적 방법을 획기적으로 능가합니다.

## 전통적 OCR

초기 OCR 시스템은 수작업으로 만든 특징과 규칙 기반 접근법을 사용했습니다:

- **전처리:** 이진화, 노이즈 제거, 기울기 보정
- **분할:** 개별 문자 또는 단어 고립
- **특징 추출:** 방향화된 기울기 히스토그램, 템플릿 매칭
- **분류:** 서포트 벡터 머신, 은닉 마르코프 모델
- **한계:** 낮은 일반화, 글꼴, 크기, 방향 변화에 민감

## 심층 학습 OCR

신경망은 데이터에서 특징을 자동으로 학습하여 OCR을 혁신했습니다.

**시퀀스-투-시퀀스 모델:**
- 인코더는 전체 이미지를 처리하고 디코더는 문자 수열 생성
- 주의 메커니즘이 디코딩 중 관련 이미지 영역에 초점
- 가변 길이 텍스트를 자연스럽게 처리
- 전통적 접근법보다 훨씬 높은 정확도

## CRNN (합성곱 순환 신경망)

장면 텍스트 인식의 표준 아키텍처:

1. **CNN 특징 추출:** 이미지를 처리하여 특징 추출
2. **RNN 수열 모델링:** LSTM/GRU 계층이 문자 의존성 모델링
3. **CTC 손실:** 연결주의 시간 분류가 예측과 텍스트 정렬
4. **장점:** 명시적 문자 분할 없이 작동, 종단간 학습
5. **한계:** 단일 라인 텍스트에서 더 나음; 복잡한 레이아웃에 어려움

## 장면 텍스트 탐지

인식 전 텍스트 영역 위치 파악:

- **임의 방향:** 텍스트는 어떤 각도든 나타날 수 있음
- **다중 규모:** 같은 이미지에 작은 텍스트와 큰 텍스트
- **복잡한 배경:** 자연 장면 복잡성

**방법:**
- **회귀 기반:** PSENet (점진적 규모 확장 네트워크)
- **분할 기반:** PixelLink, pan-Net이 텍스트 마스크 예측
- **하이브리드:** 단일 모델에서 탐지와 인식 결합

## TrOCR (트랜스포머 기반 OCR)

최신 트랜스포머 기반 아키텍처:

- **비전 트랜스포머 인코더:** 전체 이미지에서 시각적 특징 추출
- **트랜스포머 디코더:** 자기 회귀적으로 텍스트 토큰 생성
- **종단간:** 단일 모델이 텍스트 탐지와 인식 처리
- **다중언어:** 다양한 스크립트와 언어에 효과적
- **최첨단:** 표준 벤치마크에서 최고 정확도 달성

**CRNN 대비 장점:**
- 전역 주의를 통한 더 나은 맥락 이해
- 다양한 텍스트 레이아웃에 더 유연
- 다중언어 처리 개선
- 왜곡된 텍스트에 더 나은 성능

## 문서 AI

단순 OCR을 넘어선 더 넓은 범주:

- **레이아웃 분석:** 문서 구조 이해 (제목, 섹션, 표)
- **양식 이해:** 필드 식별 및 값 추출
- **표 인식:** 표 형식 데이터 추출
- **문서 분류:** 문서 유형 분류
- **키-값 추출:** 관련 정보 쌍 찾기

**응용:**
- 인보이스 처리
- 신분증 디지털화
- 의료 기록 추출
- 자동화된 문서 워크플로우

## 필기 인식

인쇄 텍스트와 다른 독특한 도전:

- **가변성:** 필기자 간 극단적 차이
- **필기체 연결:** 문자가 병합되어 모호함 생성
- **획 패턴:** 문자 식별 결정에 중요
- **역사 문서:** 품질 저하, 고풍스러운 스타일

**접근법:**
- 온라인 인식: 디지털 입력에서 시간 획 정보 사용
- 오프라인 인식: 정적 이미지에서 작동, 더 도전적
- 결합 모델: 텍스트와 비텍스트 요소 통합

## 실무 도전

**이미지 품질:** 흐림, 압축 아티팩트, 불량 조명이 정확도 감소

**텍스트 변형:** 곡선, 회전, 기울어진 텍스트는 강건한 처리 필요

**복잡한 레이아웃:** 여러 열, 혼합 글꼴, 겹친 텍스트는 정교한 분석 필요

**다중언어:** 다른 스크립트, 발음 구별 기호, 특수 문자가 인식 복잡

## 언어 모델의 OCR 응용

최근 접근법은 OCR과 언어 모델을 결합합니다:

- **수정:** 언어 모델이 OCR 오류 수정
- **맥락 인식:** 문서 의미 이해가 인식 개선
- **저자원 언어:** 언어 모델이 적은 학습 데이터로 더 나은 인식 가능

## 평가 지표

**문자 오류율 (CER):** 잘못 인식된 문자의 백분율

**단어 오류율 (WER):** 오류가 있는 단어의 백분율

**수열 오류율:** 문서 수준 정확도

## 결론

OCR은 규칙 기반 시스템에서 심층 신경망으로 진화하여 정확도를 획기적으로 개선했습니다. TrOCR 같은 현대 접근법은 최첨단 성능을 위해 트랜스포머를 활용합니다. 문서 AI는 OCR을 문자 인식 이상으로 전체 문서 이해로 확장합니다. 이 분야는 보편적이고 다중언어이며 강건한 텍스트 인식 시스템을 향해 계속 진화합니다.

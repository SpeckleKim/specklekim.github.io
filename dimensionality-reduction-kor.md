# 차원 축소: 고차원 데이터 다루기

## 차원의 저주

데이터셋의 특성 수가 증가할수록 머신러닝 모델은 차원의 저주로 알려진 점점 커지는 도전에 직면합니다. 이 현상은 여러 방식으로 나타납니다: 특성 공간의 부피는 지수적으로 증가하여 데이터 포인트가 점점 더 희소해지고; 포인트 간 거리는 점점 더 균등해져 거리 기반 메트릭의 판별력이 감소하며; 모델은 통계적 유의성을 유지하기 위해 지수적으로 더 많은 데이터를 필요로 합니다.

실제로 1,000개 샘플의 데이터셋은 100차원 공간에서 모델을 훈련하기에 부족합니다. 특성이 많을수록 성능이 더 좋다는 것은 아니며—종종 반대입니다. 훈련이 느려지고, 과적합이 증가하고, 시각화가 불가능합니다. 그러나 많은 실제 문제들은 자연스러운 고차원 표현을 가집니다: 이미지 픽셀, 텍스트 단어 임베딩, 유전체 서열, 센서 배열입니다.

차원 축소는 노이즈와 중복을 버리면서 필수 정보를 보존하는 저차원 표현을 발견함으로써 이 근본적 도전에 대응합니다. 순진한 특성 선택보다는, 정교한 축소 기법이 데이터의 기저 구조를 추출합니다.

## 주성분분석 (PCA)

PCA는 가장 기초적인 차원 축소 기법으로, 최대 분산의 직교 방향을 찾기 위해 선형 변환을 수행합니다.

### PCA의 작동 원리

PCA는 최적화 문제를 해결합니다: 데이터가 최대 분산을 갖는 새로운 좌표축(주성분)을 찾습니다. 수학적으로, 이는 공분산 행렬의 고유값 분해를 계산하는 것을 포함합니다:

**Σ = 1/n X^T X**

고유벡터는 주성분을 형성하며, 고유값(설명된 분산)에 의해 순서를 매깁니다. 첫 번째 주성분은 가장 많은 분산을 설명하고, 두 번째는 첫 번째에 직교하는 남은 분산을 가장 많이 설명하며, 이런 식으로 계속됩니다.

### PCA를 통한 차원 축소

d 차원에서 k 차원으로 축소하려면, 데이터를 가장 큰 고유값을 가진 k개의 고유벡터로 투영합니다:

**X_reduced = X × V_k**

여기서 V_k는 k개의 주성분을 포함합니다. 이 투영은 직교성 유지 제약 하에서 재구성 오류를 최소화합니다.

### 장점과 제약

**장점:**
- 계산상 효율적 (고유값 분해가 잘 최적화됨)
- 해석 가능: 각 주성분은 원래 특성의 선형 결합
- 각 성분이 설명하는 분산 제공
- 비지도학습: 레이블 불필요

**제약:**
- 선형 변환만—비선형 구조를 놓침
- 특성 스케일에 민감 (정규화 필요)
- 분산이 중요성을 나타낸다고 가정 (항상 참은 아님)
- 성분이 추상적 선형 결합이므로 해석 어려움

PCA는 선형 근사가 충분할 때 전처리와 시각화에 탁월합니다. 비선형 데이터 다양체의 경우 고급 기법이 필요합니다.

## 커널 PCA: 비선형 데이터로 확장

커널 PCA(KPCA)는 커널 함수로 암묵적으로 정의된 더 높은 차원의 특성 공간에서 PCA를 수행하여 비선형 차원 축소로 PCA를 확장합니다.

### 커널 트릭

X^T X를 직접 계산하는 대신, KPCA는 커널 행렬 K를 사용하여 작동하고, 여기서 K_ij = k(x_i, x_j)는 포인트 간 유사도를 측정합니다. 일반적인 커널은:
- **다항식**: k(x,y) = (x·y + c)^d
- **RBF (가우시안)**: k(x,y) = exp(-γ||x-y||²)
- **Sigmoid**: k(x,y) = tanh(αx·y + c)

커널 트릭에 의해, 고차원 공간의 PCA 계산은 암묵적으로 발생합니다—KPCA는 변환을 명시적으로 구성하지 않고도 특성 공간에서 주 방향을 찾습니다.

### 장점

KPCA는 선형 PCA가 놓치는 비선형 구조를 포착하여, 복잡한 데이터 분포에 강력합니다. 그러나 선형 PCA보다 계산상 비용이 더 많이 들고 해석성이 낮습니다. 커널과 모수의 선택이 결과에 크게 영향을 미칩니다.

## t-SNE: 복잡한 데이터 시각화

t-분포 확률적 근처 이웃 임베딩(t-SNE)은 고차원 데이터의 2D 또는 3D 시각화에 특별히 최적화된 비선형 차원 축소 기법입니다.

### 유사도 보존

t-SNE는 간단한 원칙에 작동합니다: 지역적 근처 구조를 보존합니다. 각 포인트에 대해 가우시안 커널을 사용하는 고차원 공간의 다른 모든 포인트에 대한 유사도를 계산합니다. 저차원 공간에서, 이는 Student-t 분포(t-분포)를 사용하여 유사도를 모델링합니다.

알고리즘은 고차원과 저차원 공간의 유사도 분포 간 쿨백-라이블러(KL) 발산을 최소화하여, 고차원 공간에서 근처인 포인트가 시각화에서 근처로 유지되는 동시에 큰 거리를 존중하도록 합니다.

### 주요 특성

- **시각화에 탁월**: 직관적이고 시각적으로 해석 가능한 2D/3D 임베딩 생성
- **국소 구조 캡처**: 근처 보존이 종종 의미 있는 군집을 드러냄
- **비결정론적**: 여러 실행이 다른 결과를 생성할 수 있음 (단, 국소적으로는 유사)
- **계산상 비용**: O(n²) 복잡도는 매우 큰 데이터셋에 비실용적
- **새로운 데이터에 부적합**: 재계산 없이 새로운 포인트를 효율적으로 임베딩할 수 없음

t-SNE는 신경망의 임베딩 시각화, 언어 모델에서 의미론적 구조 드러내기, 탐색적 데이터 분석에서 머신러닝의 상징적 존재가 되었습니다.

## UMAP: 확장 가능한 비선형 축소

균등 다양체 근사 및 투영(UMAP)은 여러 접근법의 강점을 결합하면서 t-SNE보다 계산상 효율적인 최신 차원 축소 기법입니다.

### 이론적 기초

UMAP는 고차원 데이터의 퍼지 위상 표현을 구성한 다음, 유사한 위상을 갖는 저차원 임베딩을 최적화합니다. 리만 기하학과 미분 위상에 근거하여 구조 보존에 대한 이론적 보장을 제공합니다.

주요 모수는:
- **n_neighbors**: 국소 구조를 고려하는 근처 포인트 수
- **min_dist**: 저차원의 포인트 간 최소 거리
- **metric**: 고차원 공간의 거리 메트릭

### t-SNE 대비 장점

- **확장성**: O(n log n) 복잡도로 대규모 데이터셋에 실용적
- **재현성**: 실행 간 일관된 결과 생성
- **전역 구조 보존**: t-SNE와 달리, 의미 있는 대규모 관계 유지
- **모수 해석성**: 모수가 더 명확한 의미를 가짐
- **속도**: 일반적으로 t-SNE보다 10-100배 빠름

UMAP은 빠르게 탐색적 데이터 분석과 시각화의 선호되는 도구가 되었으며, 데이터 확장성이 중요한 많은 응용에서 t-SNE를 대체했습니다.

## 차원 축소를 위한 자동인코더

PCA, t-SNE, UMAP이 특수화된 기법인 반면, 자동인코더는 데이터로부터 차원 축소를 자동으로 배우는 신경망 접근을 나타냅니다.

### 구조

자동인코더는 세 부분으로 이루어진 신경망입니다: 인코더, 병목, 디코더.

인코더는 고차원 입력 x를 저차원 잠재 표현 z로 변환합니다:
**z = f_encode(x)**

디코더는 z로부터 입력을 재구성합니다:
**x_reconstructed = f_decode(z)**

훈련은 재구성 오류를 최소화합니다:
**L = ||x - x_reconstructed||²**

병목 층은 정보 흐름을 제한하여, 네트워크가 압축된 표현을 배우도록 강제합니다. PCA와 달리, 자동인코더는 비선형 변환을 배웁니다.

### 변형과 장점

**변분 자동인코더(VAE)**: 확률 층을 추가하여 잠재 공간에 대한 분포를 학습합니다. 생성 모델링과 분리된 표현에 유용합니다.

**노이징 제거 자동인코더**: 손상된 입력으로 훈련하여 강건한 표현을 배웁니다. 데이터가 노이즈를 포함할 때 유용합니다.

**장점:**
- 비선형 변환을 자동으로 배움
- 특정 하류 작업을 위해 미세 조정 가능
- 고차원 데이터로 확장
- 다양한 아키텍처 제약을 포함할 수 있음

**단점:**
- 신중한 하이퍼파라미터 튜닝 필요
- PCA보다 해석성 낮음
- 훈련하기 계산상 더 비용 많이 듦

## 특성 선택 대 특성 추출

두 차원 축소 철학 간 중요한 구분이 있습니다:

**특성 선택**은 원래 특성의 부분집합을 선택하여 해석성을 유지합니다. 방법은:
- 필터 방법: 통계적 테스트로 특성 순위 매김
- 래퍼 방법: 모델 성능을 최대화하는 특성 선택
- 임베드된 방법: 모델 훈련 중 특성 선택

**특성 추출**은 원래 특성의 조합으로부터 새로운 특성을 만듭니다. 방법은:
- PCA, 커널 PCA: 선형/비선형 조합
- t-SNE, UMAP: 학습된 근처 임베딩
- 자동인코더: 신경망이 배운 조합

특성 선택은 해석성을 유지하지만 중요한 조합을 놓칠 수 있습니다. 특성 추출은 최적 조합을 찾지만 개별 특성의 해석성을 잃습니다. 선택은 예측을 설명하거나 성능을 최대화하는 것이 더 중요한지에 따라 달라집니다.

## 실무 응용

### 데이터 시각화와 탐색
신경망 임베딩 100차원을 2D/3D으로 시각화하면 의미론적 구조를 드러냅니다. t-SNE와 UMAP은 모델이 정보를 어떻게 구조화하는지 보여주는 탐색적 데이터 분석의 표준 도구가 되었습니다.

### 머신러닝을 위한 전처리
훈련 전 하류 모델을 위해 차원을 줄이면 계산상 효율성이 개선되고, 과적합이 감소하며, 때로는 일반화가 개선됩니다. PCA 전처리는 이미징과 신호 처리에서 특히 흔합니다.

### 시각화 및 해석을 위한 준비
복잡한 고차원 예측은 2D/3D 공간에서 시각화할 때 이해 가능해집니다. 특성 축소는 머신러닝 결정의 인간 이해를 가능하게 합니다.

### 노이즈 및 중복 제거
가장 정보로운 성분이나 특성을 선택하여, 차원 축소는 자동으로 노이즈를 필터링하고 중복 정보를 제거하여 모델 견고성을 개선합니다.

### 계산상 효율성
더 적은 차원은 더 빠른 훈련, 추론, 저장소를 의미합니다. 리소스 제약이 있는 실시간 응용을 위해 차원 축소는 필수적입니다.

## 차원 축소 기법 선택하기

어떤 기법도 모든 문제에 적합하지 않습니다. 다음을 고려하세요:

- **비선형성**: 선형 관계는 PCA, 비선형은 KPCA/t-SNE/UMAP
- **해석성 필요**: 특성 선택은 해석성을 보존
- **규모**: UMAP은 t-SNE보다 더 잘 확장; 자동인코더는 신중한 아키텍처로 확장
- **속도**: PCA가 가장 빠름; t-SNE가 가장 느림; UMAP이 균형 제공
- **하류 작업**: 자동인코더 잠재 공간은 특정 작업을 위해 최적화 가능
- **영역 지식**: 커널 선택이나 특성 선택 기준을 통해 통합

## 결론

차원 축소는 현대 머신러닝에 필수적이며, 고차원 데이터를 관리 가능하고 해석 가능하거나 시각화 가능한 형태로 변환합니다. 선형 PCA부터 t-SNE와 UMAP을 사용한 비선형 다양체 학습, 자동인코더를 통한 학습된 표현까지, 도구모음은 다양한 문제에 대한 해결책을 제공합니다. 이러한 기법, 강점, 제약을 이해하면 실무자가 복잡한 고차원 데이터에서 최대 가치를 추출하면서 계산상 효율성과 모델 성능을 유지할 수 있습니다.

데이터셋이 점점 더 고차원이고 복잡해지면서, 차원 축소를 숙달하는 것은 AI와 머신러닝의 핵심 기술로 남아 있습니다.

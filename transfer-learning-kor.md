# 전이학습

## 소개

전이학습(Transfer Learning)은 한 작업(원본 도메인)에 대해 학습된 모델을 다른 관련 작업(대상 도메인)에 적응시켜 재사용하는 머신러닝 패러다임입니다. 처음부터 학습하는 대신, 전이학습은 대규모 데이터셋에서 학습된 지식을 활용하여 더 작은 특수 데이터셋에 대한 성능을 향상시킵니다. 이 접근 방식은 실용적인 깊은 학습의 기초가 되었습니다.

## 핵심 정의

전이학습은 풍부한 학습 데이터가 있는 원본 도메인에서 학습된 표현과 매개변수를 제한된 데이터가 있는 대상 도메인으로 전달하는 것입니다. 핵심 가정은 작업이 사전 학습된 모델이 이미 발견한 공통의 기본 특성이나 패턴을 공유한다는 것입니다.

**세 가지 주요 구성 요소**:
1. **원본 도메인**: 초기 학습에 사용되는 대규모 데이터셋 (예: ImageNet)
2. **대상 도메인**: 관심 있는 특정 작업을 위한 더 작은 데이터셋
3. **전이 전략**: 원본에서 대상으로 지식을 어떻게 적응시킬 것인가

## 도메인 적응

도메인 적응은 원본 도메인과 대상 도메인이 크게 다를 때 발생하는 문제를 다룹니다. 기법들은 다음을 포함합니다:

**공변량 이동**: 입력 분포 P(X)는 변하지만 P(Y|X)는 유사하게 유지됩니다. 해결책은 다음을 포함합니다:
- 중요도 가중치 부여
- 특성 분포를 [정규화](/regularization-kor.html)하기 위한 [배치 정규화](/batch-normalization-kor.html)

**레이블 이동**: P(Y)는 변하지만 P(X|Y)는 안정적입니다. 다음을 통해 출력 공간 적응이 필요합니다:
- 레이블 스무딩
- 클래스 재가중치 부여

**일반 도메인 이동**: P(X)와 P(Y|X) 모두 변합니다. 더 정교한 기법이 필요합니다:
- 도메인 대적 학습 (판별기 사용)
- 특성 정렬 방법
- 자기 지도 적응

## 특성 추출 vs 미세 조정

**특성 추출 접근**:
- 최종 계층을 제외한 모든 사전 학습 가중치 고정
- 상단에 새로운 작업별 계층 추가
- 대상 데이터를 사용하여 새 계층만 학습
- 장점: 빠른 학습, 더 적은 대상 데이터 필요, 과적합 방지
- 사용 시기: 제한된 대상 데이터, 대상 작업이 원본과 다름

**미세 조정 접근**:
- 사전 학습된 가중치 부분 또는 전체 해제
- 작은 [학습률](/learning-rate-scheduling-kor.html)로 계속 학습
- 사전 학습된 매개변수와 새 매개변수 모두 업데이트
- 장점: 대상 작업에서 더 나은 성능, 표현 적응
- 사용 시기: 충분한 대상 데이터, 대상 작업이 원본과 유사, 계산 자원이 가능

**판별적 미세 조정**: 다양한 계층에 대해 다른 [학습률](/learning-rate-scheduling-kor.html)을 사용합니다. 더 깊은 계층이 작업별이고 얕은 계층보다 더 적극적으로 업데이트됩니다.

## 사전 학습된 모델 및 벤치마크

**ImageNet 사전 학습** ([컴퓨터 비전](/cv-kor.html)):
- 1000개 클래스 간 120만 개의 레이블 이미지에서 학습
- ResNet, VGG, EfficientNet, Vision Transformer가 일반적으로 사전 학습
- 학습된 특성은 의료 영상, 원격 감지, 자율 주행에 매우 전이 가능

**BERT** (자연어 처리):
- 마스크된 언어 모델링을 사용하여 대규모 텍스트 코퍼스에서 사전 학습
- 양방향 Transformer 인코더는 풍부한 문맥 표현을 캡처
- 분류, 명명된 개체 인식, 질문 답변을 위해 미세 조정됨
- 변형: 다양한 크기/속도 트레이드오프가 있는 RoBERTa, ALBERT, DistilBERT

**GPT 모델** (언어 생성):
- 인과 언어 모델링을 사용하여 다양한 인터넷 텍스트에서 사전 학습
- [GPT](/gpt-kor.html)-2/3/4는 강력한 제로샷 및 소수샷 전이 능력을 보여줍니다
- 미세 조정은 [코드 생성](/code-generation-kor.html), 요약의 특수한 응용을 가능하게 합니다

**도메인별 사전 학습**:
- BioBERT (생명의학 텍스트용)
- SciBERT (과학 논문용)
- ClimateBERT (기후 연구용)
- 종종 일반 사전 학습을 크게 능가합니다

## 전이학습이 잘 작동할 때

전이학습은 다음 경우에 가장 효과적입니다:

1. **원본과 대상 작업이 표현을 공유**:
   - ImageNet 특성은 위성 영상 분류로 잘 전이됨
   - 언어 모델 표현은 NLP 작업 전반에서 전이됨

2. **대상 데이터셋이 작음**:
   - 10K 샘플로, 전이학습은 처음부터의 학습을 크게 능가함
   - 수백만 샘플로, 장점은 감소함

3. **원본 도메인이 더 크고 다양함**:
   - ImageNet (120만 이미지)은 작업별 데이터셋보다 더 풍부한 특성 제공
   - 광범위한 언어 코퍼스는 더 많은 언어적 패턴 캡처

4. **계산 자원이 제한됨**:
   - 사전 학습은 몇 달 소요, 미세 조정은 몇 시간 소요
   - 개발 사이클에서 큰 실질적 이점

5. **작업이 계층적으로 연관**:
   - 저수준 특성 (모서리, 질감)은 비전 작업 전반에 전이됨
   - 구문 패턴은 NLP에 전이됨

## 부정적 전이

부정적 전이(Negative Transfer)는 전이학습이 처음부터 학습하는 것과 비교하여 성능을 해칠 때 발생합니다.

**원인**:
- 원본과 대상 도메인이 비유사함 (의료 영상 vs. 자연 영상)
- 사전 학습된 특성이 일반적이 아닌 작업별임
- 좋지 않은 적응 전략 선택

**예시**:
- ImageNet 사전 학습을 천문 영상 분석에 사용하면 성능이 떨어질 수 있음
- 일반 언어 모델은 신중한 적응 없이 매우 특수한 도메인으로 잘 전이되지 않을 수 있음
- 원본 도메인의 작업별 편견이 대상 도메인 성능을 해칠 수 있음

**완화 전략**:
- 중간 데이터셋에서 중간 사전 학습 사용
- 미세 조정 중 더 강한 [정규화](/regularization-kor.html) 적용
- 전이 및 처음부터의 학습을 결합하는 [앙상블 방법](/ensemble-methods-kor.html) 사용
- 원본-대상 도메인 조합을 체계적으로 검증

## 실제 전이학습 전략

**특성 추출로 시작**:
1. 사전 학습된 모델 로드
2. 최종 분류 계층 제거
3. 작업별 계층 추가
4. 고정된 백본으로 새 계층만 학습
5. 평가, 성능 좋으면 중단

**미세 조정 예열**:
1. 특성 추출로 시작
2. 점진적으로 더 깊은 계층 해제
3. 낮은 [학습률](/learning-rate-scheduling-kor.html) 사용 (0.001 대 0.0001)
4. 검증 성능 모니터링
5. 조기 종료 적용

**계층별 학습률 감소**:
- 새 계층에 대해 가장 높은 [학습률](/learning-rate-scheduling-kor.html) 설정
- 더 오래된 사전 학습 계층에 대해 점진적으로 낮춤
- 적응을 활성화하면서 재앙적 망각 방지

**데이터 증강**:
- 작은 대상 데이터셋에 더 강한 증강 사용
- 증강은 추가 데이터 다양성을 시뮬레이션
- 최대 효과를 위해 전이학습과 잘 결합됨

## 다중 작업 및 메타 학습 확장

**다중 작업 학습**: 여러 관련 원본 작업에서 동시에 학습합니다. 공유 표현은 모든 작업을 이롭게 하여 새로운 대상으로의 전이를 개선합니다.

**메타 학습** ("학습하는 방법 학습"): 적은 예제로 새로운 작업에 대한 빠른 학습에 잘 작동하는 초기화 및 적응 전략을 학습합니다.

## 현대 응용에서의 전이학습

**의료 영상**: ImageNet에서 사전 학습한 모델은 시각적 차이에도 불구하고 CT, MRI, X-레이 분석으로 효과적으로 전이됩니다.

**자연어 처리**: 전체 산업이 전이학습을 기반으로 구축됩니다. 미세 조정된 [BERT](/bert-kor.html) 모델이 규모에서 프로덕션 시스템을 강화합니다.

**강화 학습**: 사전 학습된 비전 모델은 제어 정책 학습을 가속화합니다. 사전 학습된 정책은 유사한 환경 간에 전이됩니다.

## 한계 및 과제

- **레이블 원본 데이터 필요**: 레이블이 없는 대규모 데이터셋을 활용할 수 없음 (자기 [지도 학습](/supervised-learning-kor.html)으로 부분적으로 해결)
- **도메인 간격**: 원본과 대상 간의 상당한 차이는 신중한 처리 필요
- **계산 비용**: 처음부터의 학습보다 빠르지만, 매우 큰 모델의 미세 조정은 여전히 비쌈
- **윤리적 고려사항**: 사전 학습된 모델은 원본 도메인의 편견을 인코딩할 수 있으며 신중한 평가 필요

## 결론

전이학습은 제한된 데이터와 자원으로 실용적인 시스템을 가능하게 하여 응용 머신러닝에 필수적이 되었습니다. [컴퓨터 비전](/cv-kor.html)에서 NLP까지, 사전 학습된 모델을 활용하는 능력은 개발을 크게 가속화하면서 종종 처음부터 학습하는 것보다 더 나은 성능을 달성합니다. 전이학습을 효과적으로 언제 어떻게 적용할지 이해하는 것은 현대 AI 실무자에게 필수적입니다.

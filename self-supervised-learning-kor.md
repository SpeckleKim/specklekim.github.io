# 자기지도학습: 데이터 자체에서 목표 만들기

자기[지도학습](/supervised-learning-kor.html)은 머신러닝의 패러다임 전환을 나타냅니다. 비싼 인간 주석에 의존하는 대신, 데이터 구조 자체에서 자동으로 훈련 신호를 생성합니다. 이는 모델이 대규모 미레이블 데이터셋에서 강력한 표현을 학습하도록 하여, 현대 기초 모델의 기초를 형성합니다.

## 자기지도학습이란?

자기[지도학습](/supervised-learning-kor.html)은 미레이블 데이터에서 인공 예측 과제(전가 작업)를 만드는데, 목표는 데이터에서 직접 파생됩니다. 이러한 전가 작업을 풀면서 모델은 후속 응용에 유용한 풍부한 특성 표현을 학습합니다.

### 핵심 아이디어

데이터를 인공적으로 입력과 목표 성분으로 나눕니다. 모델이 한 부분에서 다른 부분을 예측할 수 있다면, 의미 있는 패턴을 포착해야 합니다. 학습된 표현은 다른 작업으로 일반화됩니다.

**예시:**
- 문맥에서 마스킹된 단어 예측 ([BERT](/bert-kor.html))
- 주변 패치에서 이미지 영역 예측 (MAE)
- 동일 이미지의 증강된 뷰 일치 (SimCLR)

## 전가 작업

전가 작업은 유용한 표현을 훈련하도록 설계된 인공 예측 문제입니다.

### 문맥 예측
문맥에서 누락된 요소 예측. 언어 모델은 다음 토큰을 예측하고, 이미지 모델은 누락된 패치를 예측합니다.

**언어:** BERT는 입력 토큰의 15%를 마스킹하고 문맥에서 예측하도록 훈련합니다. 이는 모델이 의미론적 관계와 언어 구조를 이해하도록 강제합니다.

**시각:** 마스킹된 오토인코더(MAE)는 이미지 패치의 큰 부분(75%)을 마스킹하고 보이는 패치에서 복원하여 로컬과 글로벌 구조를 학습합니다.

### 회전 예측
이미지 회전(0°, 90°, 180°, 270°) 예측. 단순해 보이지만 이 작업은 모델이 객체 방향과 공간 구조를 이해하도록 강제합니다.

### 채색
그레이스케일 이미지를 컬러로 변환. 이 작업은 객체 카테고리와 의미론적 속성 이해가 필요합니다.

### 지그소우 퍼즐
이미지 패치를 뒤섞고 원래 배열 예측. 이는 공간 관계와 객체 구조 이해를 강제합니다.

## 대조 학습: 강력한 방식

대조 학습은 유사하고 상이한 예시를 비교하여 표현을 학습하는 획기적인 접근을 나타냅니다.

### 핵심 원리
동일 이미지의 다른 증강 뷰 간 일치를 최대화하면서 서로 다른 이미지의 뷰 간 일치를 최소화합니다.

**수학적 목표:** 양성 쌍(동일 이미지의 두 증강)에 대해 유사성 최대화. 음성 쌍(다른 이미지의 증강)에 대해 유사성 최소화.

### SimCLR (표현의 단순 대조 학습)

SimCLR은 단순함에서 우아하면서도 놀랍도록 효과적입니다.

**알고리즘:**
1. 각 이미지에 두 개의 확률적 증강 적용(자르기, 색상 지터, 회전) → x₁, x₂
2. 공유 인코더로 둘 다 인코드 → z₁ = f(x₁), z₂ = f(x₂)
3. 낮은 차원으로 투영 → h₁ = g(z₁), h₂ = g(z₂)
4. 대조 손실 계산 (NT-Xent: [정규화](/regularization-kor.html) 온도 스케일 교차 [엔트로피](/information-theory-kor.html))
5. h₁과 h₂의 유사성 최대화, 배치의 다른 것과는 최소화

**작동 이유:** 동일 이미지의 두 증강은 의미론적 내용을 공유하지만 저수준 세부는 다릅니다. 모델은 표면적 차이를 무시하고 의미론적 특성에 집중하도록 학습합니다.

**핵심 성분:**
- **강한 증강:** 필수적, 증강 다양성 없이 모델은 이미지 내용 무시
- **투영 헤드:** 손실 전에 저차원으로 투영이 학습된 표현 향상
- **큰 배치 크기:** 효과적 대조 학습을 위해 많은 음성 필요
- **온도 파라미터:** 손실 집중도 제어

### 모멘텀 대조 (MoCo)

MoCo는 인코딩된 예시의 모멘텀 업데이트 큐를 유지하여 SimCLR을 개선합니다.

**혁신:** 거대한 배치 크기를 요구하는 대신 MoCo는 큐와 모멘텀 인코더를 사용합니다:
- 인코더 가중치는 경사 하강으로 업데이트
- 모멘텀 인코더는 메인 인코더의 느린 이동 평균으로 업데이트
- 큐는 최근 배치에서 인코딩된 예시 저장

**장점:** 작은 배치 크기로 잘 작동하고 더 실제적입니다.

**개선:** MoCo v2는 SimCLR 통찰을 결합하여 최첨단 결과 달성.

### BYOL (당신의 자신의 잠재 부트스트래핑)

BYOL은 음성 쌍의 필요성을 제거합니다—관례적 지혜에 도전하는 놀라운 발견.

**핵심 통찰:** 명시적 음성이 필요 없고, 대신 목표 네트워크(메인 네트워크의 EMA) 사용과 예측자 네트워크로 표현 붕괴 방지.

**알고리즘:**
1. 메인 네트워크를 통해 증강된 이미지 인코드
2. 목표 네트워크를 통해 다른 증강 인코드 (EMA 업데이트)
3. 메인 예측과 목표 표현 간 일치 최대화
4. 예측자가 모든 임베딩이 동일한 자명한 솔루션 방지

**혁명적:** 음성의 필요성을 문제화하고, 학습 목표 자체가 충분한 제약을 제공함을 제시합니다.

## 마스킹 예측 방법

마스킹 예측은 대규모로 문맥 예측 전통을 계속합니다.

### BERT (쌍방향 인코더 표현 Transformers)

BERT는 입력 토큰의 15%를 마스킹하고 [트랜스포머](/llm-kor.html)가 쌍방향 문맥에서 마스킹된 토큰을 예측하도록 훈련합니다. 이 자기지도 사전훈련은 후속 NLP 작업으로 놀랍게 잘 전이됩니다.

**마스킹 전략:**
- 80%의 시간: [MASK] 토큰으로 대체
- 10%의 시간: 무작위 토큰으로 대체
- 10%의 시간: 원본 유지

이 무작위화는 모델이 견고한 표현을 학습하도록 강제합니다.

### 마스킹된 오토인코더 (MAE)

MAE는 BERT의 마스킹을 비전으로 확대합니다. 중요한 혁신: 75% 패치 마스킹(BERT의 15% 토큰 대비).

**비대칭 설계:**
- 인코더는 보이는 패치만 봅니다(25%)
- 디코더는 인코더 출력에서 모든 패치 복원
- 비대칭이 핵심—보이는 패치만 인코딩은 디코더가 의미 있는 특성을 학습하도록 강제

**결과:** 풍부한 시각 이해 교육, 후속 작업으로 우수한 전이.

### 생성 사전훈련

[GPT](/gpt-kor.html) 같은 대형 언어 모델은 다음 토큰 예측으로 학습합니다. 이 단순한 목표는 대규모로 적용되어 놀라운 [창발적 능력](/emergent-abilities-kor.html)을 가진 모델을 생산합니다—소수 샷 학습, 추론, [코드 생성](/code-generation-kor.html).

## DINO: 음성이나 모멘텀 없이 출현

DINO(교사 모델의 자기지도 Vision Transformer)는 성능을 유지하거나 개선하면서 여러 아키텍처 성분을 제거합니다.

**핵심 성분:**
1. Vision Transformer 백본
2. 음성 예시 없음
3. 모멘텀 인코더 없음
4. 학생과 교사 네트워크(교사는 학생의 EMA)
5. 날카로운 소프트맥스(작은 온도)

**과정:** 학생이 다른 증강에서 교사 출력 예측을 시도합니다. 소프트맥스의 온도는 날카로운 분포를 만들어 집중된 예측을 강제합니다.

**놀라운 발견:** DINO로 훈련된 ViT의 자기 주의 맵은 자연스럽게 의미론적 의미로 나타나며—명시적 세분화 레이블 없음.

## 기초 모델과의 관계

기초 모델은 자기[지도 학습](/supervised-learning-kor.html)으로 대규모 미레이블 데이터에서 훈련된 대형 모델이며, 다양한 후속 작업을 위해 미세 조정됩니다.

**자기지도 학습이 기초 모델을 가능하게 하는 이유:**
1. **규모:** 레이블 병목 없음, 모든 사용 가능 데이터 사용 가능
2. **전이:** 학습된 표현이 도메인과 작업 전체에 일반화
3. **효율성:** 적절한 레이블 데이터로 미세 조정이 강한 성능 달성
4. **출현:** 대규모 자기[지도 학습](/supervised-learning-kor.html)이 예상치 못한 능력 드러냄

**예시:**
- **시각:** CLIP(대조 텍스트-이미지), DINO를 가진 [ViT](/vision-transformers-kor.html)
- **언어:** [BERT](/bert-kor.html), [GPT](/gpt-kor.html) 제품군, T5
- **다중 모달:** 텍스트, 시각, 오디오 결합 기초 모델

## 자기지도 학습이 현대 AI에 권한을 주는 이유

자기[지도 학습](/supervised-learning-kor.html)은 중요한 병목을 해결했습니다: 주석 부족. 명시적 레이블이 없는 인터넷 규모의 대부분의 데이터를 활용하도록 합니다.

### 장점:
1. **데이터 효율성:** 대규모 미레이블 데이터에서 학습
2. **일반화:** 비지도에서 학습한 표현은 전이 시 지도 기준을 종종 능가
3. **레이블 편향 없음:** 인간 주석의 체계적 편향 회피
4. **창발적 특성:** 대형 모델이 예상치 못한 능력 표현

### 도전:
1. **초매개변수 민감성:** 많은 설계 선택(증강, 온도, 배치 크기)
2. **계산 비용:** 큰 계산 자원과 데이터 필요
3. **이론적 이해:** 특정 전가 작업이 왜 잘 작동하는지는 부분적으로 신비로움
4. **도메인 특이성:** 작업 설계는 다른 도메인을 위해 맞춤 필요 가능

## 기초 모델 시대

자기[지도 학습](/supervised-learning-kor.html)이 현재 기초 모델 시대에 권한을 줍니다:
- 대규모 미레이블 데이터에서 사전훈련된 Transformers
- 수많은 후속 응용으로 미세 조정
- 작업마다 조정된 파라미터가 비교적 적음
- 강한 영점 샷과 소수 샷 성능

이 패러다임은 AI를 변형했으며, [GPT](/gpt-kor.html)-4, Claude, DALL-E, Gemini 같은 시스템이 최소 작업별 감독으로 강력한 세계 모델을 학습하도록 가능하게 합니다.

자기[지도 학습](/supervised-learning-kor.html)은 단순한 기법이 아닙니다—풍부한 데이터와 드문 레이블의 시대에 머신러닝에 접근하는 방식의 근본적 전환입니다.

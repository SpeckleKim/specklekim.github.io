# 확산 모델

## 개요

확산 모델(Diffusion Models)은 통계 물리학의 확산에 영감을 받은 과정을 통해 최고 수준의 [이미지 생성](/image-generation-kor.html) 품질을 달성하는 생성 모델링의 패러다임 변화를 나타냅니다. GAN이나 VAE와 달리, 확산 모델은 무작위 잡음을 반복적으로 제거하여 이미지를 생성하며, 점진적 잡음 추가 과정을 역으로 학습합니다.

## 핵심 개념: 전진 과정과 역과정

### 전진 과정 (확산)

전진 과정은 T 타임스텝에 걸쳐 데이터에 가우스 잡음을 점진적으로 추가합니다:

```
q(x_t | x_{t-1}) = N(x_t; √(1-β_t) * x_{t-1}, β_t * I)
```

깨끗한 데이터 x_0에서 시작하여, 잡음이 점진적으로 추가되어 x_T가 가우스 분포와 구별되지 않는 순수 잡음이 됩니다. 이 과정은 결정적이며 계산 가능하며 폐쇄형이 있습니다:

```
x_t = √(ᾱ_t) * x_0 + √(1-ᾱ_t) * ε, 여기서 ε ~ N(0,I)
```

계수 ᾱ_t (1-β_t의 누적 곱)는 각 단계의 잡음 수준을 결정합니다.

### 역과정 (제거)

생성은 이 과정을 역으로 하는 방법을 학습함으로써 발생합니다: 잡음 x_T에서 시작하여 반복적으로 제거하여 x_0을 복구합니다. 역과정은 또한 가우스 분포입니다:

```
p(x_{t-1} | x_t) = N(x_{t-1}; μ_θ(x_t, t), Σ_θ(x_t, t))
```

[신경망](/neural-kor.html)은 각 제거 단계에서 평균 μ_θ과 분산 Σ_θ을 학습합니다. 생성 중에, 우리는 반복적으로 학습된 역과정에서 샘플링합니다.

## DDPM: 제거 확산 확률 모델

Ho et al. (2020)이 소개한 DDPM은 현대 확산 모델의 기초를 수립합니다. 핵심 통찰은 네트워크가 각 단계에서 추가된 잡음을 예측하도록 단순화될 수 있다는 것입니다:

```
손실 = ||ε - ε_θ(x_t, t)||²
```

평균을 직접 예측하는 대신, 모델은 전진 과정 중에 추가된 잡음 ε을 예측합니다. 이 재구성은 훈련 안정성과 수렴을 극적으로 개선합니다.

훈련 절차:
1. 1부터 T까지 무작위 t 샘플링
2. 잡음 ε ~ N(0,I) 샘플링
3. 전진 공식을 사용하여 노이즈 이미지 x_t 계산
4. x_t와 t에서 ε을 예측하도록 네트워크 훈련
5. 재구성 오류 [역전파](/backpropagation-kor.html)

## 잡음 스케줄링

잡음 스케줄링은 β_t가 타임스텝 전체에서 어떻게 변하는지를 정의하여 각 단계에서 추가되는 잡음의 양을 제어합니다. 효과적인 스케줄링은 모델 성능에 중요합니다.

### 선형 스케줄

초기 단계에서는 작은 β부터 나중 단계에서는 큰 β까지의 단순 선형 진행입니다. 구현하기 쉽지만 최적이 아닐 수 있습니다.

### 코사인 스케줄

```
β_t = sin²(π/2 * t/T)
```

원칙적인 방식으로 잡음을 점진적으로 증가시키면서 경험적으로 더 잘 수행하여, 타임스텝 간에 더 부드러운 전환을 만듭니다.

### 제곱 스케줄 및 변형

다양한 경험적 스케줄이 제안되었으며, 성공은 데이터셋과 응용에 따라 달라집니다. 최근 작업은 학습되거나 적응형 스케줄링 전략을 탐색합니다.

## 제거 점수 매칭

더 깊은 이론적 관점은 확산 모델을 점수 기반 생성 모델과 연결합니다. 점수 함수는 로그 확률의 기울기를 추정합니다:

```
s_θ(x_t, t) = ∇_{x_t} log p(x_t)
```

제거 점수 매칭은 이 점수를 예측하도록 네트워크를 훈련하여, 잡음 예측 목표에 대한 이론적 정당성을 제공합니다. 이 연결은 개선된 샘플링과 이해를 위한 경로를 엽니다.

## 분류기 없는 안내

분류기 없는 안내는 별도의 분류기를 훈련하지 않고도 조건부 생성을 가능하게 합니다. 분류기를 사용하여 생성을 목표 클래스로 안내하는 대신, 모델은 조건부 및 무조건부 예측을 비교하도록 학습합니다.

생성 중에, 제거 단계가 조정됩니다:

```
ˆε_θ(x_t, t) = ε_θ(x_t, t, ∅) + w * (ε_θ(x_t, t, c) - ε_θ(x_t, t, ∅))
```

여기서 w는 안내 가중치입니다. 더 높은 w 값은 클래스 조건화 강도를 증가시키지만 다양성을 감소시킬 수 있습니다. 이 기법은 추가 네트워크 없이 강력한 텍스트-[이미지 생성](/image-generation-kor.html)을 가능하게 합니다.

## Stable Diffusion 아키텍처

Stable Diffusion은 잠재 공간 확산을 도입하여 전체 해상도 이미지가 아닌 압축된 잠재 표현에서 확산을 수행합니다. 이는 품질을 유지하면서 계산 비용을 극적으로 감소시킵니다.

아키텍처는 다음으로 구성됩니다:

**오토인코더:** 이미지를 컴팩트 잠재 공간으로 압축하여 차원 8-16배 감소

**확산 U-Net:** 잠재 공간에서 작동하여 반복적 제거 수행

**CLIP 텍스트 인코더:** 텍스트 프롬프트를 조건부 생성을 위한 임베딩으로 변환

**변분 디코더:** 잠재 표현에서 고해상도 이미지 재구성

이 3가지 구성 요소 설계는 고품질 확산 모델을 더 넓은 대상에 접근 가능하게 만들었으며 광범위한 채택을 가능하게 했습니다.

## 잠재 확산 모델

잠재 확산은 픽셀 공간이 아닌 학습된 잠재 공간에 확산을 적용하여 원칙을 더 확장합니다. 이점은 다음을 포함합니다:

- 계산 효율성: 훈련 및 추론에서 4-16배 속도 향상
- 지각 거리 지표와의 더 나은 정렬
- 다른 잠재 공간 방법과의 구성 능력
- 조작을 위한 더 깨끗한 특성 표현

잠재 확산은 소비자 하드웨어에서 고해상도 생성의 실질적 배포를 가능하게 합니다.

## 훈련 및 샘플링

**훈련:** GAN과 비교하여 간단하고 안정적입니다. 무작위 타임스텝에서 잡음을 예측해야 하며, 직관적인 [역전파](/backpropagation-kor.html)를 수행합니다. 모드 붕괴나 훈련 불안정성이 없습니다.

**샘플링:** T 단계를 통해 잡음에서 반복적 제거입니다. 단일 통과 GAN보다 느리지만, 품질-속도 트레이드오프는 확산 모델을 선호합니다. DDIM과 같은 기법은 품질을 희생하지 않고 필요한 단계를 줄입니다.

## 응용

**텍스트-이미지:** DALL-E 3, Stable Diffusion, Midjourney—설명에서 포토 리얼리스틱 [이미지 생성](/image-generation-kor.html)

**이미지-이미지:** 스타일 이전, 스케치 또는 마스크 기반 조건부 생성

**인페인팅:** 지능형 이미지 편집 및 마스크된 영역의 콘텐츠 생성

**비디오 생성:** 일관된 프레임 합성을 위한 시간적 확산 모델

**3D 생성:** 3D 형태 및 장면 합성에 확산 확장

**의료 영상:** 아티팩트 제거, 초고해상도, [데이터 증강](/data-augmentation-kor.html)

## 장점과 과제

**장점:**
- 최고 수준의 생성 품질
- 대항적 역학 없이 안정적이고 원칙적인 훈련
- 유연한 조건화 메커니즘 (분류기 없는 안내)
- 통계 물리학의 이론적 기초
- 고해상도로 확장 가능

**과제:**
- 많은 제거 단계가 필요한 느린 생성
- 샘플링 중 높은 계산 비용
- 훈련은 상당한 [GPU](/gpu-hardware-kor.html) 메모리 필요
- 생성된 콘텐츠에 대한 미세한 제어 어려움
- 모드 평균화의 가능성 (예상보다 덜 다양함)

## 결론

확산 모델은 원칙적인 확률 학습을 통해 우수한 품질을 달성함으로써 생성형 AI를 혁신했습니다. 안정적인 훈련, 유연성, 확장성은 지배적인 생성 패러다임으로 포지셔닝합니다. 일관성 증류 및 잠재 흐름과 같은 기법을 통해 샘플링 속도를 해결하면서, 확산 모델은 무수한 응용에 걸쳐 실제 배포를 향해 계속 진화하고 있습니다.

# CNN 심층 분석: 합성곱 신경망 이해하기

## 소개

합성곱 [신경망](/neural-kor.html)(CNN)은 합성곱 연산과 계층적 특징 추출을 통해 [컴퓨터 비전](/cv-kor.html)을 혁신했습니다. 완전 연결 네트워크와 달리 CNN은 매개변수 공유와 공간 구조를 활용하여 이미지 처리에서 뛰어난 효율성을 보입니다.

## 합성곱 연산

합성곱의 핵심은 입력과 커널(필터) 사이의 수학적 연산입니다. 커널은 입력 전체를 슬라이딩하며 원소별 곱을 계산하고 합산합니다. 이 연산은 이미지의 다양한 위치에서 엣지, 텍스처, 패턴 등 국소 특징을 추출합니다.

수식으로 2D 합성곱은:
```
(I * K)[i,j] = ∑∑ I[i+m, j+n] × K[m,n]
```

여기서 I는 입력, K는 커널입니다.

## 필터와 특징맵

필터는 특정 패턴을 감지하는 학습 가능한 커널입니다. 초기 계층은 저수준 특징(엣지, 모서리)을, 깊은 계층은 고수준 의미적 특징을 학습합니다. 각 필터는 특징맵을 생성하는데, 이는 입력에서 패턴이 나타나는 위치를 2D로 표현합니다.

## 스트라이드와 패딩

**스트라이드**는 각 단계에서 필터가 이동하는 픽셀 수입니다. 스트라이드 1은 밀집된 특징맵을, 스트라이드 2는 다운샘플링을 생성합니다. 이는 계산량을 줄이고 수용영역을 증가시킵니다.

**패딩**은 입력 경계에 픽셀(보통 0)을 추가합니다. 유효 패딩은 경계 정보를 버리고, 동일 패딩은 공간 차원을 보존합니다. 패딩은 엣지 정보 손실을 방지하고 출력 차원을 제어합니다.

## 풀링 연산

풀링 계층은 작은 영역에서 집계 통계를 취해 특징맵을 다운샘플링합니다:

- **최댓값 풀링**: 최댓값 추출 (가장 일반적)
- **평균 풀링**: 평균값 계산
- **확률적 풀링**: 확률 기반 랜덤 샘플링

풀링은 계산량을 줄이고 과적합을 방지하며 작은 공간 변환에 대한 강건성을 증가시킵니다.

## 수용영역(Receptive Field)

수용영역은 원본 입력의 얼마나 많은 부분이 출력 뉴런에 영향을 미치는지 측정합니다. 네트워크 깊이와 필터 크기가 증가하면 확장됩니다. 아키텍처 설계에서 중요합니다—큰 영역은 맥락을 포착하고 작은 영역은 세부사항에 집중합니다.

수용영역 공식:
```
RF = previous_RF + (filter_size - 1) × stride_product
```

## 특징맵 시각화

다양한 깊이의 특징맵은 계층적 학습을 드러냅니다. 얕은 계층은 엣지와 모서리를 감지하고, 중간 계층은 이들을 텍스처와 부분으로 결합하며, 깊은 계층은 객체와 장면을 인식합니다. 이 계층적 표현이 CNN의 효율성을 설명합니다.

## 유명한 아키텍처 개요

**AlexNet (2012)**: 8계층의 선구적 깊은 CNN. ReLU와 [드롭아웃](/dropout-kor.html)이 얕은 접근보다 뛰어남을 입증했습니다.

**VGGNet (2014)**: 작은 3×3 필터를 통한 깊이 강조. AlexNet보다 더 단순하고 균일한 구조입니다.

**ResNet (2015)**: 매우 깊은 네트워크(50-152 계층)를 가능하게 하는 [잔차 연결](/residual-connections-kor.html) 도입. [스킵 연결](/residual-connections-kor.html)이 [기울기 소실](/vanishing-gradients-kor.html) 문제를 해결합니다.

**Inception/GoogLeNet (2014)**: 다양한 크기의 필터 경로를 병렬로 사용한 다중 규모 특징 추출.

**MobileNet (2017)**: 깊이별 분리 합성곱을 사용한 효율적 아키텍처로 모바일/임베디드 장치용입니다.

**DenseNet (2016)**: 모든 계층 간 밀집 연결로 기울기 흐름을 개선하고 매개변수를 감소시킵니다.

## 현대 동향

최신 아키텍처는 효율성(MobileNets, ShuffleNets), 하이브리드 접근(CNN과 주의 메커니즘 결합), [비전 트랜스포머](/vision-transformers-kor.html)에 초점을 맞춥니다. 그러나 합성곱은 기본으로 남아있으며—[트랜스포머](/llm-kor.html) 기반 비전 모델도 합성곱 요소를 통합합니다.

## 결론

CNN은 공간적 국소성, 변환 불변성, 계층적 구성이라는 도메인 지식을 활용하므로 성공합니다. 합성곱, 필터, 패딩, 풀링, 수용영역의 이해는 현대 [컴퓨터 비전](/cv-kor.html)의 기초입니다. 이 개념들은 아키텍처가 진화해도 관련성을 유지합니다.

# AI를 위한 확률과 통계

확률과 통계는 머신러닝의 수학적 기초입니다. 불확실성 하에서의 의사결정부터 확률 모델 학습까지, 확률분포, 베이지안 추론, 그리고 통계적 가설검정을 이해하는 것은 효과적인 AI 개발에 필수적입니다.

## 확률의 기초

**확률**은 사건이 발생할 가능성을 측정합니다. 0 (불가능)부터 1 (확실)까지의 범위를 가집니다. 확률론은 데이터와 모델의 불확실성에 대한 추론의 기초를 제공합니다.

**주요 개념:**
- **표본공간(Ω)**: 모든 가능한 결과
- **사건**: 표본공간의 부분집합
- **사건 A의 확률**: P(A) = (유리한 결과의 수) / (전체 결과의 수)

AI에서는 종종 **연속확률분포**를 다루며, 이는 확률밀도함수(PDF)를 사용하여 구간에 대해 정의됩니다.

## 조건부 확률과 독립성

**조건부 확률**은 사건 B가 발생했을 때 사건 A가 발생할 확률입니다: **P(A|B) = P(A∩B) / P(B)**

이는 원인과 결과에 대해 추론하는 데 근본적입니다. 예를 들어, 스팸 감지에서 우리는 P(스팸|이메일 내용)에 관심이 있습니다.

**독립성**: 사건 A와 B가 독립적이면 **P(A|B) = P(A)**입니다. 즉, B는 A에 대한 정보를 제공하지 않습니다.

**연쇄법칙**: **P(A,B,C) = P(A)·P(B|A)·P(C|A,B)**

결합 확률을 더 간단한 조건부 확률로 분해하면 복잡한 확률을 계산할 수 있습니다.

## 베이즈 정리

베이즈 정리는 확률적 추론의 핵심입니다: **P(A|B) = P(B|A)·P(A) / P(B)**

머신러닝 용어로:
- **P(A|B)**: 사후확률(posterior, 계산하려는 것)
- **P(B|A)**: 우도(likelihood, 모델이 데이터를 얼마나 잘 설명하는가?)
- **P(A)**: 사전확률(prior, 데이터를 보기 전의 믿음)
- **P(B)**: 증거(evidence, 데이터의 확률)

베이즈 정리는 **베이지안 추론**을 가능하게 합니다: 새로운 증거를 받으면 믿음을 업데이트합니다. 더 많은 데이터를 수집할수록, 사후확률은 더 확실해지며 참 값 근처에 집중됩니다.

## 일반적인 확률분포

**정규분포(가우스분포)**: AI와 통계에서 가장 중요한 분포입니다. 평균 μ와 분산 σ²로 정의됩니다. 종 모양의 곡선은 신장, 측정 오차, 신경망 활성화 같은 현상을 설명합니다.

**베르누이분포**: 이진 결과(동전 앞뒤, 참/거짓)를 모델링합니다. 매개변수 p (성공 확률)로 정의됩니다. 이진 분류에 사용됩니다.

**범주분포(카테고리분포)**: 베르누이분포의 여러 결과로의 일반화입니다. 각 범주는 확률 p_i를 가집니다. 다중클래스 분류에 사용됩니다.

**포아송분포**: 횟수 데이터(고정 시간 내 사건 수)를 모델링합니다. 매개변수 λ (발생률)로 정의됩니다. 사건 계수, 도착 프로세스에 사용됩니다.

**지수분포**: 다음 사건까지의 시간(대기 시간)을 모델링합니다. 생존 분석과 지속 시간 모델링에 사용됩니다.

**균등분포**: 범위 내 모든 결과에 동일한 확률을 가집니다. 도메인 지식이 없을 때 기본 사전으로 사용됩니다.

많은 AI 알고리즘은 데이터가 특정 분포에서 나온다고 가정합니다 (예: 가우스 나이브 베이즈는 특성이 정규분포를 따른다고 가정).

## 기댓값과 분산

**기댓값(평균)**: 확률변수가 취하는 평균값입니다: **E[X] = Σ x·P(x)** (이산) 또는 **E[X] = ∫ x·p(x)dx** (연속)

기댓값은 선형입니다: **E[aX + b] = a·E[X] + b**

**분산**: 평균 주위의 퍼짐을 측정합니다: **Var(X) = E[(X - E[X])²]**

높은 분산은 값들이 퍼져있다는 의미; 낮은 분산은 평균 주위에 몰려있다는 의미입니다.

**표준편차**: σ = √Var(X), X와 같은 단위입니다.

**공분산**: **Cov(X,Y) = E[(X - E[X])·(Y - E[Y])]** 두 변수가 함께 움직이는 방식을 측정합니다.

**상관계수**: 정규화된 공분산입니다: **ρ = Cov(X,Y) / (σ_X · σ_Y)**, -1부터 +1 범위입니다.

## 최대우도추정(MLE)

MLE는 관측된 데이터를 가장 가능하게 만드는 매개변수 값을 찾습니다. 데이터 D와 매개변수 θ가 주어졌을 때:

**L(θ|D) = P(D|θ)** (우도함수)

MLE는 다음을 선택합니다: **θ_MLE = argmax_θ P(D|θ)**

종종 로그우도(수치적으로 안정적)를 최대화합니다: **θ_MLE = argmax_θ Σ log P(d_i|θ)**

**왜 MLE를 사용할까?**
- 매개변수 추정에 대한 원칙적인 접근
- 특정 조건에서, MLE 추정량은 일관성 있고 점근적으로 정규분포를 따릅니다
- 로지스틱 회귀, 가우스 나이브 베이즈, 그리고 많은 다른 모델을 통해 활용됩니다

## 최대사후확률(MAP)

MAP는 우도에 사전 지식을 결합합니다: **θ_MAP = argmax_θ P(θ|D) = argmax_θ P(D|θ)·P(θ)**

이는 베이지안 추론입니다: 데이터가 말하는 것(우도)과 우리가 처음에 믿었던 것(사전)의 균형을 잡습니다.

사전 P(θ)가 균등할 때, MAP은 MLE와 같습니다. 강한 사전은 추정을 예상 값으로 끌어당겨 과적합을 방지할 수 있습니다.

## 가설검정

가설검정은 통계적 의사결정을 위한 프레임워크를 제공합니다:

1. **귀무가설(H₀)**: 기본 가정 (예: "효과 없음")
2. **대립가설(H₁)**: 우리가 의심하는 것 (예: "효과 있음")
3. **검정통계량**: 데이터로부터 계산
4. **P값**: H₀를 가정할 때, 관측된 것만큼 또는 더 극단적인 데이터를 관측할 확률
5. **유의수준(α)**: 임계값 (일반적으로 0.05)

P값 < α이면, H₀을 기각합니다. 작은 P값은 귀무가설에 대한 증거를 제공합니다.

**1종 오류**: H₀가 참일 때 기각 (거짓양성)
**2종 오류**: H₀가 거짓일 때 기각하지 않음 (거짓음성)

## 머신러닝에서의 응용

**분류**: 확률적 분류기를 위해 베이즈 정리를 사용합니다. 나이브 베이즈는 클래스가 주어졌을 때 특성 독립성을 가정합니다.

**회귀**: MLE 추정은 선형 및 로지스틱 회귀 모델의 매개변수를 맞춥니다.

**불확실성 정량화**: 베이지안 모델은 단순 점 추정이 아닌 예측에 대한 확률분포를 출력합니다.

**모델 선택**: 교차검증과 정보 기준(AIC, BIC)은 우도 기반 추론을 사용합니다.

**생성 모델**: VAE와 GAN은 명시적으로 데이터의 확률분포를 모델링합니다.

**강화학습**: MDP 이론은 근본적으로 확률론적입니다; 에이전트는 상태 전환과 보상을 확률론적으로 추론합니다.

확률과 통계는 불확실하고 불완전한 정보를 원칙적인 의사결정으로 변환합니다. 기계가 데이터로부터 학습하면서 예측의 신뢰도를 정량화할 수 있게 합니다.

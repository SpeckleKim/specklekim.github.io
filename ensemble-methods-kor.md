# 앙상블 학습

## 소개

앙상블 방법은 여러 기본 학습기를 결합하여 어떤 단일 학습기보다 강력한 모델을 만듭니다. 핵심 원리: 다양하고 중간 정도 정확한 모델들이 집계를 통해 집단적으로 높은 정확도를 달성할 수 있습니다. 앙상블 방법은 현대 머신러닝에서 가장 실용적이고 성공적인 접근법 중 하나이며 많은 최고 수준의 솔루션을 구동합니다.

## 앙상블이 작동하는 이유

### 편향-분산 트레이드오프

앙상블은 편향과 분산을 모두 줄여 성능을 향상시킵니다:

**평균화 효과:**
```
Var(평균) = Var(개별) / n (상관성이 없으면)
```

상관성 없는 예측을 결합하면 분산이 크게 감소합니다.

**다양성 요구사항:**
- 동일한 학습기는 도움이 되지 않음(분산 감소 없음)
- 다양성은 다양한 훈련 데이터, 피처 또는 알고리즘에서 나옴
- 비상관 오류가 필수적

### 수학적 직관

각 학습기가 정확도 p > 0.5를 달성하고 독립적으로 오류를 만든다면:

```
앙상블_정확도 ≈ Σ C(n,k) × p^k × (1-p)^(n-k) for k > n/2
```

충분한 다양성이 있으면 앙상블 정확도는 개별 정확도에 관계없이 1에 접근합니다.

## 배깅(부트스트랩 집계)

배깅은 훈련 데이터의 다양한 부트스트랩 샘플에서 동일한 기본 학습기의 여러 사본을 훈련합니다.

**과정:**
1. 복원 추출로 B개의 부트스트랩 샘플 생성
2. 각 샘플에서 기본 학습기를 독립적으로 훈련
3. 예측 평균화:
   - 분류: 다수결 투표
   - 회귀: 예측 평균

**핵심 통찰:**
- 평균화를 통한 분산 감소
- 병렬화 가능(독립적 훈련)
- 높은 분산, 낮은 편향 학습기에 최적
- 랜덤 포레스트는 피처 무작위성으로 배깅 확장

**사용 시기:**
- 의사결정나무(과적합 감소)
- 복잡하고 불안정한 모델
- 계산 리소스 가용

## 부스팅

부스팅은 순차적으로 학습기를 훈련하며, 각각의 새 학습기는 이전 학습기에 의해 오분류된 샘플에 초점을 맞춥니다. 이는 다양성을 증가시키고 편향과 분산을 모두 줄입니다.

### AdaBoost(적응형 부스팅)

**알고리즘:**
1. 모든 샘플에 동일한 가중치로 초기화
2. 각 반복:
   - 가중된 샘플에서 약한 학습기 훈련
   - 가중 오류율 계산
   - 학습기 가중치 계산(낮은 오류 → 높은 가중치)
   - 샘플 가중치 업데이트(오분류 샘플의 가중치 증가)
3. 최종 예측: 가중 투표

**핵심 공식:**
```
w_m = (1/2) × ln((1 - ε_m) / ε_m)
```

여기서 ε_m은 학습기 m의 가중 오류입니다.

**장점:**
- 편향을 효과적으로 감소
- 이론적 기반
- 모든 약한 학습기와 작동

**단점:**
- 순차적(병렬화 불가)
- 노이즈와 이상치에 민감

### 그래디언트 부스팅

그래디언트 부스팅은 경사 하강법을 사용하여 각각의 새 학습기를 이전 학습기의 잔차(오류)에 적합시킵니다:

**과정:**
1. 평균 예측으로 초기화
2. 각 반복:
   - 현재 예측에서 잔차 계산
   - 새 학습기를 잔차에 적합
   - 단계 크기(학습률)로 앙상블에 추가
3. 최종 예측: 모든 학습기의 합

**핵심 장점:**
```
F(x) = F_(m-1)(x) + η × h_m(x)
```

여기서 η는 기여도를 제어하는 학습률(0 < η ≤ 1)입니다.

**인기 있는 변형:**
- **XGBoost**: 정규화를 가진 최적화된 그래디언트 부스팅
- **LightGBM**: 빠르고 메모리 효율적인 그래디언트 부스팅
- **CatBoost**: 범주형 피처를 자연스럽게 처리

**장점:**
- 강력하고 종종 최고의 단일 알고리즘
- 유연함, 다양한 손실 함수 처리
- 피처 중요도 계산
- 이상치에 상대적으로 강건

**단점:**
- 순차적(느린 훈련)
- 더 많은 하이퍼파라미터 튜닝
- 신중하지 않으면 과적합 위험

## 스태킹

스태킹은 기본 학습기의 예측을 최적으로 결합하기 위해 메타 학습기를 훈련합니다.

**과정:**
1. 훈련 데이터 분할
2. 한 부분에서 기본 학습기 훈련
3. 보류된 부분에서 예측 생성
4. 이 예측에서 메타 학습기 훈련
5. 테스트 데이터: 기본 학습기 예측 → 메타 학습기 결합

**장점:**
- 다양한 학습기 강점 결합
- 유연한 아키텍처
- 유사한 기본 학습기에서도 개선 가능

**단점:**
- 계산 비용이 높음
- 신중한 검증 없이 과적합 경향
- 올바르게 구현하기 복잡

## 투표

투표는 간단한 다수결 또는 가중 투표를 사용하여 여러 학습기의 예측을 결합합니다.

**경질 투표(분류):**
```
예측 = 다수결(모든 학습기의 예측)
```

**연질 투표(분류):**
```
예측 = argmax(모든 학습기의 확률 평균)
```

**장점:**
- 구현이 간단
- 다양한 학습기 유형과 작동
- 기준선 앙상블 접근법

## 랜덤 포레스트

랜덤 포레스트는 피처 무작위성이 있는 의사결정나무에 특화된 배깅입니다:

**단순 배깅과의 핵심 차이:**
- 피처 부분 샘플링: 각 분할에서 무작위로 피처 선택
- 높은 트리 깊이: 완전한 트리 성장(편향 감소)
- 비상관화: 피처 무작위성이 트리 상관성 감소

**효과적인 이유:**
- 부트스트랩이 분산 감소
- 피처 무작위성이 다양성 증가
- 많은 트리가 복잡한 패턴 포착
- 이상치와 노이즈에 강건

## 실무 팁

### 하이퍼파라미터 튜닝

**배깅:**
- 기본 학습기 수: 수확 체감까지 증가가 도움
- 기본 학습기 복잡도: 편향 감소를 위해 증가

**부스팅:**
- 학습률: 낮은 값은 느리지만 더 정확
- 반복 수: 과적합까지 개선
- 기본 학습기 복잡도: 일반적으로 약한 학습기(얕은 깊이)

**스태킹:**
- 기본 학습기의 다양성 중요
- 메타 학습기 훈련 시 교차 검증 폴드
- 메타 학습기 복잡도: 일반적으로 간단(로지스틱 회귀)

### 모범 사례

**다양성:**
- 다양한 알고리즘 사용(트리, SVM, 신경망)
- 하이퍼파라미터 변화
- 다양한 피처 부분집합
- 다양한 데이터 전처리

**검증:**
- 안정적인 추정을 위한 교차 검증
- OOB(Out-of-bag) 오류 모니터링
- 최종 평가를 위한 별도의 테스트 세트

**계산 효율성:**
- 독립 학습기 병렬화(배깅)
- 스태킹의 캐시 예측
- 효율적인 구현 사용(XGBoost, LightGBM)

## 비교

| 방법 | 속도 | 해석 가능성 | 병렬화 | 편향/분산 |

## 실무의 앙상블

**앙상블 사용 시기:**
- 정확도가 가장 중요
- 계산 리소스 가용
- 데이터셋이 중규모-대규모
- 증가된 모델 복잡도 허용 가능

**표준 접근법:**
- 기본값: 랜덤 포레스트(빠름, 효과적, 간단)
- 최대 정확도: 그래디언트 부스팅(XGBoost, LightGBM)
- 안전성: 투표 앙상블(여러 입증된 방법 사용)

**수확 체감:**
- 개별 학습기 정확도 평탄화가 먼저 발생
- 앙상블 개선은 더 오래 지속
- 배깅의 전형적인 최적점: 50-100개 트리
- 부스팅의 전형적인 최적점: 100-1000 반복

## 요약

앙상블 방법은 예측 접근 방식의 기본적 변화를 나타냅니다: 단일한 완벽한 모델을 추구하는 대신 군중의 지혜를 활용합니다. 다양한 기본 학습기의 신중한 설계와 지능적인 결합 전략을 통해 앙상블은 분류, 회귀, 순위 매김 작업에서 놀라운 성능을 달성합니다. 이들은 실무 머신러닝 성공의 중심에 남아 있습니다.

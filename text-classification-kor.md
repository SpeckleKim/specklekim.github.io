# 텍스트 분류

## 개요

텍스트 분류는 미리 정의된 범주 또는 레이블을 텍스트 문서에 할당하는 작업입니다. 감정 분석, 스팸 탐지, 주제 분류 및 의도 인식을 포함한 광범위한 응용 프로그램을 포함하는 자연어 처리의 가장 기본적인 문제 중 하나입니다. 목표는 텍스트 입력에서 이산 클래스 레이블로의 매핑을 학습하는 것입니다.

## 단어 주머니(Bag-of-Words, BoW)

단어 주머니 모델은 문법과 단어 순서를 무시하고 텍스트를 정렬되지 않은 단어 컬렉션으로 나타냅니다. 각 문서는 벡터로 표현되며, 각 차원은 어휘의 고유 단어에 해당하고 값은 해당 문서에서 단어의 빈도를 나타냅니다.

**장점:**
- 단순하고 해석 가능
- 계산이 빠름
- 많은 분류 작업에서 합리적으로 작동

**단점:**
- 단어 순서 정보 손실
- "좋은 나쁘지 않은"과 "나쁜 좋지 않은"을 동일하게 처리
- 단어 간의 의미 관계 무시
- 매우 자주 나타나는 일반 단어에 민감

## TF-IDF (용어 빈도-역 문서 빈도)

TF-IDF는 용어의 중요도를 기반으로 가중치를 부여하여 단어 주머니를 개선합니다. 용어의 TF-IDF 점수는 문서에 자주 나타나지만 전체 코퍼스에서는 드물게 나타나면 높으며, 일반 단어의 영향을 줄입니다.

공식은 다음과 같습니다: TF-IDF(t,d) = TF(t,d) × IDF(t)

여기서:
- TF(t,d) = 문서 d에서 용어 t의 빈도
- IDF(t) = log(전체 문서 / t를 포함하는 문서)

이 가중치 지정 방식은 드물고 특별한 단어가 일반 단어보다 분류에 더 정보를 제공한다는 직관을 캡처합니다.

## 텍스트 분류를 위한 CNN

합성곱 신경망은 단어 임베딩에 1D 합성곱을 적용하여 텍스트에 맞게 조정될 수 있습니다. 전형적인 아키텍처는 다음을 포함합니다:

1. **임베딩 계층**: 단어를 조밀한 벡터로 매핑(사전 훈련된 단어 임베딩 또는 학습)
2. **합성곱 계층**: n-그램 패턴을 캡처하기 위해 다양한 너비의 필터 적용
3. **최대 풀링**: 각 필터에 대해 가장 중요한 특성 추출
4. **완전 연결 계층**: 풀된 특성을 클래스 로짓에 매핑

CNN은 로컬 단어 패턴을 효율적으로 캡처하며 계산 효율적입니다. 문서 클래스를 나타내는 핵심 구문과 패턴을 식별하는 데 탁월합니다.

## 텍스트 분류를 위한 RNN/LSTM

순환 신경망은 이전 토큰의 정보를 축적하는 숨겨진 상태를 유지하면서 수열을 반복적으로 처리합니다. 장단기메모리(LSTM) 단위는 사라지는 그래디언트 문제를 해결하여 네트워크가 장거리 의존성을 캡처하도록 합니다.

전형적인 RNN/LSTM 분류기:
1. 단어를 벡터에 임베드
2. LSTM 셀을 통해 수열 처리
3. 최종 숨겨진 상태 또는 풀된 표현을 분류에 사용
4. 완전 연결 계층을 통해 클래스 확률 생성

LSTM은 단어 순서와 장거리 의미 의존성을 캡처하며 문맥이 중요한 많은 데이터셋에서 CNN을 능가합니다.

## 텍스트 분류를 위한 BERT 미세 조정

BERT(양방향 인코더 표현 트랜스포머)는 양방향 트랜스포머에서 상황화된 단어 표현을 제공함으로써 NLP를 혁명화했습니다. 분류 작업의 경우 표준 접근 방식은 다음과 같습니다:

1. **토큰화**: 텍스트를 BERT 토큰으로 분해
2. **분류 토큰 추가**: [CLS] 토큰 추가
3. **BERT 실행**: 사전 훈련된 계층을 통해 전달
4. **표현 추출**: [CLS] 토큰의 표현 사용
5. **미세 조정**: 분류 헤드를 추가하고 하류 작업에서 훈련

이 접근 방식은 대규모 텍스트 코퍼스에서 사전 훈련 중에 학습된 풍부한 상황화된 표현을 활용하여 최첨단 성능을 달성합니다.

## 다중 레이블 분류

다중 레이블 분류는 단일 레이블을 할당하는 이진 또는 다중 클래스 분류와 달리 각 문서에 여러 레이블을 할당합니다. 예를 들어, 뉴스 기사는 "정치"와 "경제" 모두로 레이블이 지정될 수 있습니다.

주요 차이점:
- **손실 함수**: 범주형 교차 엔트로피 대신 이진 교차 엔트로피 사용
- **평가**: 각 레이블당 정밀도, 재현율 및 F1 계산
- **임계값**: 종종 확률 임계값(예: 0.5)이 필요하여 예측을 이진화

다중 레이블 문제는 레이블 의존성과 클래스 불균형을 신중하게 처리해야 합니다.

## 감정 분석

감정 분석은 텍스트가 긍정, 부정 또는 중립 감정을 표현하는지 결정합니다. 브랜드 인식을 모니터링하고, 고객 리뷰를 분석하고, 여론을 추적하는 데 널리 사용됩니다.

**과제:**
- 풍자와 아이러니는 종종 감정을 역전시킴
- 문맥 의존적 감정(기능은 음수 또는 양수일 수 있음)
- 단일 문서 내의 혼합 감정
- 도메인 적응(리뷰에서 훈련된 모델은 트윗에서 실패할 수 있음)

현대적 접근 방식은 감정 데이터셋에서 미세 조정된 BERT 및 유사한 트랜스포머를 사용하여 표준 벤치마크에서 90% 이상의 정확도를 달성합니다.

## 스팸 탐지

스팸 분류는 원하지 않거나 악의적인 메시지(이메일, SMS, 소셜 미디어 게시물)를 식별하려고 합니다. 이것은 중요한 실제 영향을 미치는 이진 분류 작업입니다.

**접근 방식:**
- **전통적**: TF-IDF + 로지스틱 회귀 또는 SVM
- **딥 러닝**: LSTM 또는 BERT 기반 분류기
- **앙상블**: 견고성을 위해 여러 모델 결합

주요 특성은 발신자 평판, 메시지 콘텐츠(의심스러운 링크, 일반적인 스팸 구문) 및 사용자 행동 패턴을 포함합니다.

## 주제 분류

주제 분류는 문서를 미리 정의된 주제 범주(뉴스 주제, 제품 범주 등)에 할당합니다. 감정은 감정 콘텐츠에 관한 것과 달리 주제 분류는 의미론적 콘텐츠에 집중합니다.

**접근 방식:**
- **감독형**: 레이블이 지정된 문서에서 분류기 훈련(CNN, LSTM, BERT)
- **비감독형**: 주제 모델링 사용(잠재 디리클레 할당)
- **제로샷**: 프롬프트 기반 분류와 함께 사전 훈련된 모델 사용

주제 분류는 콘텐츠 구성, 추천 시스템 및 콘텐츠 중재의 기초입니다.

## 결론

텍스트 분류는 단순 통계 방법(BoW, TF-IDF)에서 복잡한 딥 러닝 접근 방식(BERT, 트랜스포머)에 이르는 다양한 기술을 포함합니다. 방법 선택은 사용 가능한 데이터, 계산 자원 및 작업 요구 사항에 따라 달라집니다. 현대적 접근 방식은 최첨단 결과를 위해 사전 훈련된 트랜스포머를 활용하는 반면, 전통적 방법은 자원 제약 시나리오에서 가치 있게 유지됩니다. 기술의 전체 스펙트럼을 이해하면 실무자들이 특정 분류 도전 과제에 적절한 도구를 선택할 수 있습니다.

# 모델 기반 강화학습: 학습 모델로 계획하기

## 소개

모델 기반 RL은 에이전트가 계획할 수 있게 하는 환경 모델을 학습합니다. 경험에서 직접 학습하는 모델 프리 방법과 달리 모델 기반 접근법은 학습 모델과 함께 계획 알고리즘(탐색, 동적 계획)을 사용합니다. 이 패러다임은 표본 효율성, 해석 가능성, 전이 학습 기회를 제공하되 모델 학습 복잡성이 비용입니다.

## 월드 모델

**핵심 개념:** 환경이 현재 상태와 행동이 주어질 때 어떻게 진화하는지 설명하는 동학 모델을 학습합니다.

**표현:** 다음일 수 있습니다:
- **잠재 모델:** 관찰을 학습된 잠재 공간으로 압축
- **픽셀 모델:** 다음 프레임을 직접 예측
- **확률적 모델:** 행동/환경 확률성 포착
- **결정론적 모델:** 결정론적 동학 가정

## 모델 학습

**요소:**

1. **상태 인코더:** 관찰을 상태 표현으로 압축
2. **동학 예측기:** p(s_{t+1}|s_t, a_t) 모델
3. **보상 모델:** r(s_t, a_t) 예측
4. **종료 모델:** 에피소드 종료 예측

**학습 목표:**
```
L = ||s_{t+1} - f(s_t, a_t)||² + ||r - r_model(s_t, a_t)||²
```

**도전:**
- 누적 오차: 모델 오차는 긴 지평선에서 전파
- 분포 외: 계획이 학습 중 드물게 보인 상태 선택 가능
- 확률성: 미래에 대한 불확실성이 모델링하기 어려움

## 계획 방법

**모델 예측 제어 (MPC):**
```
각 타임스텝에 대해:
  각 행동 궤적에 대해:
    모델을 사용한 H 스텝 시뮬레이션
    누적 보상 평가
  최적 행동 궤적 실행
```

근시안적이지만 짧은 지평선에 대해 계산적으로 실행 가능합니다.

**교차 엔트로피 방법:** 행동 분포를 반복적으로 정제합니다.

**궤적 최적화:** 행동 수열의 기울기 기반 최적화.

## Dreamer 아키텍처

최근 최첨단 모델 기반 접근:

**요소:**
1. **월드 모델:** 확률적 변수를 포함한 VAE 기반 잠재 동학
2. **액터:** 상상된 반환을 최대화하는 정책 네트워크
3. **크리틱:** 상상 공간의 가치 네트워크
4. **상상 롤아웃:** 학습된 잠재 공간에서 궤적 계획

**핵심 통찰:** 상상 내에서 정책을 학습(잠재 공간)하고 픽셀 수준 모델 오차를 무시합니다.

**학습 루프:**
```
1. 실제 세계 경험 수집
2. 경험에 대한 월드 모델 학습
3. 월드 모델을 사용한 궤적 상상
4. 상상된 궤적에서 액터/크리틱 학습
5. 실제 환경에서 정책 실행
6. 반복
```

**장점:**
- 상상 루프가 정책 학습과 모델 오차 분리
- 표본 효율적: 한 실제 궤적이 많은 상상된 것을 생성
- 확장 가능: 잠재 공간이 차원을 감소
- 안정적: 명시적 MPC 없는 순수하게 학습된 목표

## MuZero 아키텍처

명시적 환경 모델 없는 AlphaZero 스타일 계획:

**핵심 혁신:** 가치 동등 모델을 학습—상태가 아닌 보상과 가치를 모델링합니다.

**요소:**
1. **표현:** 관찰을 잠재 상태로 인코딩
2. **예측:** 잠재에서 가치, 보상, 정책 예측
3. **동학:** 상태를 재구성하지 않고 잠재 공간에서 전이

**장점:** 계획에 관련된 것(가치, 보상)만 모델링하고 무관한 세부사항 무시.

**응용:**
- Atari 게임: 최첨단 성능
- 보드 게임: 바둑, 체스에서 인간 초월 성능
- 일반 도메인: 다양한 환경에 효과적

## 상상 기반 학습

**핵심 루프:**
1. **실제 경험:** 실제 환경에서 궤적 수집
2. **모델 학습:** 동학, 보상, 가치 모델 업데이트
3. **상상:** 학습 모델을 사용한 궤적 계획
4. **정책 학습:** 상상된 궤적에서 정책 학습

**장점:**
- 대규모 표본 효율성: 적은 실제 전이, 많은 상상
- 안정성: 상상의 가치 목표는 TD보다 더 안정적
- 전이: 학습 모델이 작업 간 전이 가능

## 불확실성 정량화

모델 불확실성 처리가 강건성 개선:

**앙상블 방법:**
- 여러 모델 학습
- 불일치를 불확실성 추정으로 사용
- 불확실한 영역을 향한 탐험 권장

**베이지안 접근:**
- 불확실성을 가진 확률 모델
- 가중치에 대한 후 분포
- 불확실성에서 Thompson 샘플링

## 모델 학습 도전

**누적 오차:**
```
t+1 단계 오차는 단계 t 오차에 따라 결정
장기간 오차는 지수적으로 증가
```

완화: 짧은 상상 지평선, 빈번한 모델 재학습.

**분포 외 시프트:**
```
계획이 학습 데이터에 없는 상태 선택 가능
모델 정확도가 학습 분포 외에서 감소
```

완화: 앙상블 불확실성, 제약 최적화.

**고차원 관찰:**
```
픽셀 수준 모델링 어려움
확장성을 위해 잠재 모델 필요
```

## 모델 기반 + 모델 프리 하이브리드

두 패러다임의 강점 결합:

**MBPO (모델 기반 정책 최적화):**
- 실제 경험에서 동학 모델 학습
- 모델을 사용한 허수 전이 생성
- 혼합 실제/허수 데이터에 대해 모델 프리 알고리즘(SAC)으로 정책 학습
- 모델 프리 안정성 유지, 모델의 표본 효율성 획득

**장점:**
- 순수 모델 프리보다 더 나은 표본 효율성
- 순수 모델 기반보다 더 안정적
- 패러다임 간 실용적 트레이드오프

## 모델 기반 RL의 장점

**표본 효율성:** 더 적은 실제 상호작용에서 학습.

**계획 능력:** 즉각적 행동 이상으로 미래에 대해 신중하게 고민.

**전이 학습:** 한 작업에서 학습한 모델이 관련 작업으로 전이.

**해석 가능성:** 예측이 학습된 동학에 대한 통찰 제공.

**오프 정책 학습:** 정책 제약 없이 과거 데이터로 계획.

## 한계

**모델 복잡성:** 특히 고차원 공간에서 정확한 모델 학습이 도전적.

**계산 비용:** 계획은 종종 모델 프리 추론보다 비용.

**상상 발산:** 모델 오차는 계획 중에 축적.

**제한된 탐험:** 순수 계획은 중요한 탐험 영역을 놓칠 수 있음.

## 현대 접근법

**확산 기반 모델:** RL의 수열 모델링을 위한 확산 사용.

**후속 모델:** 추상 후속 표현 학습.

**잠재 월드 모델:** 압축 관찰 공간에서 학습.

**하이브리드 접근:** 모델 기반과 모델 프리를 지속적으로 결합.

## 실무 고려사항

**모델 기반 사용 시점:**
- 표본 효율성이 중요함 (로봇공학, 실제 작업)
- 환경 동학이 학습 가능 (물리 시뮬레이션)
- 계획과 해석 가능성 필요
- 관련 작업으로의 전이 중요

**모델 프리 사용 시점:**
- 풍부한 데이터 사용 가능
- 환경이 매우 확률적/복잡
- 빠른 학습 필요 (대규모 도메인)
- 상태 표현이 데이터에서 쉽게 학습됨

## 결론

모델 기반 RL은 학습 모델로 계획을 가능하게 하여 표본 효율성과 해석 가능성 장점을 제공합니다. Dreamer와 MuZero는 모델 학습과 정책 최적화를 효과적으로 균형있게 유지하는 현대 접근법을 나타냅니다. 모델 기반과 모델 프리를 결합하는 하이브리드 방법이 점점 더 매력적이 됩니다. 표본 효율성과 실제 적용성이 증가하면서 모델 기반 RL은 실무 응용에 점점 더 관련성이 높아집니다.

# 편향-분산 트레이드오프

편향-분산 트레이드오프는 머신러닝에서 가장 기본적인 개념 중 하나입니다. 때때로 단순한 모델이 복잡한 모델보다 더 잘 일반화되는 이유를 설명하고, 예측 오류의 원인을 이해하기 위한 틀을 제공합니다.

## 편향: 체계적 오류

**편향**은 모델의 예측치의 기댓값과 우리가 예측하려는 정확한 값 사이의 차이입니다. 모델의 예측이 평균적으로 얼마나 벗어나 있는지를 측정합니다.

수학적으로, 목표값 y와 모델 예측 ŷ에 대해:
```
편향 = E[ŷ - y]
```

편향은 기본 문제에 대한 잘못된 가정에서 비롯됩니다:
- 비선형 데이터로 학습한 선형 모델은 높은 편향을 갖습니다
- 참 관계가 복잡할 때 너무 적은 특성을 사용하면 높은 편향이 발생합니다
- 과소적합(모델이 너무 단순함)은 높은 편향으로 이어집니다

**낮은 편향 모델**은 복잡한 관계를 포착할 수 있을 정도로 유연합니다. 예시:
- 높은 차수의 다항식
- 깊은 신경망
- 많은 분할이 있는 결정 트리

## 분산: 데이터에 대한 민감도

**분산**은 다른 데이터셋에서 학습할 때 모델의 예측이 얼마나 변하는지를 측정합니다. 높은 분산은 모델이 특정 훈련 데이터에 매우 민감함을 의미합니다.

수학적으로:
```
분산 = E[(ŷ - E[ŷ])²]
```

분산은 다음의 경우에 증가합니다:
- 모델이 훈련 데이터에 비해 많은 파라미터를 가질 때
- 모델이 훈련 세트의 특정 패턴을 암기할 때
- 과적합(모델이 데이터에 비해 너무 복잡함)

**높은 분산 모델**은 훈련 데이터에는 매우 잘 맞지만 새로운 데이터에서 실패합니다:
- 작은 데이터셋의 매우 깊은 신경망
- 높은 차수의 다항식 적합
- 가지치기되지 않은 결정 트리
- 많은 하이퍼파라미터를 가지고 크게 튜닝된 모델

## 분해: 전체 오류

모든 모델의 기대 제곱 오류(손실)는 다음과 같이 분해될 수 있습니다:

```
E[(y - ŷ)²] = 편향² + 분산 + 기약적 오류
```

이것이 편향-분산 분해입니다. 각 항은 예측 오류의 다른 원인을 나타냅니다:

1. **편향²**: 과도하게 단순화된 가정에서의 오류
2. **분산**: 훈련 데이터 변동에 대한 민감성으로 인한 오류
3. **기약적 오류** (베이즈 오류): 목표의 무작위 노이즈, 감소시킬 수 없음

핵심 통찰: **편향과 분산을 동시에 최소화할 수 없습니다**. 항상 트레이드오프가 있습니다.

## 과소적합 vs 과적합

이 개념들은 편향과 분산과 직접 관련됩니다:

**과소적합 (높은 편향, 낮은 분산):**
- 모델이 문제에 비해 너무 단순함
- 예측이 체계적으로 일관되게 잘못됨
- 훈련 오류와 검증 오류 모두 높음
- 단순 모델이 노이즈에 적응하지 않으므로 낮은 분산

**최적 모델 (균형 잡힌 편향 & 분산):**
- 모델 복잡도가 데이터 복잡도와 일치
- 훈련 및 검증 오류 모두 합리적으로 낮음
- 전체 오류 최소화

**과적합 (낮은 편향, 높은 분산):**
- 모델이 데이터에 비해 너무 복잡함
- 훈련 데이터에 완벽하게 맞지만 새로운 데이터에서 실패
- 훈련과 검증 오류 사이의 큰 격차
- 모델이 훈련 세트의 노이즈에 적응하므로 높은 분산

## 모델 복잡도 곡선

모델 복잡도와 전체 오류 간의 관계는 U자 모양 곡선으로 시각화됩니다:

```
        오류
          ^
          |     과적합
          |    /
          |   /    검증 오류
      최소 |  /
          | /
          |/__________
          |  \    훈련 오류
          |   \
          |    \
          +-----------> 모델 복잡도
      낮음              높음
```

- **왼쪽**: 과소적합 영역. 복잡도가 증가하면 편향이 높으므로 훈련 및 테스트 오류 모두 감소
- **아래**: 최적 지점. 오류가 최소화됨
- **오른쪽**: 과적합 영역. 훈련 오류는 감소하지만 분산이 지배적이므로 테스트 오류 증가

최적 복잡도는 검증 오류가 최소화되는 지점입니다.

## 실무 적용

### 진단 방법

**높은 편향 (과소적합):**
- 확인: 훈련 오류와 검증 오류 모두 높음
- 두 오류 모두 비슷함 (격차 없음)
- 모델이 너무 단순함

**높은 분산 (과적합):**
- 확인: 훈련 오류는 낮지만 검증 오류는 높음
- 훈련과 검증 오류 사이의 큰 격차
- 모델이 너무 복잡함

### 높은 편향 해결 방법
- 모델 복잡도 증가
- 더 많은 특성 추가
- 더 오래 훈련 (신경망의 경우)
- 더 표현력 있는 모델 클래스 사용

### 높은 분산 해결 방법
- 모델 복잡도 감소
- 더 많은 훈련 데이터 수집
- 정규화 적용 (L1, L2, 드롭아웃)
- 앙상블 방법 사용
- 특성 선택으로 차원성 감소

## 데이터 크기의 영향

모델 복잡도는 데이터셋 크기와 함께 조정되어야 합니다. 제한된 데이터로 인해, 복잡한 모델도 과적합으로 인해 높은 분산을 갖습니다. 풍부한 데이터로 인해, 복잡한 모델을 정당화할 수 있습니다.

이것이 전이 학습이 작동하는 이유입니다: 큰 데이터셋에서의 사전훈련 후 작은 데이터셋에서의 미세조정은 낮은 분산 학습 표현으로부터 시작하여 분산을 감소시킵니다.

## 예: 다항식 적합

증가하는 차수의 다항식을 데이터에 적합시키는 것은 편향-분산 트레이드오프를 잘 보여줍니다:

- **차수 1** (선형): 높은 편향, 낮은 분산. 비선형성이 있으면 선형 데이터도 과소적합
- **차수 3**: 균형 잡힘. 노이즈를 암기하지 않고 참 패턴 포착
- **차수 10**: 낮은 편향, 높은 분산. 모든 점을 통과하며 노이즈에 과적합

차수 3 다항식은 훈련 데이터에 차수 10만큼 잘 맞지 않음에도 불구하고 종종 가장 잘 일반화됩니다.

## 교차검증과 트레이드오프

교차검증은 편향-분산 트레이드오프를 다음을 통해 탐색하는 데 도움을 줍니다:
- **분산 추정**: CV 폴드 점수가 광범위하게 변하면, 모델의 분산이 높습니다
- **과적합 감지**: 훈련 정확도 >> CV 정확도이면 과적합(높은 분산)이 발생 중입니다
- **모델 비교**: CV는 전체 오류를 최소화하는 복잡도 수준을 식별하는 데 도움을 줍니다

## 핵심 요점

1. 모든 오류 원인은 편향, 분산 또는 기약적 노이즈에서 비롯됩니다
2. 복잡도 추가는 편향을 감소시키지만 분산을 증가시킵니다
3. 최적 모델은 훈련 데이터에 가장 잘 맞는 모델이 거의 아닙니다
4. 검증/테스트 성능이 훈련 성능보다 더 중요합니다
5. 모델 복잡도를 데이터 양과 문제 복잡도에 일치시킵니다
6. 정규화는 모델 용량을 줄이지 않으면서 분산을 감소시키는 주요 도구입니다

편향-분산 트레이드오프는 머신러닝에서 "더 복잡함 = 더 좋음"이 거짓임을 설명합니다. 최고의 모델은 단순성과 표현력을 균형 잡힙니다.

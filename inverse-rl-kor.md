# 역강화학습: 시연에서 보상 학습하기

## 소개

역강화학습(IRL)은 전형적인 RL 문제를 역전시킵니다: 전문가 행동의 시연이 주어질 때, 보상 함수를 추론합니다. 보상 함수를 수작업으로 엔지니어링하는 대신(종종 어렵고 오류 발생), IRL은 인간 전문성에서 학습합니다. 응용은 자율주행, 로봇 조작, 모방 학습을 포함합니다.

## 문제 정식화

**표준 RL:** 보상 함수 R을 최대화하는 정책 π를 찾습니다.

**IRL:** 전문가 시연 D가 주어질 때, 전문가 정책이 최적화하는 보상 함수 R을 추론합니다.

**도전:** 여러 보상 함수가 관찰된 행동을 설명할 수 있습니다. IRL은 시연과 일치하는 보상을 찾으며, 이상적으로 새로운 상황으로 일반화합니다.

## 최대 엔트로피 IRL

**핵심 원칙:** 시연과 일치하는 모든 보상 함수 중에서 최대 엔트로피 정책을 선택합니다.

**직관:** 전문가가 단일 보상 함수를 최적화한다고 가정하지 마십시오; 관찰된 행동이 충분히 좋은 정책 간에 무작위로 분배된다고 가정합니다.

**수학적 프레임워크:**
```
max_R H(π_R) subject to:
E_π_R[f(s,a)] = E_π_E[f(s,a)]
여기서 f는 특징 기대값을 나타냅니다
```

**특징 매칭:**
전문가 정책과 학습 정책은 동일한 예상 특징 수를 가집니다.

**알고리즘:**
```
1. 보상 함수 초기화
2. 현재 보상에서 최적 정책 계산
3. 두 정책 모두에 대한 특징 기대값 계산
4. 보상을 업데이트하여 특징 매칭 증가
5. 수렴할 때까지 반복
```

## 도제 학습

**아이디어:** 선형 특징 조합이라고 가정하고 보상 함수 학습:
```
R(s,a) = w^T φ(s,a)
```

**과정:**
1. 시연에서 특징 표현 φ 추출
2. 전문가 정책이 E[w^T φ]를 최대화하도록 가중치 벡터 w 찾기
3. 선형 계획법을 사용한 최적 w 찾기
4. 정책이 편차가 있으면 더 많은 예제 추가, 반복

**최대 마진 학습과의 연결:** 서포트 벡터 방법과 유사합니다.

## GAIL (생성적 적대적 모방 학습)

적대적 학습을 사용한 현대 접근:

**핵심 아이디어:** 전문가 vs 학습된 궤적을 구분하는 판별기를 속이도록 정책 학습.

**설정:**
- **생성기:** 궤적을 생성하는 정책 π_θ
- **판별기:** 궤적을 전문가 또는 학습된 것으로 분류

**목표:**
```
min_θ max_D E_expert[log D(τ)] + E_π_θ[log(1-D(τ))]
```

**장점:**
- 명시적 보상 함수 불필요
- 원시 관찰에서 직접 학습
- 적대적 프레임워크로 안정적인 학습
- 행동 복제 대비 표본 효율적

**핵심 통찰:** 판별기가 암시적으로 보상 함수를 학습하고 정책이 전문가 행동과 일치하도록 학습합니다.

## 행동 복제 기준

**가장 단순한 접근:** 시연에서의 감독 학습.
```
min_θ ||π_θ(a|s) - π_expert(a|s)||²
```

**한계:**
- 분포 시프트: 전문가의 상태 분포에서만 학습
- 누적 오차: 작은 실수가 축적
- 탐험 부족
- 새로운 상태로의 나쁜 일반화

**언제 작동하는가:** 명확한 행동 패턴의 단순 작업.

## 비교 분석

**IRL vs 행동 복제:**
- IRL은 기본 보상 구조 추론
- BC는 행동을 직접 모방
- IRL이 분포 시프트에 더 강건
- BC가 단순 작업에 더 빠름

**IRL vs 강화학습:**
- IRL은 시연에서 학습
- RL은 희박/형성 보상 신호에서 학습
- IRL은 보상 설계가 어려울 때 유용
- RL은 명확한 목표에 효과적

## 응용

**자율주행:**
- 전문가 인간 운전자로부터 학습
- 안전 고려사항, 편의성 선호도 추론
- 새로운 도로/조건으로 일반화

**로봇 조작:**
- 원격 조종 시연에서 학습
- 명시적 보상 이상의 작업 목표 이해
- 새로운 시나리오로 전이

**선호도 학습:**
- 행동에서 사용자 선호도 추론
- 명시적 지시 없는 개인화

**안전 중요 시스템:**
- 인간 가치/선호도 학습
- 인간 우려를 존중하는 자율 시스템 구축

## 실무 도전

**시연 품질:** 전문가 시연이 좋아야 합니다. 나쁜 시연이 학습을 오도합니다.

**특징 엔지니어링:** 좋은 특징 표현이 IRL 성공에 필수입니다.

**확장성:** 고차원 문제에 대한 최적 정책 계산이 비용이 높습니다.

**모호성:** 많은 보상 함수가 동일한 행동을 설명할 수 있으며 다른 가중치를 학습할 수 있습니다.

## 고급 주제

**베이지안 IRL:** 보상에 대한 후 분포. 모호성을 포착합니다.

**깊은 IRL:** 신경망이 원시 관찰에서 보상 함수를 직접 학습합니다.

**다중 목표 IRL:** 시연에서 여러 경합하는 목표를 학습합니다.

**대화형 IRL:** 에이전트가 학습 중에 명확화를 위해 인간에게 쿼리합니다.

## 심층 학습 통합

**DeepIRL:** 보상 함수에 신경망 사용:
```
R_θ(s,a) = neural_net_θ(s,a)
```

고차원 관찰에서 직접 학습합니다. 종단간 학습을 가능하게 합니다.

**정책 학습과의 통합:**
- 시연을 사용하여 보상과 정책을 동시에 학습
- 반복적 정제

## 윤리적 고려사항

**가치 학습:** IRL이 인간 가치를 학습하며 정렬을 보장해야 합니다.

**편향 증폭:** 시연이 편향을 포함하면 학습된 보상이 이를 영속화합니다.

**목표 명시:** 시스템이 실제로 최적화하는 목표는 무엇입니까?

**투명성:** 추론된 보상이 해석 가능해야 합니다.

## 모방 학습과의 연결

**모방 학습 스펙트럼:**
1. **행동 복제:** 직접 행동 복사
2. **IRL → RL:** 보상 추론, 정책 최적화
3. **GAIL:** 명시적 보상 없는 적대적 모방
4. **온라인 학습:** 학습 중 전문가에게 쿼리

**트레이드오프:**
- 행동 복제: 빠르고 단순하지만 일반화 안 됨
- GAIL: 유연하고 안정적이지만 더 복잡
- IRL: 해석 가능한 보상이지만 계산 비용 높음

## 최근 진전

**오프라인 IRL:** 환경 상호작용 없이 고정 시연 데이터셋에서 학습.

**선호도 학습:** 전체 시연이 아닌 쌍별 비교에서 학습.

**메타 IRL:** 작업 분포에서 보상을 추론하는 방법 학습.

**불확실성 인식 IRL:** 추론된 보상의 모호성 정량화.

## 결론

역강화학습은 시연에서 보상 함수를 학습하는 프레임워크를 제공하여 보상 명시 문제를 해결합니다. 최대 엔트로피 IRL은 원칙적 접근을 제공하고 GAIL은 실무적이고 효율적인 학습을 제공합니다. 이 방법들은 에이전트가 인간 전문성에서 학습할 수 있게 하여 안전 중요 및 선호도 민감 응용에 중요합니다. AI 시스템이 더 자율적이 되면서 IRL을 통한 인간 가치 학습이 점점 더 중요해집니다.

# 액터-크리틱 방법: 가치와 정책 학습 결합

## 소개

액터-크리틱 방법은 정책 기반 및 가치 기반 접근법을 결합합니다. 액터(정책)는 행동을 결정하고 크리틱(가치 함수)은 이를 평가합니다. 이 조합은 분산 감소와 정책 최적화를 균형있게 유지하여 다양한 도메인에서 안정적이고 효율적인 학습을 가능하게 합니다.

## 액터-크리틱 프레임워크

**두 가지 요소:**

1. **액터:** 행동을 선택하는 정책 π_θ(a|s)
2. **크리틱:** 가치를 추정하는 가치 함수 V_ϕ(s) 또는 Q_ϕ(s,a)

**장점:** 크리틱이 분산 감소를 위한 기준선을 제공하면서 액터는 크리틱의 추정을 사용하여 정책을 최적화합니다.

**정보 흐름:**
```
상태 → 액터 → 행동 → 환경
        ↓
    크리틱이 행동 평가
        ↓
    신호로 TD 오차 제공
```

## 정책 업데이트

액터는 크리틱의 가치 추정을 사용합니다:

```
∇_θ J(θ) = E[∇_θ log π_θ(a|s) Q_ϕ(s,a)]
```

전체 궤적을 기다리는 대신(REINFORCE) 크리틱의 Q값 추정을 사용합니다. 분산을 줄이고 저분산 업데이트를 가능하게 합니다.

## 가치 함수 업데이트

크리틱은 시간차에서 학습합니다:

```
δ_t = r_t + γV_ϕ(s_{t+1}) - V_ϕ(s_t)
ϕ ← ϕ - β∇_ϕ [V_ϕ(s_t) - (r_t + γV_ϕ(s_{t+1}))]²
```

TD 오차 δ_t는 이중 목적을 수행합니다: 크리틱 업데이트와 액터 우위 추정.

## 우위 추정

TD 오차를 우위로 사용:

```
Â_t = δ_t = r_t + γV_ϕ(s_{t+1}) - V_ϕ(s_t)
```

액터 업데이트는:

```
θ ← θ + α ∇_θ log π_θ(a_t|s_t) δ_t
```

REINFORCE 대비 저분산이면서도 효율적.

## A2C (우위 액터-크리틱)

배치 업데이트를 사용한 동기식 버전:

**알고리즘:**
```
각 배치에 대해:
  현재 정책 사용 전이 수집
  가치 목표 계산: V_target = r + γV_ϕ(s')
  우위 계산: Â = V_target - V_ϕ(s)

  액터 손실: -log π_θ(a|s) × Â
  크리틱 손실: (V_ϕ(s) - V_target)²

  두 네트워크 업데이트
```

**특성:**
- 동기식: 병렬 환경, 동기 업데이트
- 효율적: 병렬 데이터 수집 활용
- 안정적: 우위 절단과 가치 손실이 학습을 안정화
- 실무적: 좋은 표본 효율성과 벽시계 시간

## A3C (비동기 우위 액터-크리틱)

비동기 병렬 버전:

**핵심 혁신:** 여러 병렬 워커가 경험을 비동기로 수집합니다.

**아키텍처:**
- 전역 정책 및 가치 네트워크
- 로컬 네트워크를 가진 워커 스레드
- 전역 네트워크에 대한 비동기 기울기 누적

**장점:**
- 자연스러운 병렬화: 여러 워커가 독립적으로 탐험
- 온 정책: 각 워커는 가장 최신 정책으로 작동
- 감소된 메모리: 경험 재생 불필요
- 탐험: 다양한 워커 경험이 범위를 개선

**단점:**
- 복잡한 구현
- 비동기성이 수렴을 해칠 수 있음
- 오프 정책 수정 어려움

## SAC (소프트 액터-크리틱)

견고성을 위한 엔트로피 정규화 접근:

**핵심 통찰:** 예상 반환을 최대화하면서 탐험을 권장하도록 엔트로피도 최대화:

```
J(π) = E[r(s,a) + αH(π(·|s))]
```

여기서 H는 정책 엔트로피, α는 탐험 강도를 제어하는 온도입니다.

**메커니즘:**
1. 소프트 가치 함수가 엔트로피를 통합: V_soft = Q - α log π
2. 소프트 Q값을 최대화하도록 정책 학습
3. Q 네트워크가 소프트 벨만 목표로 업데이트

**장점:**
- 엔트로피 정규화를 통한 더 안정적인 학습
- 더 나은 탐험-이용 균형
- 하이퍼매개변수에 강건
- 연속 제어에 효과적

## TD3 (쌍둥이 지연 심층 결정론적 정책 기울기)

결정론적 정책 기울기의 과대 추정 해결:

**세 가지 핵심 개선:**

1. **쌍둥이 Q 네트워크:** 두 개의 Q 네트워크 사용, 목표에서 최솟값 사용
   ```
   Target = r + γ min(Q_ϕ1(s',π_θ(s')), Q_ϕ2(s',π_θ(s')))
   ```

2. **지연 정책 업데이트:** 가치 네트워크보다 덜 자주 정책 업데이트
   ```
   매 스텝 Q 업데이트; 매 d 스텝마다 π와 목표 네트워크 업데이트
   ```

3. **목표 정책 스무딩:** 목표 정책 행동에 노이즈 추가
   ```
   목표 행동 = π_θ(s') + ε, 여기서 ε ~ clip(N(0,σ), -c, c)
   ```

**결과:** 크게 감소된 과대 추정, 더 안정적인 학습.

## 오프 정책 액터-크리틱

A2C는 온 정책이지만 많은 액터-크리틱 방법은 오프 정책을 사용합니다:

**DDPG (심층 결정론적 정책 기울기):**
- 연속 행동을 위한 결정론적 정책
- 경험 재생 버퍼
- 안정성을 위한 목표 네트워크
- 부최적 경험에서 학습 가능

**오프 정책 장점:**
- 더 나은 표본 효율성
- 이전에 수집한 데이터에서 학습 가능
- 더 효율적인 탐험 지원

## 안정성 고려사항

**크리틱 수렴:** 신뢰할 수 있는 기울기 추정을 위해 크리틱은 액터보다 빠르게 수렴해야 합니다.

**학습 속도 비율:** 일반적으로 α_critic > α_actor로 크리틱이 따라가도록 보장합니다.

**분산 원천:**
1. 크리틱의 함수 근사 오차
2. 부트스트랩으로부터의 편향
3. 비정상 목표 (움직이는 크리틱)

**완화:**
- 목표 네트워크: 목표 계산을 위한 별도 네트워크
- 우위 절단: 우위 추정 경계
- 가치 정규화: Q러닝 전 반환 정규화

## 실무 고려사항

**하이퍼매개변수 튜닝:**
- 액터 및 크리틱 학습 속도
- 할인 계수 γ
- 우위 절단 범위
- 네트워크 아키텍처 및 크기

**네트워크 초기화:**
- 초기 가중치의 적절한 스케일링
- 편향 초기화가 학습에 영향
- 출력 범위 중요 (특히 결정론적 정책)

**데이터 수집:**
- 배치 크기가 수렴과 안정성에 영향
- 병렬 워커가 표본 효율성 개선
- 연속 행동 공간에서 탐험 노이즈 필수

## 다른 방법과의 비교

**vs 가치 기반 (DQN):**
- 연속 행동에 더 나음
- 더 안정적인 학습
- 더 나은 탐험

**vs 순수 정책 경사:**
- 가치 기준선으로부터 분산 감소
- 더 빠른 수렴
- 더 많은 데이터 효율성

**vs 모델 기반:**
- 모델 프리: 환경 모델 불필요
- 더 간단한 구현
- 실무에서 모델 기반과 종종 경쟁력

## 현대 응용

- **로봇공학:** 연속 제어, 실제 로봇 학습
- **게임:** 연속 행동 공간 (매크로 행동 StarCraft)
- **자율 시스템:** 실시간 의사결정
- **자원 할당:** 경제 부하 분산, 작업 스케줄링

## 결론

액터-크리틱 방법은 정책 최적화를 가치 기반 학습과 통합합니다. A2C와 A3C가 실무적 효과를 입증했고, SAC가 엔트로피 정규화를 도입했으며, TD3가 과대 추정을 해결했습니다. 이 방법들은 현대 깊은 RL을 지배하며 특히 연속 제어에서 우수합니다. 유연성, 표본 효율성, 이론적 기반이 많은 응용에서 선택하는 접근법입니다.

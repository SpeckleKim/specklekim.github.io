# 변분 오토인코더 (VAE)

## 개요

변분 오토인코더(Variational Autoencoders, VAE)는 깊은 학습과 변분 추론을 결합한 확률적 생성 모델링 접근법입니다. 입력을 결정적 코드로 매핑하는 표준 오토인코더와 달리, VAE는 데이터를 확률적 잠재 공간으로 매핑하는 방법을 학습하여 데이터 재구성과 새로운 샘플 생성을 모두 가능하게 합니다.

## 오토인코더 기본 개념

### 표준 오토인코더 아키텍처

표준 오토인코더는 두 부분으로 구성됩니다:

**인코더(Encoder):** 고차원 입력 데이터를 저차원 잠재 표현 z로 압축합니다. 인코더는 일반적으로 [신경망](/neural-kor.html)을 사용하여 이 매핑을 결정적으로 학습합니다.

**디코더(Decoder):** 잠재 코드에서 원본 입력을 재구성하여 역매핑을 학습합니다. 목표는 원본과 재구성된 데이터 사이의 재구성 손실을 최소화하는 것입니다.

표준 오토인코더는 압축에는 효과적이지만 잠재 공간에 구조를 강제하지 않아 불규칙한 분포와 간극을 만들기 때문에 샘플 생성이 부족합니다.

## VAE의 잠재 공간

VAE는 인코더가 점 추정이 아닌 분포의 매개변수를 생성하는 확률적 잠재 표현을 도입합니다. 구체적으로:

- 인코더는 가우스 분포의 평균(μ)과 표준 편차(σ) 출력
- 잠재 코드는 결정적으로 계산되는 대신 이 분포에서 샘플링됨
- 디코더는 이러한 확률적 샘플을 받아 입력을 재구성해야 함

이 확률성은 잠재 공간의 인접한 점이 유사한 출력을 생성하도록 하여 생성에 적합한 매끄럽고 연속적인 표현을 만듭니다.

## 재매개변수화 트릭

VAE 훈련의 중요한 과제는 확률적 샘플링 연산을 통한 [역전파](/backpropagation-kor.html)입니다. 무작위 연산을 직접 계산할 수 없기 때문입니다.

재매개변수화 트릭은 이를 우아하게 해결합니다:

```
z = μ + σ ⊙ ε, 여기서 ε ~ N(0,1)
```

N(μ, σ²)에서 z를 직접 샘플링하는 대신, 표준 정규 분포에서 잡음 ε을 샘플링하고 결정적으로 변환합니다. 이를 통해 기울기가 σ와 μ를 통해 흐르면서 ε을 보조 변수로 처리할 수 있습니다.

## ELBO와 훈련 목표

VAE는 증거 하한(Evidence Lower Bound, ELBO)을 최대화하도록 훈련됩니다:

```
ELBO = E_q[log p(x|z)] - KL(q(z|x) || p(z))
       = 재구성 항 - KL 발산 항
```

여기서:
- **재구성 항:** 모델이 잠재 샘플에서 입력을 정확하게 재구성하도록 권장
- **KL 발산 항:** 학습된 잠재 분포가 사전 분포(표준 정규)에서 벗어나지 않도록 페널티

이 결합 목표는 두 가지 목표의 균형을 맞춥니다: 정확한 재구성과 [정규화](/regularization-kor.html)된 잠재 공간 유지

## KL 발산 이해

쿨백-라이블러(Kullback-Leibler, KL) 발산은 한 확률 분포가 다른 분포와 어떻게 다른지를 측정합니다. VAE에서:

```
KL(q(z|x) || p(z)) = -0.5 * Σ(1 + log(σ²) - μ² - σ²)
```

KL 항은 [정규화](/regularization-kor.html)기로 작용하여 사후 분포가 사전 분포에서 너무 멀어지지 않도록 합니다. 이는 다음을 보장합니다:
- 잠재 공간에 의미 있는 구조 존재
- 잠재 공간의 점들 사이 보간이 매끄러움
- 모델이 사전에서 샘플링하여 새로운 샘플 생성 가능

## VAE vs GAN

VAE와 GAN은 모두 현실적인 샘플을 생성하지만 근본적으로 다른 접근법을 사용합니다:

**VAE:**
- 확률 분포를 명시적으로 모델링
- 명확한 [손실 함수](/loss-functions-kor.html)로 안정적 훈련
- 더 흐릿한 생성 이미지 (재구성 손실 때문)
- 잠재 공간 해석 용이

**GAN:**
- 대항적 학습을 통한 암묵적 밀도 모델링
- 더 선명하고 현실적인 [이미지 생성](/image-generation-kor.html) 가능
- 훈련 불안정성 및 모드 붕괴 문제
- 잠재 공간 해석이 어려움

최근 하이브리드 접근법은 두 아키텍처의 이점을 결합합니다.

## Beta-VAE와 분산된 표현

Beta-VAE는 KL 발산 항을 확대하여 표준 VAE를 확장합니다:

```
ELBO_β = E_q[log p(x|z)] - β * KL(q(z|x) || p(z))
```

더 높은 β 값은 모델이 분산된 잠재 요소를 학습하도록 강제합니다. 여기서 서로 다른 차원이 서로 다른 생성 요소(예: 자세, 조명, 정체성)를 캡처합니다. 이는 해석 가능하고 제어 가능한 생성을 가능하게 하여 단일 잠재 차원을 변경하면 의미 있는 속성 변화를 생성합니다.

## VAE의 응용

**데이터 생성:** 학습된 잠재 분포에서 합성 샘플 생성

**차원 축소:** 중요한 구조를 보존하면서 고차원 데이터 압축

**이상 탐지:** 재구성 오류 분석을 통한 비정상 샘플 식별

**표현 학습:** 다운스트림 작업을 위한 의미 있는 특성 임베딩 학습

**데이터 보간:** 잠재 공간에서 보간하여 데이터 포인트 간 부드러운 전환 생성

**이미지 향상:** 조건부 VAE 변형을 통한 선택적 흐릿함 감소 또는 아티팩트 제거

## 장점과 제한 사항

**장점:**
- 명확한 목표를 가진 원칙적인 확률 프레임워크
- 대항적 역학 없이 안정적 훈련
- 제어 가능한 생성을 가능하게 하는 해석 가능한 잠재 공간
- 밀도 추정 및 가능성 계산 가능
- 상각된 인코딩을 통한 효율적 추론

**제한 사항:**
- 재구성된 샘플은 종종 [GAN](/gan-kor.html) 출력보다 흐릿함
- KL 항이 비활성화될 수 있음 (KL 붕괴), 잠재 공간 활용 감소
- 후방 붕괴: 후방 분포가 사전으로 수렴하여 정보 손실
- 선명한 이미지 세부 사항 캡처에 덜 효과적

## 변형 및 확장

**조건부 VAE (CVAE):** 제어된 합성을 위해 클래스 레이블 또는 추가 정보로 생성을 조건화

**계층 VAE (HVAE):** 여러 수준의 잠재 변수를 사용하여 다양한 규모에서 학습

**순서 VAE:** 비디오 또는 시계열 생성을 위해 VAE를 시간 시퀀스로 확장

**VQ-VAE:** 이산 표현을 위해 잠재 공간에서 벡터 [양자화](/quantization-kor.html) 사용

## 결론

변분 오토인코더는 재구성과 [정규화](/regularization-kor.html)의 균형을 맞추는 원칙적인 확률 프레임워크를 제공함으로써 생성 모델링을 혁신했습니다. 최근 [확산 모델](/diffusion-models-kor.html)이 우수한 이미지 품질을 달성했지만, VAE는 해석 가능성, 가능성 계산, 분산된 표현 학습에 계속 가치가 있습니다. VAE를 이해하는 것은 현대 생성 모델과 확률적 깊은 학습의 이론적 기초를 이해하는 데 필수적입니다.

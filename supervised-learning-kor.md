# 지도학습: 레이블이 있는 데이터로 예측 모델 만들기

지도학습은 현대 머신러닝의 기초로, 레이블이 있는 데이터셋에서 학습하여 미지의 데이터에 대해 예측하는 방식입니다. "지도"라고 불리는 이유는 학습 데이터와 함께 정답인 레이블이 주어지기 때문입니다. 이 정답들이 학습 과정을 안내합니다.

## 지도학습이란?

지도학습에서는 입출력 쌍 (X, y)으로 이루어진 데이터셋을 가지고 있습니다. X는 특성(피처)을 나타내고, y는 목표 레이블이나 값입니다. 모델은 함수 f: X → y를 학습하여 새로운 데이터에도 잘 예측할 수 있도록 일반화됩니다. 이는 명시적인 레이블 없이 입력만 있는 [비지도학습](/unsupervised-learning-kor.html)과 대조됩니다.

### 핵심 원리
학습의 목표는 예측값과 실제값 사이의 오차를 최소화하는 것입니다. 훈련 중 모델은 이 오차를 줄이도록 파라미터를 조정하며, 궁극적으로 데이터 속의 패턴을 포착합니다.

## 분류 vs 회귀

지도학습의 두 가지 주요 과제는 서로 다른 예측 목표를 다룹니다.

**분류(Classification)**는 이산 카테고리를 예측합니다. 예시:
- 이메일 스팸 탐지 (스팸/정상)
- 이미지 인식 (개, 고양이, 새 등)
- 질병 진단 (양성, 음성)
- 감정 분석 (긍정, 부정, 중립)

분류의 출력은 클래스 레이블이며, 정확도, 정밀도, 재현율, F1-점수로 성능을 평가합니다.

**회귀(Regression)**는 연속적인 수치 값을 예측합니다. 예시:
- 주택 가격 예측
- 온도 예보
- 주가 예측
- 재고 관리를 위한 수요 예측

회귀의 출력은 실수이며, 평균제곱오차(MSE), 평균절대오차(MAE), R² 점수로 성능을 평가합니다.

## 주요 알고리즘

### 선형회귀
선형회귀는 특성과 목표 간의 관계를 선형 방정식으로 모델링합니다: y = w₁x₁ + w₂x₂ + ... + wₙxₙ + b. 해석 가능하고 계산 효율이 좋으며 기초선 모델로 유용합니다. 다만 선형 관계를 가정하므로 실제 데이터의 복잡한 패턴을 놓칠 수 있습니다.

**변형:** 능선회귀(Ridge)는 L2 [정규화](/regularization-kor.html)를 추가하여 과적합을 방지합니다. 라소 회귀(Lasso)는 L1 정규화를 사용하여 특성 선택도 수행합니다.

### 로지스틱 회귀
이름과 달리 로지스틱 회귀는 분류 알고리즘입니다. 선형회귀 결과에 로지스틱 시그모이드 함수를 적용하여 0과 1 사이의 확률로 변환합니다. 이진 분류에 널리 사용되며 해석 가능성이 높아 의료와 금융에서 인기 있습니다.

### 의사결정나무
[의사결정나무](/decision-trees-kor.html)는 특성값을 기준으로 데이터를 재귀적으로 분할하여 나무 구조를 만듭니다. 각 내부 노드는 의사결정, 각 가지는 결과, 각 잎은 클래스 예측입니다. 해석 가능하고 비선형 관계를 잘 처리하지만 가지치기 없이는 과적합되기 쉽습니다.

**앙상블 변형:** 랜덤포레스트는 여러 [의사결정나무](/decision-trees-kor.html)를 결합하여 정확도를 높이고 과적합을 줄입니다. [그래디언트 부스팅](/gradient-boosting-kor.html)([XGBoost](/gradient-boosting-kor.html), LightGBM)은 순차적으로 나무를 구축하며 각 나무가 이전 오류를 보정하여 많은 데이터셋에서 최고의 성능을 달성합니다.

### 서포트 벡터 머신 (SVM)
SVM은 클래스 간 마진을 최대화하는 최적의 초평면을 찾습니다. 커널 트릭을 사용하면 암시적으로 데이터를 고차원 공간으로 매핑하여 비선형 문제를 처리합니다. 고차원 데이터와 작은 데이터셋에서 잘 작동하지만 매우 큰 데이터셋에는 확장성이 떨어집니다.

### K-최근접이웃 (KNN)
KNN은 게으른 학습 알고리즘으로, 새로운 예시를 가장 가까운 k개의 훈련 이웃의 투표로 분류합니다. 단순하고 훈련이 필요 없지만 예측 시 계산이 많이 필요합니다. 유클리드 거리 같은 거리 메트릭이 "가까움"을 결정합니다.

## 훈련 과정

### 단계 1: 데이터 준비
데이터를 수집하고 정제하며, 결측값을 처리하고 중복을 제거하고 품질 문제를 확인합니다.

### 단계 2: 특성 공학
관련성 높은 특성을 선택하고, 기존 특성에서 새 특성을 만들고, 데이터를 [정규화](/regularization-kor.html)/표준화합니다. 좋은 특성은 예측력 있고, 상관관계 낮고, 적절한 스케일을 가집니다.

### 단계 3: 훈련-테스트 분할
데이터를 훈련, 검증, 테스트 세트로 나눕니다(일반적으로 70-80% 훈련, 10-15% 검증, 10-15% 테스트). 이는 데이터 누수를 방지하고 정직한 성능 평가를 가능하게 합니다.

### 단계 4: 모델 훈련
알고리즘이 훈련 데이터에서 파라미터를 학습합니다. 예를 들어 [경사 하강법](/calculus-kor.html)은 반복적으로 가중치를 업데이트하여 손실을 줄입니다.

### 단계 5: 초매개변수 튜닝
검증 데이터를 사용하여 초매개변수([학습률](/learning-rate-scheduling-kor.html), 나무 깊이, [정규화](/regularization-kor.html) 강도)를 조정합니다. 격자 탐색이나 무작위 탐색이 초매개변수 조합을 체계적으로 탐색합니다.

### 단계 6: 평가 및 테스트
별도로 보관한 테스트 세트에서 최종 모델의 성능을 평가합니다.

## 손실 함수

[손실 함수](/loss-functions-kor.html)는 예측 오차를 정량화합니다.

**평균제곱오차(MSE):** 예측과 정답 간 제곱 차의 평균으로, 큰 오류를 강조합니다. 회귀에 사용됩니다.

**교차 엔트로피 손실:** 예측과 실제 확률 분포 간 차이를 측정합니다. 특히 [신경망](/neural-kor.html)과 함께 분류에 표준입니다.

**힌지 손실:** SVM에서 사용되며, 마진과 함께 정확한 분류를 권장합니다.

**평균절대오차(MAE):** 절대 차의 평균으로, MSE보다 이상값에 더 견고합니다.

[손실 함수](/loss-functions-kor.html)의 선택은 모델의 동작에 영향을 미칩니다. MSE는 이상값에 더 높은 페널티를 주고, MAE는 모든 오류를 동등하게 처리합니다.

## 평가 지표

**정확도(Accuracy)**: 올바른 예측의 백분율입니다. 단순하지만 불균형한 데이터셋에서 오도할 수 있습니다.

**정밀도(Precision):** 예측한 양성 중 실제로 양성인 비율입니다. 거짓 양성이 비용이 클 때 중요합니다.

**재현율(Recall):** 실제 양성 중 발견한 비율입니다. 거짓 음성이 비용이 클 때 중요합니다(예: 질병 탐지).

**F1-점수:** 정밀도와 재현율의 조화평균으로 둘을 균형있게 봅니다.

**혼동행렬:** 참양성, 거짓양성, 참음성, 거짓음성을 보여주며 다른 지표의 기초입니다.

**ROC-AUC:** 수신자 조작 특성 곡선 아래 면적으로, 다양한 임계값에서 분류 성능을 측정합니다.

**평균제곱오차(회귀):** 평균 예측 오차의 제곱입니다.

**R² 점수(회귀):** 모델이 설명한 분산의 비율입니다(0~1, 높을수록 좋음).

## 과적합과 과소적합

**과적합**은 모델이 훈련 데이터를 지나치게 잘 학습하여 노이즈와 일반화되지 않는 특성까지 학습합니다. 훈련 정확도는 높지만 테스트 정확도는 낮습니다. 원인:
- 훈련 데이터 대비 모델 복잡도가 너무 높음
- 훈련 데이터 부족
- [정규화](/regularization-kor.html) 없음

방지 전략: 더 단순한 모델 사용, 더 많은 데이터 수집, [정규화](/regularization-kor.html) 적용(L1, L2, [드롭아웃](/dropout-kor.html)), 조기 종료, 다양한 훈련 데이터.

**과소적합**은 모델이 기본 패턴을 포착하기에 너무 단순합니다. 훈련과 테스트 정확도 모두 낮습니다. 원인:
- 모델 복잡도가 너무 낮음
- 훈련 부족
- 잘못된 특성

해결책: 더 복잡한 모델 사용, 더 나은 [특성 공학](/feature-engineering-kor.html), 더 오래 훈련, 더 많은 관련 데이터.

## 균형점 찾기

목표는 패턴을 포착할 수 있을 정도로 복잡하지만 일반화할 수 있을 정도로 단순한 모델입니다. 이는 적절한 훈련-검증-테스트 분할, [교차 검증](/cross-validation-kor.html), 신중한 초매개변수 튜닝으로 달성됩니다.

지도학습은 의료 진단부터 자율주행 자동차까지 수많은 응용을 구현하고 있어, 모든 AI 실무자에게 필수적인 지식입니다.

# 이미지 세그멘테이션: 픽셀 수준 분류

## 소개

이미지 세그멘테이션은 개별 픽셀에 클래스 레이블을 할당하여 밀도 있는 예측을 가능하게 합니다. 세 가지 주요 유형이 있습니다: 픽셀당 클래스 할당 의미론적 세그멘테이션, 개별 객체 구분 인스턴스 세그멘테이션, 둘을 결합하는 파노프틱 세그멘테이션입니다. 이는 로봇공학, 의료 영상, 자율주행에서 장면 이해의 기초입니다.

## 의미론적 세그멘테이션

각 픽셀에 클래스 레이블을 할당합니다. 같은 클래스의 여러 픽셀이 하나의 영역으로 병합됩니다. 같은 클래스의 개별 객체를 구분할 수 없습니다.

**FCN (완전 합성곱 네트워크, 2014):**
- 완전 연결 계층을 합성곱 계층으로 대체
- 조잡한 특징과 세분화된 특징을 결합하기 위해 조기 계층에서 스킵 연결 사용
- 업샘플링을 통해 출력 해상도가 입력과 일치
- 종단간 픽셀 단위 예측을 가능하게 하는 선구적 아키텍처

**DeepLab 시리즈:**
- **DeepLab v1-v3+:** 팽창 합성곱과 공간 피라미드 풀링을 통한 점진적 개선
- 팽창(확장) 합성곱은 해상도 손실 없이 수용영역 증가
- ASPP (팽창 공간 피라미드 풀링)은 다중 규모 맥락 포착
- 표준 벤치마크에서 우수한 성능

## 인스턴스 세그멘테이션

개별 객체 인스턴스를 구분하면서 정확한 픽셀 수준 마스크를 제공합니다. 의미론적 세그멘테이션보다 더 도전적입니다.

**Mask R-CNN (2017):**
- Faster R-CNN을 세그멘테이션 브랜치로 확장
- 각 영역 제안에 대해 픽셀 멤버십을 예측하는 이진 마스크 생성
- 경계 상자 탐지와 인스턴스 마스크 결합
- 표준 접근법이 되어 산업에 널리 채택
- 실시간 응용을 위한 빠른 Mask R-CNN 변형 이용 가능

**다른 접근법:**
- YOLACT: 속도 중심 인스턴스 세그멘테이션
- Panoptic Feature Pyramid Networks: 사물과 배경 클래스 처리

## 파노프틱 세그멘테이션

의미론적 세그멘테이션과 인스턴스 세그멘테이션을 통합합니다. 사물(사람, 자동차 같은 가산 객체)은 인스턴스 ID를 받고, 배경(하늘, 잔디 같은 배경 영역)은 의미론적 레이블만 받습니다.

**주요 도전:** 서로 다른 객체 범주의 이질적 특성을 동시에 처리합니다.

## U-Net (2015)

스킵 연결을 포함한 인코더-디코더 아키텍처:
- 인코더: 합성곱 계층이 점진적으로 다운샘플링
- 디코더: 업샘플링 계층과 스킵 연결이 공간 해상도 복원
- 의료 이미지 세그멘테이션에 널리 채택
- 우수한 성능에도 불구하고 적은 매개변수
- 생의학 영상 커뮤니티에서 매우 영향력 있음

## SAM (Segment Anything 모델)

대화형 세그멘테이션을 가능하게 하는 최근 기초 모델:
- 프롬프트(포인트, 박스, 텍스트)를 받아 마스크 생성
- 11억 개의 마스크를 포함한 다양한 데이터셋으로 학습
- 미세 조정 없이 새 도메인에 제로샷 일반화
- 세그멘테이션을 민주화—사용자가 ML 전문가일 필요 없음
- 세그멘테이션 워크플로우의 잠재적 혁명

## 손실 함수

**교차 엔트로피:** 표준 픽셀 단위 분류 손실

**Dice 손실:** IoU 유사 지표를 직접 최적화하며 클래스 불균형에 강건

**초점 손실:** 어려운 픽셀에 집중, 불균형 데이터셋에 유용

**Lovász-Hinge:** mIoU를 위한 대체 함수, 미분 가능한 완화

## 실무 고려사항

- **클래스 불균형:** 배경이 일반적으로 우위; 가중 손실과 어려운 음성 마이닝 도움
- **다중 규모 맥락:** 여러 규모의 정보 포착 필수
- **경계 정밀도:** 후 처리와 정제 네트워크가 엣지 정확도 개선
- **계산 효율성:** 실시간 세그멘테이션은 모델 최적화 필요

## 평가 지표

**mIoU (평균 IoU):** 클래스별 평균 IoU, 주요 지표

**픽셀 정확도:** 올바르게 분류된 픽셀의 백분율 (불균형 시 오도할 수 있음)

**경계 IoU:** 엣지 정확도를 구체적으로 측정

## 결론

이미지 세그멘테이션은 의미론적에서 인스턴스에서 파노프틱 접근법으로 진화하며 각각이 복잡성과 가치를 더합니다. FCN, U-Net, Mask R-CNN 같은 아키텍처가 기초를 확립했고, SAM과 확산 기반 접근법 같은 최신 모델이 추가 진전을 약속합니다. 이 패러다임을 이해하면 밀도 예측 작업에 대한 정보에 기반한 결정이 가능합니다.

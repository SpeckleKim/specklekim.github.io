# 플로우 기반 모델

## 개요

플로우 기반 모델은 단순 분포와 복잡한 데이터 분포 간의 변환을 학습하는 별개의 생성 모델 클래스를 나타냅니다. 역함수 변환을 사용함으로써, 이들 모델은 추적 가능한 가능성 계산을 제공하고 정확한 밀도 평가를 가능하게 합니다—GAN에 부족한 능력이며 VAE에서는 더 비싼 능력입니다.

## 정규화 흐름: 핵심 원칙

[정규화](/regularization-kor.html) 흐름은 단순 분포(일반적으로 표준 정규)를 일련의 역함수 미분 가능한 변환을 통해 복잡한 분포로 변환합니다:

```
z_0 ~ p(z_0)  (단순 분포)
z_1 = f_1(z_0)
z_2 = f_2(z_1)
...
x = z_K = f_K(f_{K-1}(...f_1(z_0)))
```

각 변환 f_i는 다음을 만족해야 합니다:
- **역함수 가능:** 전진과 역 변환을 모두 계산 가능
- **미분 가능:** 기울기 기반 최적화 허용
- **효율적:** 합리적 시간에 야코비안 행렬식 계산 가능

## 변수 변환과 가능성

핵심 수학 도구는 변수 변환 공식입니다. 역함수 변환 f를 통해 z에서 x로 변환할 때:

```
p(x) = p(z) * |det(∂f^{-1}/∂x)| = p(z) / |det(∂f/∂z)|
```

이는 정확한 로그 가능성을 계산할 수 있게 합니다:

```
log p(x) = log p(z) - log |det(∂f/∂z)|
```

정확한 가능성을 계산하는 능력은 [이상 탐지](/anomaly-detection-kor.html) 및 과학적 모델링과 같이 밀도 추정이 중요한 작업에 플로우 모델을 가치 있게 만듭니다.

## 역함수 변환

### 선형 변환

단순 역함수 선형 계층은 제한된 표현력을 제공하지만 효과적으로 쌓을 수 있습니다. 행렬식은 고유값에서 직관적으로 계산됩니다.

### 결합 계층

결합 계층은 입력 차원을 두 그룹으로 나눕니다. 첫 번째 그룹은 변경되지 않고 통과하고 두 번째 그룹은 첫 번째를 기반으로 변환됩니다:

```
x_1' = x_1
x_2' = t(x_1) ⊙ x_2 + s(x_1)
```

여기서 t와 s는 임의의 [신경망](/neural-kor.html)입니다. 이는 역함수 가능성을 유지하면서 임의로 복잡한 네트워크를 통한 강력한 변환을 허용합니다. 야코비안은 삼각형이므로 행렬식 계산이 효율적입니다.

### 자기회귀 계층

자기회귀 흐름은 각 출력 차원을 이전 입력에만 기반하여 계산합니다:

```
x_i' = t(x_1, ..., x_{i-1}) ⊙ x_i + s(x_1, ..., x_{i-1})
```

이는 역함수 가능성을 보장하고 삼각형 야코비안을 생성하지만, 변환 중에 순차 계산이 필요합니다.

## RealNVP: 마스크된 오토인코더 밀도 추정

RealNVP는 마스크된 오토인코더 원칙을 [정규화](/regularization-kor.html) 흐름과 결합합니다. 아키텍처는 다음 사이를 번갈아 사용합니다:

1. 다양한 차원 집합이 마스크되는 결합 계층
2. 정보 흐름을 가능하게 하는 차원의 학습 가능한 순열

마스킹은 각 계층의 야코비안이 삼각형이 되도록 하며, 순열은 정보가 초기 차원에 갇히지 않도록 합니다. 이는 전진과 역 변환의 효율적이고 병렬적인 계산을 가능하게 합니다.

## Glow: Actnorm으로 RealNVP 개선

Glow는 다음을 통해 RealNVP를 개선합니다:

**Actnorm 계층:** 데이터 종속 초기화를 사용한 채널별 아핀 변환을 통한 활성화 [정규화](/regularization-kor.html)입니다. 명시적 [배치 정규화](/batch-normalization-kor.html) 없이 빠르고 안정적인 훈련을 제공합니다.

**1x1 합성곱:** 고정 순열을 학습 가능한 1x1 합성곱으로 대체하여 유연성을 개선합니다.

**다중 규모 아키텍처:** 다양한 해상도 수준을 처리하여 표현력을 유지하면서 계산 효율성을 개선합니다.

Glow는 GAN과 비교할 수 있는 [이미지 생성](/image-generation-kor.html) 품질을 달성하면서 정확한 가능성을 제공합니다—고품질 생성 모델 중 고유한 이점입니다.

## 연속 정규화 흐름

신경 ODE 기반 접근법은 흐름을 연속 변환으로 모델링합니다:

```
z_T = z_0 + ∫_0^T v_θ(z_t, t) dt
```

여기서 v_θ은 훈련 중에 학습된 [신경망](/neural-kor.html) 속도 함수입니다. 행렬식 계산은 다음을 해결해야 합니다:

```
log |det(∂z_T/∂z_0)| = ∫_0^T tr(∂v_θ/∂z_t) dt
```

연속 흐름은 다음을 제공합니다:
- 아키텍처 제약 없는 유연한 아키텍처
- 연속 모델링을 통한 더 나은 표현력의 가능성
- 이론적 통찰을 제공하는 미분 방정식과의 연결

그러나 훈련과 샘플링 중에 비싼 수치 적분이 필요합니다.

## 비교: 플로우 모델 vs GAN vs VAE

**플로우 모델:**
- 정확한 가능성 계산
- 안정적 훈련
- GAN보다 느린 생성
- [확산 모델](/diffusion-models-kor.html)에 비해 제한된 생성 품질
- 역함수 가능성을 위한 아키텍처 제약

**GAN:**
- 선명한 [이미지 생성](/image-generation-kor.html)
- 빠른 샘플링
- 불안정한 훈련 및 모드 붕괴
- 사용 가능한 가능성 없음
- 더 유연한 아키텍처

**VAE:**
- 해석 가능한 잠재 공간
- 안정적 훈련
- 흐릿한 생성
- 추적 가능한 (대략적이지만) 가능성
- 중간 계산 비용

**확산 모델:**
- 최고 수준의 생성 품질
- 안정적 훈련
- 느린 샘플링
- 수정을 통한 가능성 계산 허용
- [이미지 생성](/image-generation-kor.html)에 점점 선호됨

## 플로우 모델의 응용

**밀도 추정:** 과학 데이터 분석, [이상 탐지](/anomaly-detection-kor.html)를 위한 가능성 계산

**역함수 신경망:** 특정 응용을 위한 역함수 변환 학습

**변분 추론:** 복잡한 사후 분포를 근사하기 위해 흐름 사용

**데이터 생성:** 합성 샘플 생성, 일반적으로 확산보다 낮은 품질

**특성 학습:** 밀도 모델링을 통한 비지도 표현 학습

**샘플링:** 복잡한 분포에서 효율적으로 샘플링

## 실제 장점과 제한 사항

**장점:**
- 정확하고 추적 가능한 가능성 계산
- 대항적 역학 없이 안정적 훈련
- 역함수 속성은 생성 및 추론을 모두 가능하게 함
- 이론적으로 원칙적인 접근
- 결정적 인코딩 및 디코딩

**제한 사항:**
- 아키텍처 제약은 모델링 유연성 감소
- 생성된 샘플은 일반적으로 GAN보다 선명하지 않음
- GAN보다 느린 생성
- 훈련 계산 비용이 상당할 수 있음
- 고해상도 [이미지 생성](/image-generation-kor.html)에서 제한된 성공
- 야코비안 계산은 신중한 구현 필요

## 현재 연구 방향

최근 흐름 연구는 다음을 탐색합니다:
- 흐름을 [확산 모델](/diffusion-models-kor.html)과 결합한 하이브리드 접근법
- 빠른 샘플링을 위한 개선된 역 변환
- 고차원 데이터를 위한 확장 가능한 플로우 아키텍처
- 과학 컴퓨팅 및 물리학의 응용
- 더 효율적인 수치 방법을 사용한 연속 흐름

## 결론

플로우 기반 모델은 보장된 가능성 계산을 통한 생성 모델링에 대한 수학적으로 우아한 접근법을 제공합니다. 최근 [확산 모델](/diffusion-models-kor.html)이 우수한 [이미지 생성](/image-generation-kor.html)을 달성했지만, 플로우는 [이상 탐지](/anomaly-detection-kor.html), 과학적 추론, 가능성 기반 분석과 같이 정확한 밀도를 필요로 하는 응용에 계속 가치가 있습니다. [정규화](/regularization-kor.html) 흐름을 이해하는 것은 역함수 변환과 확률에 대한 직관을 개발하며, 이는 깊은 학습에 걸쳐 점점 관련성이 높아지는 개념입니다.

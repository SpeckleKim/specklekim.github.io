# AI 편향성과 공정성

## 소개

AI 편향성은 모델이 특정 그룹에 대해 체계적으로 불공정한 결정을 내릴 때 발생합니다. 편향성은 학습 데이터, 모델 아키텍처, 배포 컨텍스트에서 나타납니다. 공정성은 다양한 공정성 개념을 이해하고, 편향성을 측정하고, 완화를 설계해야 합니다.

## 편향의 유형

### 선택 편향

비무작위 데이터 수집:
- 채용 데이터셋: 고성과 인구통계 과다 표현
- 의료 데이터셋: 희귀 질환 환자 제외
- 결과: 모델이 왜곡된 모집단 분포 학습

### 확인 편향

데이터 라벨링의 인간 편향:
- 주석자가 기대에 따라 다르게 라벨
- 불일치한 라벨 초래
- 모델이 편향된 패턴 학습

### 알고리즘 편향

모델 설계 내재:
- 특성 상호작용이 기존 편향 증대
- 최적화 목표가 공정성을 놓침
- 피드백 루프: 편향된 예측 → 편향된 학습 데이터

### 측정 편향

실제 결과의 불완전한 프록시:
- 재범률: 실제 범죄 아닌 체포 측정
- 학업 성공: 실제 능력 아닌 시험 성적 측정
- 모델이 불완전한 프록시 최적화

## 공정성 메트릭

### 인구통계적 패리티

그룹 간 예측율 동등:
- 그룹별 공정성 확인
- 다양한 지표 측정

## 결론

AI 편향성 해결은 기술 솔루션과 공정성의 신중한 고려가 필요합니다.

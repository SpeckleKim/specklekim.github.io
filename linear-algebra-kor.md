# AI를 위한 선형대수학

선형대수학은 머신러닝과 [인공지능](/ai-kor.html)의 기초입니다. [신경망](/neural-kor.html)부터 추천 시스템까지 거의 모든 ML 알고리즘은 선형대수 연산에 의존합니다. 벡터, 행렬, 그리고 그들의 변환을 이해하는 것은 AI 모델을 구축하고, 학습시키고, 최적화하는 데 필수적입니다.

## 벡터와 행렬

**벡터**는 순서대로 배열된 수의 목록입니다. AI에서 벡터는 데이터 포인트, 특성(feature), 또는 임베딩을 나타냅니다. 예를 들어, 이미지는 픽셀 강도의 벡터로 표현될 수 있습니다.

**행렬**은 행과 열로 배열된 직사각형 수 그리드입니다. 행렬은 변환, 데이터셋, 또는 [신경망](/neural-kor.html)의 가중치 매개변수를 나타냅니다. m×n 행렬은 m개의 행과 n개의 열을 가집니다.

```
벡터 v = [3, 1, 4]
행렬 A = [[1, 2],
         [3, 4],
         [5, 6]]
```

## 행렬 연산

**행렬 덧셈과 뺄셈**: 같은 크기의 행렬 간 원소별(element-wise) 연산입니다.

**행렬 곱셈**: 행과 열의 내적입니다. A가 m×n이고 B가 n×p이면, AB는 m×p입니다. [신경망](/neural-kor.html)에서 입력 벡터에 가중치 행렬을 곱할 때 근본적입니다.

**전치(Transpose)**: 행렬을 대각선 기준으로 뒤집는 것입니다. A^T로 표기하며, 행과 열을 바꿉니다. 많은 ML 공식과 최적화 알고리즘에서 필수적입니다.

**행렬식(Determinant)**: 행렬이 역행렬 가능한지 여부를 나타내는 스칼라 값입니다. 행렬식이 0이면 그 행렬은 특이행렬(singular)입니다.

**역행렬(Inverse)**: A^(-1)은 A·A^(-1) = I (단위행렬)를 만족하는 A의 역행렬입니다. 선형 시스템 해결과 최적화에 사용됩니다.

## 고유값과 고유벡터

행렬 A의 **고유벡터**는 A·v = λ·v를 만족하는 0이 아닌 벡터 v입니다. 여기서 λ는 **고유값**입니다. 기하학적으로, A를 고유벡터에 적용하면 방향은 바뀌지 않고 크기만 변합니다.

고유값은 중요한 성질을 드러냅니다:
- 고유벡터 방향을 따른 변환의 크기를 나타냅니다
- 양반정치(positive semi-definite) 행렬의 모든 고유값은 양수입니다 (ML에서 흔함)
- 가장 큰 고유값들이 데이터의 가장 중요한 분산을 캡처합니다

## 특이값 분해(SVD)

SVD는 임의의 행렬 A를 다음과 같이 분해합니다: **A = U·Σ·V^T**

여기서:
- **U**: 좌 특이벡터의 직교행렬
- **Σ**: 특이값의 대각행렬
- **V^T**: 우 특이벡터의 직교행렬의 전치

**AI에서 SVD의 중요성:**
- 최대 분산을 보존하면서 [차원 축소](/dimensionality-reduction-kor.html)
- 이미지 압축과 노이즈 제거에 활용
- 주성분분석(PCA)의 기초
- 추천 시스템의 행렬 인수분해에 사용

## 행렬 인수분해

행렬 인수분해는 큰 행렬을 더 작은 행렬들의 곱으로 근사합니다. 추천 시스템에서, 사용자-항목 평점 행렬 R은 **R ≈ U·V^T**로 근사됩니다. U는 사용자 특성을, V는 항목 특성을 포함합니다.

이를 통해:
- **차원 축소**: 잠재 패턴 찾기
- **계산 효율성**: 더 작은 행렬로 작업
- **협업 필터링**: 패턴을 바탕으로 누락된 평점 예측

## 주성분분석(PCA)

PCA는 고유값과 고유벡터를 사용하여 데이터 차원을 축소하면서 분산을 보존합니다:

1. 데이터 중심화 (평균 뺄셈)
2. 공분산 행렬 계산
3. 공분산 행렬의 고유값과 고유벡터 찾기
4. 가장 큰 고유값을 가진 상위 k개 고유벡터 선택
5. 데이터를 이 k개 차원으로 투영

**응용:**
- 데이터 시각화 (MNIST 784차원 이미지를 2D로 축소)
- 빠른 학습을 위한 전처리
- 상관된 특성 제거

## 임베딩과 변환

현대 AI에서, **임베딩**은 단어, 항목, 또는 개념을 고차원 공간의 벡터로 표현합니다. 임베딩은 의미론적 관계를 캡처합니다: 유사한 항목은 유사한 벡터를 가집니다.

행렬을 사용한 선형 변환은:
- **회전**: 좌표계 변경
- **스케일링**: 특성 크기 조정
- **투영**: 저차원으로 매핑
- **합성**: 여러 변환 연결

[신경망](/neural-kor.html)은 각 계층에서 임베딩을 변환하는 가중치 행렬을 학습하며, 궁극적으로 작업에 관련된 표현을 학습합니다.

## 신경망에서의 선형대수

**순전파(Forward Pass)**: 계층 출력 계산은 행렬 곱셈을 포함합니다: y = W·x + b. 여기서 W는 가중치 행렬, x는 입력, b는 편향입니다.

**역전파(Backpropagation)**: 그래디언트 계산은 연쇄 법칙과 행렬 연산을 사용합니다. 야코비 행렬(편미분의 행렬)은 그래디언트가 어떻게 역방향으로 흐르는지 이해하는 데 중심적입니다.

**최적화**: 경사하강법 업데이트는 선형대수 연산입니다: W ← W - α·∇W. 여기서 ∇W는 그래디언트 행렬, α는 [학습률](/learning-rate-scheduling-kor.html)입니다.

## 왜 선형대수를 배워야 할까?

- **알고리즘 효율성**: 어떤 행렬 연산이 계산상 효율적인지 알기
- **깊은 이해**: 모델이 내부적으로 정말 무엇을 하는지 파악
- **구현**: 효율적인 커스텀 계층 또는 연산 구축
- **디버깅**: 모델이 특정 방식으로 동작하는 이유 이해
- **연구**: 새로운 아키텍처와 알고리즘 개발

선형대수는 추상적인 데이터를 우리가 조작하고, 분석하고, 최적화할 수 있는 기하학적 객체로 변환합니다. 이것이 머신러닝이 생각하는 언어입니다.

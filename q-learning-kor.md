# Q러닝과 심층 Q 네트워크: 모델 프리 가치 학습

## 소개

Q러닝은 환경 모델을 필요로 하지 않고 최적 행동 가치를 학습하는 기초적인 모델 프리 [강화학습](/rl-kor.html) 알고리즘입니다. 심층 Q 네트워크는 [신경망](/neural-kor.html)을 통해 고차원 상태 공간으로 이를 확장합니다. 둘 모두 실용적인 RL 응용에서 매우 영향력 있습니다.

## Q값과 Q테이블

Q(s,a)는 상태 s에서 행동 a를 취하고 그 후 최적 정책을 따르는 예상 누적 할인 보상을 나타냅니다.

**Q의 벨만 최적성:**
```
Q*(s,a) = E[R_{t+1} + γ max_{a'} Q*(s',a') | S_t = s, A_t = a]
```

**Q테이블:** 이산 상태/행동 공간의 경우 Q(s,a)를 조회 테이블에 저장합니다. 업데이트는 간단한 테이블 연산이 됩니다.

## 시간차 학습

TD 방법은 부트스트랩—현재 가치 추정을 사용하여 미래 가치 추정합니다:

```
NewEstimate = OldEstimate + α[Target - OldEstimate]
```

여기서 Target은 다음 상태의 현재 추정을 통합합니다.

**Monte Carlo에 대한 장점:**
- 에피소드 종료 전에 학습
- 전체 궤적 반환보다 낮은 분산
- 단일 스텝 피드백

## Q러닝 알고리즘

**오프 정책 업데이트 규칙:**
```
Q(S_t, A_t) ← Q(S_t, A_t) + α[R_{t+1} + γ max_{a'} Q(S_{t+1}, a') - Q(S_t, A_t)]
```

**주요 특성:**
- 오프 정책: 모든 행동 정책에서 학습 가능
- 부트스트랩을 위해 탐욕 정책(max Q) 사용
- 실제 행동은 다른 정책을 따를 수 있음 (탐험)
- 온화한 조건 하에서 수렴이 보장됨

**오프 정책이 중요한 이유:**
행동 정책(탐험)을 목표 정책(탐욕)과 분리합니다. 부최적 궤적에서 학습을 가능하게 하고 표본 효율성을 향상시킵니다.

## 경험 재생

**문제:** 순차 경험은 높은 상관관계를 가져 감독 학습의 독립동일분포 가정을 위반합니다.

**해결책:** 재생 버퍼에 전이를 저장하고 업데이트를 위해 미니배치 샘플:

```
버퍼에 (s, a, r, s') 저장
무작위 미니배치 샘플
미니배치로 Q 업데이트
```

**장점:**
- 샘플 간 상관관계 제거
- 과거 경험을 여러 번 재사용
- 학습 안정화
- 비동기 학습 활성화

## 심층 Q 네트워크 (DQN, 2015)

Q러닝을 연속 상태 공간으로 확장:

**요소:**
1. **신경망:** 상태를 모든 행동에 대한 Q(s,a)로 매핑
2. **경험 재생:** 메모리 버퍼에서 무작위 샘플링
3. **목표 네트워크:** 부트스트랩을 위한 별도 네트워크

**목표 네트워크:**
목표 계산을 위해 네트워크 사본 유지:
```
Target = r + γ max_{a'} Q_target(s', a')
Loss = [Target - Q(s,a)]²
```

매 N 스텝마다 목표 네트워크 업데이트합니다. 발산하는 부트스트랩을 줄여 학습을 안정화합니다.

**목표 네트워크 필요 이유:**
움직이는 목표 추적을 방지합니다. Q 네트워크와 목표 네트워크는 종종 발산하여 불안정성을 초래합니다.

## Double DQN

**과대 추정 문제:** Q러닝의 최댓값 연산은 과대 추정된 가치를 초래합니다.

**해결책:** 현재 네트워크를 행동 선택에 사용하고 목표 네트워크를 평가에 사용:
```
a* = 현재 네트워크로 argmax_a Q(s',a)
Target = r + γ Q_target(s',a*) 목표 네트워크 사용
```

행동 선택을 평가와 분리하여 과대 추정을 줄입니다.

## Dueling DQN

**아키텍처 혁신:** 가치와 우위 스트림을 분리:

```
V(s): 상태 가치, 행동과 무관
A(s,a): 우위, 행동이 평균보다 얼마나 나은지
Q(s,a) = V(s) + A(s,a) - mean(A)
```

**장점:**
- 행동 전역에서 상태 가치의 특징 공유
- 행동 우위를 독립적으로 학습
- 더 빠른 학습과 향상된 안정성
- 드물게 방문한 상태에 더 나은 학습 신호

## Rainbow DQN

여러 개선을 결합합니다:
1. Double DQN: 과대 추정 감소
2. Dueling DQN: 가치/우위 분해
3. 우선순위 재생: 중요한 전이를 더 샘플링
4. 노이즈 네트워크: 매개변수 노이즈를 통한 탐험
5. 분포적 RL: 전체 Q 분포 학습
6. 다중 스텝 TD: n스텝 부트스트랩

바닐라 DQN을 크게 능가합니다.

## 탐험 vs 이용

**ε-탐욕:** 확률 ε로 무작위 행동 실행, 그 외에는 탐욕적.
- 단순하고 많은 문제에 효과적
- ε-감소: 시간에 따라 ε 감소하여 더 많은 이용

**다른 전략:**
- Boltzmann 탐험: Q값에 대한 소프트맥스
- Thompson 샘플링: 불확실성 기반 탐험
- 상한 신뢰도 (UCB): 불확실성 하에서의 낙관주의

## 실무 고려사항

**하이퍼매개변수:**
- 학습 속도 α: 업데이트 단계 크기 제어
- 할인 γ: 미래 vs 즉각적 보상 가중치
- 재생 버퍼 크기: 과거 경험 메모리
- 목표 네트워크 업데이트 빈도
- ε와 ε-감소 스케줄

**안정성 문제:**
- 함수 근사 발산
- 과대 추정으로 인한 높은 편향
- 비정상 목표
- 상관 샘플

## 한계

- 재생 버퍼에도 불구하고 표본 비효율성
- 이산 행동 공간으로 제한 (확장 있음)
- 최댓값 연산의 이산 특성
- 매우 큰 행동 공간에서 고투

## 현대 확장

- **Rainbow DQN:** 여러 개선 결합
- **노이즈 네트워크:** 매개변수화된 탐험
- **분포적 RL:** 가치 분포 학습
- **오프라인 RL:** 고정 데이터셋에서 학습
- **다중 스텝 학습:** n스텝 TD 반환

## 결론

Q러닝은 모델 프리 RL의 기초를 제공합니다. DQN은 [신경망](/neural-kor.html)과 안정화 기법을 통해 Q러닝을 고차원 상태로 성공적으로 확장했습니다. Double DQN과 Dueling DQN이 특정 문제를 해결했습니다. Rainbow는 여러 개선을 통합된 아키텍처로 통합했습니다. 이 방법들은 깊은 RL을 확립했고 최신 정책 기울기 방법에도 불구하고 영향력을 유지합니다.

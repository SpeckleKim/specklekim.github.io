# AI를 위한 볼록 최적화

볼록 최적화는 기계학습의 수학적 척추입니다. [신경망](/neural-kor.html) 학습에서 [서포트 벡터 머신](/svm-kor.html)까지 최적화 문제는 AI의 중심입니다. 이 가이드는 볼록 최적화의 이론과 실제 응용을 살펴봅니다.

## 볼록 집합: 기초 기하학

집합 C는 그 안의 모든 두 점 x, y ∈ C에 대해, 그들 사이의 전체 선분이 C에 포함되면 볼록입니다:

**λx + (1-λ)y ∈ C for all λ ∈ [0,1]**

**볼록 집합의 예:**
- 초평면: {x | aᵀx = b} (아핀 부분집합)
- 반공간: {x | aᵀx ≤ b} (선형 부등식)
- 공: {x | ||x - c|| ≤ r}
- 다면체: 반공간의 교집합
- 원뿔: {αx | α ≥ 0, x ∈ C}

**비볼록 집합:**
- 원과 타원 (경계만)
- 분리된 영역의 합집합

볼록 기하학은 최적화에서 중요합니다. 볼록 집합에서 모든 극소값은 전역 최소값이기 때문입니다. 이는 여러 극소값의 저주를 제거합니다.

## 볼록 함수: 핵심 성질

함수 f는 볼록 집합 C 위에서 다음을 만족하면 볼록입니다:

**f(λx + (1-λ)y) ≤ λf(x) + (1-λ)f(y) for all x,y ∈ C and λ ∈ [0,1]**

직관적으로, 함수는 그래프의 두 점을 연결하는 모든 현 아래에 있습니다.

**볼록 함수의 예:**
- 선형 함수: f(x) = aᵀx + b
- 이차 함수: f(x) = xᵀQx + bᵀx + c (Q는 양반정치)
- 노름: ||x||₂, ||x||₁ (L1과 L2 노름)
- 지수: f(x) = eᵃˣ
- 로그-합-지수: f(x) = log(∑ eˣⁱ)
- 음의 [엔트로피](/information-theory-kor.html): f(x) = ∑ xᵢ log(xᵢ)

**볼록성을 보존하는 연산:**
- 음이 아닌 가중 합: f, g가 볼록이고 α, β ≥ 0이면, αf + βg는 볼록
- 합성: g가 볼록이고 h가 음이 아닌 비감소 볼록이면, h(g(x))는 볼록
- 점별 최대: f, g가 볼록이면, max(f, g)는 볼록
- 부분 최소화: f가 (x,y)에서 볼록이면, min_y f(x,y)는 x에서 볼록

이러한 연산은 단순한 함수에서 복잡한 볼록 함수를 구성하는 것을 가능하게 합니다.

## 기본 정리: 전역 vs 극소값

**핵심 통찰:** 볼록 최적화 문제에서 모든 극소값은 전역 최소값입니다.

**증명 개요:** x*가 극소값이지만 전역 최소값이 아니라고 가정합시다. 그러면 f(y) < f(x*)를 만족하는 y가 존재합니다. 이들 사이의 선분을 생각합시다: z(λ) = λy + (1-λ)x*. 볼록성에 의해:

f(z(λ)) ≤ λf(y) + (1-λ)f(x*) < f(x*)

작은 λ > 0에 대해, f(z(λ)) < f(x*)이므로, x*가 극소값이라는 것에 모순입니다.

이는 비볼록 최적화의 주요 어려움을 제거합니다. 또한 목적 함수가 순볼록이면, 최소값은 유일합니다.

**1차 필요 조건:** 미분 가능 f의 최소값 x*에서:
- ∇f(x*) = 0

볼록 f에 대해, 이 조건은 또한 충분합니다. 이는 단순한 그래디언트 기반 알고리즘을 가능하게 합니다: 음의 그래디언트를 따르면, 전역 최적값에 도달합니다.

## 라그랑주 승수: 제약 처리

실제 최적화 문제는 종종 제약을 가집니다. 라그랑주 승수는 비제약 최적화 이론을 확장합니다.

**문제 공식화:**
minimize f₀(x)
subject to: fᵢ(x) ≤ 0, i = 1,...,m
           hⱼ(x) = 0, j = 1,...,p

**라그랑주 함수:** L(x, λ, ν) = f₀(x) + ∑ λᵢfᵢ(x) + ∑ νⱼhⱼ(x)

여기서 λᵢ는 부등식 제약 승수 (λᵢ ≥ 0)이고 νⱼ는 등식 제약 승수 (제약 없음)입니다.

**상호 보완 이완:** 최적성에서, 부등식 제약이 비활성화되면 (fᵢ(x*) < 0), 그의 승수는 0입니다: λᵢ = 0. 활성화되면 (fᵢ(x*) = 0), 승수는 양수일 수 있습니다.

직관: 제약을 구속하는 것만이 최적해에 영향을 미칩니다. 승수 λᵢ는 제약 i를 완화하는 한계 비용을 나타냅니다.

## KKT 조건: 최적성 특징화

미분 가능한 목적 함수와 제약을 가진 볼록 문제에 대해, 카루시-쿤-터커(KKT) 조건은 최적성의 필요충분조건입니다:

1. **주 문제의 실행 가능성:**
   - fᵢ(x*) ≤ 0 for i = 1,...,m
   - hⱼ(x*) = 0 for j = 1,...,p

2. **쌍대 문제의 실행 가능성:**
   - λᵢ* ≥ 0 for i = 1,...,m

3. **상호 보완 이완:**
   - λᵢ* fᵢ(x*) = 0 for i = 1,...,m

4. **정상성:**
   - ∇f₀(x*) + ∑ λᵢ* ∇fᵢ(x*) + ∑ νⱼ* ∇hⱼ(x*) = 0

이 조건들은 문제의 구조를 모르고도 최적성을 검증하기 위해 확인될 수 있습니다.

**해석:** 최적성에서, 목적 함수의 그래디언트는 제약 그래디언트의 선형 결합입니다. 가중치(승수)는 각 제약의 중요성을 나타냅니다.

## 쌍대성: 문제 재공식화

**쌍대 문제:** 모든 최적화 문제(주 문제)에 대해, 관련된 쌍대를 공식화할 수 있습니다.

**쌍대 함수:** g(λ, ν) = inf_x L(x, λ, ν)

쌍대 문제는:
maximize g(λ, ν)
subject to: λ ≥ 0

**약 쌍대성:** g(λ, ν) ≤ f₀(x*)이 모든 실행 가능한 (x, λ, ν)에 대해 성립합니다.

이는 주 문제를 풀지 않고도 최적값의 하한을 제공합니다.

**강 쌍대성:** 볼록 문제에 대해(슬레이터 조건과 같은 온화한 정규성 조건 하에서), 최적 주 값과 쌍대 값은 같습니다.

p* = d*

이는 때로 계산상 더 효율적인 쌍대를 대신 풀 수 있게 합니다.

**쌍대 분해:** 쌍대성은 대규모 문제에 대한 병렬 알고리즘을 가능하게 합니다. 문제를 분해하고, 쌍대 변수를 사용하여 부분 문제를 독립적으로 해결하고 조정합니다.

## 서포트 벡터 머신에서의 응용

SVM은 볼록 최적화의 정규적 응용입니다.

**SVM 주 문제** (선형 경우):
minimize (1/2)||w||² + C∑ξᵢ
subject to: yᵢ(wᵀφ(xᵢ) + b) ≥ 1 - ξᵢ
           ξᵢ ≥ 0

여기서 w는 결정 경계이고, φ는 특성 맵이며, ξᵢ는 오분류를 허용하는 슬랙 변수이고, C는 여유(margin) 너비 대 오분류를 절충합니다.

이는 선형 제약을 가진 볼록 이차 계획법입니다: 볼록 목적 함수를 선형 제약 하에서 최소화합니다.

**SVM 쌍대 문제:**
maximize ∑ αᵢ - (1/2)∑∑ αᵢαⱼyᵢyⱼK(xᵢ,xⱼ)
subject to: ∑ αᵢyᵢ = 0
           0 ≤ αᵢ ≤ C

여기서 K(xᵢ,xⱼ) = φ(xᵢ)ᵀφ(xⱼ)는 커널 함수입니다.

쌍대는 커널 방법을 가능하게 합니다: 명시적 특성 맵 없이 K를 직접 계산하여, 높은 차원 공간에서 효율적으로 비선형 분류를 가능하게 합니다.

**KKT 조건의 상호 보완 이완:**
- αᵢ > 0이면 오직 yᵢ(wᵀφ(xᵢ) + b) = 1일 때만 (여유 위에)
- αᵢ = C이면 오직 ξᵢ > 0일 때만 (오분류)

부분집합만(지원 벡터)이 w를 결정하므로, 효율적인 희소 솔루션을 가능하게 합니다.

## 기계학습에서의 제약 최적화

**정규화된 손실 최소화:**
minimize ∑ loss(yᵢ, fθ(xᵢ)) + λR(θ)

여기서 R(θ)는 [정규화](/regularization-kor.html) 항입니다. 이는 제약 최적화와 동치입니다:
minimize ∑ loss(yᵢ, fθ(xᵢ))
subject to: ||θ||² ≤ t

라그랑주 승수 λ는 제약 강도를 제어합니다.

**연합 학습 제약:** [분산 학습](/distributed-training-kor.html) 문제는 종종 라그랑주 쌍대성을 통해 시행되는 프라이버시 제약을 가집니다.

**포트폴리오 최적화:** 위험과 레버리지 제약을 받을 때 수익률을 최대화합니다—금융 분야의 볼록 최적화의 정규적 응용입니다.

## 최적화 알고리즘: 이론에서 실제로

**그래디언트 강하:** 단순하지만 효과적입니다. 볼록 f에 대해:
xₖ₊₁ = xₖ - αₖ∇f(xₖ)

적절한 스텝 크기 αₖ로, 최적값으로 수렴합니다. 조건 수가 나쁜 문제에서는 느립니다.

**뉴턴 방법:** 2차 정보(헤시안)를 사용합니다:
xₖ₊₁ = xₖ - [∇²f(xₖ)]⁻¹∇f(xₖ)

2차 수렴: 에러는 각 반복마다 제곱됩니다. 비싼 헤시안 계산은 적용성을 제한합니다.

**좌표 강하:** 한 번에 하나의 변수를 최적화합니다:
xₖ₊₁ = argmin_xⱼ f(x₁,...,xₖ,xⱼ,xₖ₊₁,...)

분리 가능한 구조를 가진 고차원 볼록 문제에 유용합니다.

**근처 방법:** 근처 연산자를 사용하여 매끄럽지 않은 볼록 함수를 처리합니다.

**사영 그래디언트 강하:** 제약 문제의 경우:
xₖ₊₁ = Π_C(xₖ - αₖ∇f(xₖ))

여기서 Π_C는 실행 가능 집합 C로의 사영입니다.

## 비볼록을 볼록으로 근사

많은 관심 문제는 비볼록입니다. 볼록 완화는 근사 솔루션을 제공합니다.

**볼록 완화:** 비볼록 제약을 볼록으로 교체합니다. 예를 들어, 압축 센싱에서, 비볼록 "minimize ||x||₀ subject to Ax = b"를 볼록 L1 완화 "minimize ||x||₁ subject to Ax = b"로 교체합니다.

적절한 조건 하에서, L1 최소화는 L0 최소화로 달성 가능한 희소 솔루션을 복원합니다.

**반정치 계획법:** 이차 제약을 양반정치 제약으로 완화하여 볼록 계획법을 생성합니다.

## 한계와 볼록 최적화가 실패하는 경우

볼록 최적화는 다음을 가정합니다:
1. 볼록 목적 함수
2. 볼록 실행 가능 집합

많은 실제 문제가 이를 위반합니다: 심층 [신경망](/neural-kor.html) 학습은 매우 비볼록입니다. 극소값은 지형이 비볼록일 때 중요합니다.

그러나 [신경망](/neural-kor.html)은 종종 유리한 기하학을 가집니다: 많은 극소값이 전역 최적값 근처에 있습니다. 또한 현대 기법([정규화](/regularization-kor.html), 규칙화)은 암묵적으로 구조를 시행합니다.

## 결론

볼록 최적화는 온화한 조건 하에서 전역 최적값으로의 수렴을 보장하는 엄격한 수학적 틀을 제공합니다. 볼록 집합, 함수, 쌍대성을 이해하는 것은 알고리즘 설계와 문제 공식화를 알립니다. SVM에서 [연합 학습](/privacy-ai-kor.html)까지 대규모 ML 시스템에 이르기까지, 볼록 최적화 원칙은 성공적인 알고리즘을 지원합니다. 현대적 딥러닝이 비볼록 문제를 넘어 확장되는 동안, 볼록 이론은 AI 시스템에서의 최적화를 이해하기 위한 기초로 남아있습니다.

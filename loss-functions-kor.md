# 손실 함수: 모델 오류 측정

손실 함수는 모델 예측과 실제 레이블 간의 불일치를 정량화합니다. 올바른 손실 함수 선택은 중요합니다. 모델이 최적화하는 목표를 직접 결정하고 궁극적으로 학습하는 해를 결정합니다.

## 평균제곱오차 (MSE)

MSE는 회귀의 가장 흔한 손실 함수입니다:

`MSE = (1/m) * Σ(y_참 - y_예측)²`

**특징**: 큰 오류에 작은 오류보다 더 큰 패널티(제곱 패널티). 모든 곳에서 부드러운 기울기.

**장점**: 간단함, 미분 가능, 볼록.

**단점**: 이상치에 민감. 참값에서 먼 예측에 큰 패널티.

**사용 사례**: 표준 회귀 작업. 이상치가 드물거나 가중치를 두어야 할 때.

## 평균절대오차 (MAE)

MAE는 평균 절댓값 차이를 계산합니다:

`MAE = (1/m) * Σ|y_참 - y_예측|`

**특징**: 모든 오류에 선형 패널티. 이상치에 견고.

**장점**: 이상치 저항성. 원래 데이터 단위로 해석 가능.

**단점**: 0에서 미분 불가(기울기 정의 안 됨). 매우 작은 오류에 덜 민감.

**사용 사례**: 이상치 있는 회귀. 모든 크기의 오류가 중요할 때.

## Huber 손실

Huber는 작은 오류는 MSE, 큰 오류는 MAE처럼 작동합니다:

`Huber = (1/m) * Σ L_δ(y - y_예측)` (L은 잔차 크기와 δ 임계값에 따라 결정)

**특징**: 혼합 접근법. 모든 곳에서 부드러움. 견고하면서도 작은 오류에 민감.

**장점**: 둘의 장점—이상치 견고성과 작은 오류 패널티.

**단점**: 튜닝할 하이퍼파라미터 δ.

**사용 사례**: 가끔 이상치 있는 회귀. 견고한 학습 선호.

## 이진 교차 엔트로피

이진 분류의 경우:

`BCE = -(1/m) * Σ[y*log(ŷ) + (1-y)*log(1-ŷ)]`

**특징**: 자신감 있는 잘못된 예측에 큰 패널티. 확률 기반 출력에 자연스러움.

**장점**: 확률론적 해석과 일치. 목표에서 멀 때 강한 기울기.

**단점**: 로그합-지수 트릭을 사용하지 않으면 수치적으로 불안정. ŷ=0 또는 1일 때 정의 안 됨.

**사용 사례**: 시그모이드 출력의 이진 분류.

## 다중클래스 교차 엔트로피

다중 분류의 경우:

`CCE = -(1/m) * Σ Σ y_k*log(ŷ_k)`

**특징**: BCE의 일반화. Softmax 출력과 함께 작동.

**장점**: 다중 분류 문제에 자연스러움. 표준 선택.

**단점**: 원-핫 인코딩된 레이블 필요. 수치적으로 민감.

**사용 사례**: 다중 분류. 대부분 분류 작업의 표준.

## Focal 손실

Focal 손실은 쉬운 예시에 낮은 가중치를 두어 클래스 불균형을 다룹니다:

`FL = -(1/m) * Σ (1 - ŷ_t)^γ * log(ŷ_t)`

(γ는 포커싱 파라미터)

**특징**: 곱셈식 손실 가중치를 통한 어려운 예시 채굴. 적응형 강조.

**장점**: 심한 클래스 불균형 처리. 클래스 불균형 시 BCE보다 우수.

**단점**: 튜닝할 하이퍼파라미터 γ. 더 복잡.

**사용 사례**: 불균형 데이터셋. 배경이 지배적인 객체 탐지.

## Hinge 손실

Hinge 손실은 SVM과 마진 기반 학습의 표준입니다:

`Hinge = (1/m) * Σ max(0, 1 - y_참 * y_예측)`

**특징**: 마진 기반. 마진을 넘어 올바르게 분류된 예시는 손실이 0.

**장점**: 자연스러운 마진 해석. 희소 해.

**단점**: 마진에서 매끄럽지 않음. 직접적으로 확률론적이지 않음.

**사용 사례**: SVM 유사 문제. 명시적 마진으로 학습.

## 대조 손실

대조 손실은 예시 쌍을 비교하여 임베딩을 학습합니다:

`Contrastive = (1/m) * Σ [y*d² + (1-y)*max(0, m-d)²]`

(d는 거리, y는 쌍의 유사성 표시)

**특징**: 유사도 메트릭 학습. 메트릭 학습에 유용.

**장점**: 해석 가능한 유사성. 다양한 작업에 유연.

**단점**: 쌍 데이터 필요. 계산 비용 높음.

**사용 사례**: 샴 네트워크. 얼굴 인식. 유사성 학습.

## 삼중(Triplet) 손실

삼중 손실은 앵커, 양수, 음수 예시를 비교합니다:

`Triplet = max(0, d(a,p) - d(a,n) + margin)`

**특징**: 유사하고 다른 예시 간 분리를 장려.

**장점**: 대조 손실보다 표현력 있음. 순위 매김에 더 좋음.

**단점**: 삼중 채굴 필요. 계산 집약적.

**사용 사례**: 대규모 메트릭 학습. 얼굴 검증. 개인 재식별.

## CTC 손실 (연결주의 시간 분류)

CTC 손실은 정렬이 알려지지 않은 시퀀스-대-시퀀스 문제를 다룹니다:

`CTC = -log(Σ_경로 P(경로 | 입력))`

**특징**: 가능한 모든 정렬에 대해 주변화.

**장점**: 정렬된 레이블 불필요. 가변 길이 시퀀스 처리.

**단점**: 구현 복잡. 정렬된 손실보다 느림.

**사용 사례**: 음성 인식. OCR. 레이블 정렬이 알려지지 않은 작업.

## 커스텀 손실

영역별 특수 목표를 위해 커스텀 손실을 설계할 수 있습니다:

- **Dice 손실**: 클래스 불균형이 있는 의미 분할에 유용
- **IoU 손실**: 객체 탐지의 평가 메트릭과 일치
- **지각 손실**: 사전 훈련된 네트워크를 사용하여 의미론적 유사성 측정
- **대립 손실**: GAN에서 판별자 대 생성기 측정

**설계 원칙**: 가능할 때 손실을 평가 메트릭과 일치시킵니다. 기울기가 정보성이 있는지 확인합니다.

## 손실 선택 가이드

**회귀**:
- 기본값: MSE
- 이상치 있을 때: Huber 또는 MAE
- 커스텀 범위: 조정된 MSE/MAE

**분류**:
- 이진: 이진 교차 엔트로피
- 다중클래스: 다중클래스 교차 엔트로피
- 불균형: Focal 손실
- 메트릭 학습: 대조 또는 삼중 손실

**시퀀스 작업**:
- 정렬 미알려: CTC 손실
- 단어 수준: 교차 엔트로피
- 커스텀 정렬: CTC 변형

## 요약

손실 함수는 모델이 최적화하는 목표입니다. 올바른 손실 선택은 아키텍처 선택만큼 중요합니다. 현대 실무는 종종 손실 조합(다중 작업 학습)과 신중한 가중치 조정을 포함합니다. 손실 특성(부드러움, 민감성, 계산 비용) 이해는 효과적인 모델 훈련에 필수적입니다.

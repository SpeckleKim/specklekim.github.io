# 서포트 벡터 머신

## 소개

서포트 벡터 머신(SVM)은 분류와 회귀를 위한 강력한 [지도 학습](/supervised-learning-kor.html) 알고리즘입니다. 클래스 간 마진을 최대화하는 최적의 초평면을 찾으므로 이진 및 다중 클래스 문제에서 특히 효과적입니다. SVM은 커널 방법을 통해 선형 및 비선형 분리 가능 데이터를 모두 처리합니다.

## 선형 SVM

### 마진 개념

SVM의 기본 아이디어는 **최대 마진 원칙**입니다: 최적의 결정 경계는 초평면과 각 클래스의 가장 가까운 데이터 포인트 간 거리(마진)를 최대화하는 것입니다.

**수학적 공식:**
```
최대화: 마진 = 2 / ||w||
제약조건: y_i(w·x_i + b) ≥ 1 (모든 i에 대해)
```

여기서 w는 가중치 벡터, b는 편향, y_i ∈ {-1, +1}은 클래스 레이블입니다. 제약조건은 최대 마진으로 올바른 분류를 보장합니다.

### 서포트 벡터

서포트 벡터는 결정 경계에 가장 가까운 훈련 포인트입니다. 초평면을 정의하는 데 이 포인트들만 중요하며, 다른 모든 포인트는 해를 변경하지 않고 제거할 수 있습니다. 이 성질이 SVM을 효율적이고 강건하게 만듭니다.

**핵심 통찰:**
```
f(x) = sign(Σ α_i × y_i × (x_i · x) + b)
```

여기서 α_i는 서포트 벡터에만 0이 아닙니다.

## 소프트 마진

실제 데이터는 완벽하게 분리 가능한 경우가 드뭅니다. 소프트 마진 공식화는 노이즈와 중첩된 클래스를 처리하기 위해 일부 오분류를 허용합니다.

**목적 함수:**
```
최소화: (1/2)||w||² + C × Σ ξ_i
```

여기서 ξ_i는 위반을 허용하는 슬랙 변수, C는 [정규화](/regularization-kor.html) 파라미터:
- 큰 C: 오분류를 심하게 처벌(오류 허용도 낮음)
- 작은 C: 더 많은 위반 허용(오류 허용도 높음)

## 커널 트릭

커널 트릭은 고차원 피처 변환을 명시적으로 계산하지 않고도 비선형 결정 경계를 처리하게 합니다.

### 커널 함수

**선형 커널:**
```
K(x_i, x_j) = x_i · x_j
```
간단하고 빠름. 선형 분리 가능 데이터에 최적.

**다항식 커널:**
```
K(x_i, x_j) = (x_i · x_j + c)^d
```
다항식 결정 경계를 생성. 파라미터 d가 다항식 차수를 제어합니다.

**방사 기저 함수(RBF) 커널:**
```
K(x_i, x_j) = exp(-γ ||x_i - x_j||²)
```
가장 다목적이며 복잡한 비선형 패턴을 처리합니다. 파라미터 γ는 영향 범위를 제어합니다.

**시그모이드 커널:**
```
K(x_i, x_j) = tanh(k × (x_i · x_j) + c)
```
[신경망](/neural-kor.html)과 유사한 동작.

### 커널의 동작 원리

커널은 선형 분리가 가능해지는 고차원 공간으로 데이터를 암묵적으로 매핑하여 비용이 높은 명시적 피처 계산을 피합니다. 이것이 "커널 트릭"입니다—좌표 없이 유사성을 계산합니다.

## 분류를 위한 SVM

### 이진 분류

SVM은 자연스럽게 이진 분류를 처리합니다. 초평면은 마진을 최대화하면서 두 클래스를 분리합니다. 결정 함수는 경계로부터의 거리를 제공합니다(신뢰도 점수에 유용).

### 다중 클래스 분류

3개 이상 클래스가 있는 문제의 경우, 일반적인 전략:
- **일대다(OvR)**: k개 클래스에 대해 k개 분류기 훈련
- **일대일(OvO)**: k(k-1)/2개 쌍별 분류기 훈련
- **구조화된 SVM**: 직접 다중 클래스 공식화

## 회귀를 위한 SVM

서포트 벡터 회귀(SVR)는 SVM을 연속 출력으로 확장합니다. 클래스 간 마진을 최대화하는 대신, SVR은 지정된 폭 ε의 예측 주변 튜브에 적합시킵니다.

**목적:**
```
최소화: (1/2)||w||² + C × Σ max(0, |y_i - (w·x_i + b)| - ε)
```

ε-둔감 손실은 튜브 내의 작은 오류를 무시하여 이상치에 대한 민감도를 줄입니다.

## 장점

- **고차원에서 효과적**: 많은 피처를 가진 문제에 좋음
- **메모리 효율적**: 결정이 서포트 벡터에만 의존
- **다목적**: 분류, 회귀, 순위 매김 처리
- **강력한 이론적 토대**: 전역 최적값으로 수렴하는 [볼록 최적화](/convex-optimization-kor.html)
- **이상치에 강건**: 소프트 마진과 서포트 벡터 메커니즘
- **비선형 능력**: 커널 트릭이 복잡한 관계 처리
- **유일한 해**: 국소 최솟값 없이 전역 최적값 발견

## 단점

- **계산 복잡도**: n개 샘플에 대해 O(n²) 또는 O(n³) 훈련 시간
- **파라미터 튜닝**: C와 커널 파라미터 신중한 선택 필요
- **확장성**: 매우 큰 데이터셋에 도전적
- **블랙박스**: 피처 기여도 해석 어려움
- **클래스 불균형**: 클래스 가중치 조정 필요할 수 있음
- **확률 추정**: SVM에 자연스럽지 않음, 후처리 필요
- **커널 선택**: 적절한 커널 선택이 도메인 지식 필요

## 다른 방법과의 비교

**vs. 로지스틱 회귀:**
- SVM은 비선형, 고차원 문제에 더 좋음
- 로지스틱 회귀가 더 해석 가능하고 빠름

**vs. 신경망:**
- SVM은 소규모-중규모 데이터셋에 더 좋음
- [신경망](/neural-kor.html)이 더 잘 확장되고 유연함
- SVM은 국소 최솟값에 덜 취약

**vs. 랜덤 포레스트:**
- SVM은 고차원에서 더 좋음
- [랜덤 포레스트](/decision-trees-kor.html)가 더 해석 가능하고 혼합 피처 처리
- 테이블 데이터에서 비교 가능한 정확도

## 실무 팁

**전처리:**
- 피처 표준화(SVM에 중요)
- 미리 결측값 처리

**파라미터 튜닝:**
- C와 γ에 대한 그리드 탐색 또는 무작위 탐색
- 과적합 방지를 위한 [교차 검증](/cross-validation-kor.html)

**클래스 불균형:**
- 클래스 가중치 조정: `class_weight = 'balanced'`
- 적절한 샘플링 전략 사용

**커널 선택:**
- RBF로 시작(가장 다목적)
- 다항식처럼 보이는 경계에 다항식 시도
- 해석 가능성과 속도를 위해 선형 사용

## 요약

서포트 벡터 머신은 여전히 가장 우아하고 이론적으로 기반이 탄탄한 머신러닝 알고리즘 중 하나입니다. 커널을 통한 비선형 처리 능력과 강력한 일반화 보장이 분류와 회귀에서 귀중합니다. 대규모 데이터셋에서 계산 집약적이지만, SVM은 복잡한 결정 경계를 가진 중규모 문제에서 탁월합니다.

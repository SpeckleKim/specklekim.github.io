# 정책 경사법: 정책 직접 학습

## 소개

정책 경사법은 예상 반환의 기울기를 계산하여 정책을 직접 최적화합니다. Q값을 학습하는 가치 기반 방법과 달리 정책 경사법은 매개변수화된 정책을 학습합니다. 이 접근법은 연속 제어, 온 정책 학습, 이론적 성질에서 장점을 제공합니다.

## 정책 경사 정리

**핵심 결과:** 정책 매개변수에 대한 예상 반환의 기울기:

```
∇_θ J(θ) = E[∇_θ log π_θ(a|s) Q^π(s,a)]
```

이 우아한 결과는 로그 확률과 Q값을 사용하여 정책 기울기를 계산하는 방법을 보여줍니다. 모델 지식 없이 효율적인 정책 최적화를 가능하게 합니다.

**직관:** 높은 가치 행동의 확률을 증가시키고 낮은 가치 행동은 감소시킵니다.

## REINFORCE

가장 단순한 정책 경사 알고리즘:

**알고리즘:**
```
각 에피소드에 대해:
  궤적 τ = (s_0,a_0,r_0,...,s_T,a_T) 샘플
  반환 계산 G_t = Σ_{k≥t} γ^{k-t} r_k
  업데이트: θ ← θ + α ∇_θ log π_θ(a_t|s_t) G_t
```

**특성:**
- 온 정책: 현재 정책에서 샘플해야 함
- Monte Carlo 반환: 전체 궤적 필요
- 높은 분산: 반환이 매우 변함
- 국소 최적값으로의 수렴이 보장됨

**장점:**
- 단순한 구현
- 연속 및 이산 행동으로 작동
- 이론적으로 견실함

**단점:**
- 높은 분산이 많은 샘플 필요
- 느린 수렴
- 신중하지 않은 학습 속도로 발산 가능

## 기준선 함수

**문제:** 원시 반환 G_t를 비평자로 사용하면 분산이 높습니다.

**해결책:** 기준선 b(s)를 반환에서 빼기:

```
∇_θ J(θ) = E[∇_θ log π_θ(a|s) (Q^π(s,a) - b(s))]
```

기준선은 예상 기울기에 영향을 주지 않지만 분산을 줄입니다. 최적 기준선은 가치 함수 V^π(s)입니다.

**장점:**
- 분산을 극적으로 감소시킴
- 불편 기울기 유지
- 동일한 수렴 보장
- 단순한 구현

## 근접 정책 최적화 (PPO)

현대 최첨단 정책 경사 방법:

**핵심 혁신:** 잘린 목표는 과도한 정책 변화를 방지:

```
L(θ) = E[min(r_t(θ)Â_t, clip(r_t(θ),1-ε,1+ε)Â_t)]
```

여기서 r_t(θ) = π_θ(a|s) / π_old(a|s)는 확률 비율, Â_t는 우위입니다.

**절단이 중요한 이유:**
절단 없이 기울기는 큰 확률 비율 점프를 권장합니다. 절단은 비율을 [1-ε, 1+ε]로 제한하여 업데이트당 정책 변화를 제한합니다.

**실무 장점:**
- 안정적이고 신뢰할 수 있는 학습
- 다양한 도메인에서 작동
- 하이퍼매개변수에 강건
- REINFORCE 대비 표본 효율적

**PPO 의사 코드:**
```
각 에포크에 대해:
  π_old를 사용한 궤적 수집
  우위와 반환 계산
  각 미니배치에 대해:
    K 에포크 동안 절단 목표 최적화
```

## 신뢰 영역 정책 최적화 (TRPO)

정책 변화를 제약하는 원칙적 접근:

**목표:** KL 발산을 제약하면서 예상 반환 최대화:

```
maximize E[r_t(θ)Â_t]
subject to E[KL(π_old || π_θ)] ≤ δ
```

**자연 기울기:** Fisher 정보 행렬을 통합합니다. 정책 매개변수화에 불변 업데이트.

**PPO와의 연결:** PPO는 명시적 KL 제약이 아닌 더 간단한 절단 목표로 TRPO를 근사합니다.

## 우위 추정 (GAE)

일반화된 우위 추정이 편향-분산 트레이드오프를 균형있게 유지:

```
Â_t = Σ_{l=0}^∞ (γλ)^l δ_t^{V}
```

여기서 δ_t^{V} = r_t + γV(s_{t+1}) - V(s_t)는 TD 잔차입니다.

**λ 매개변수:**
- λ = 0: 부트스트랩된 가치 사용 (낮은 분산, 높은 편향)
- λ = 1: 전체 궤적 사용 (높은 분산, 낮은 편향)
- λ ∈ (0,1): 가중 결합

GAE는 유연한 편향-분산 제어를 가능하게 하여 실무 성능을 개선합니다.

## 행위자 전용 정책 경사

기준선 (가치 함수) 없음:

```
∇_θ J(θ) = E[∇_θ log π_θ(a|s) r_t]
```

- 매우 높은 분산
- 느린 수렴
- 실무에서 일반적으로 회피

## 연속 vs 이산 행동

**이산 행동:**
- 정책이 행동 확률 분포 출력
- 범주형 분포에 자연스러움

**연속 행동:**
- 정책이 가우시안의 평균과 분산 출력: π_θ(a|s) = N(μ_θ(s), σ_θ(s))
- 부드러운 제어 가능

## 실무 고려사항

**하이퍼매개변수 민감성:**
- 학습 속도 중요; 수렴 속도와 안정성에 영향
- 정책 업데이트 단계 수
- 우위 추정 람다 (GAE)
- 절단 입실론 (PPO)

**구현 세부사항:**
- 우위 정규화: 안정성과 일반화 개선
- 엔트로피 보너스: 탐험을 권장하고 조기 수렴 방지
- 가치 함수 손실: (V_θ(s) - target)² 최소화

## 한계

- 온 정책 학습: 환경과 계속 상호작용해야 함
- 상관 궤적 샘플: 에피소드 내 샘플은 의존적
- 탐험 도전: 국소 최적값에 갇힐 수 있음
- 표본 비효율성: 일반적으로 가치 기반 방법보다 낮은 표본 효율성

## 현대 확장

- **분산 PPO:** 여러 워커 전역 학습
- **비동기 방법:** 병렬 행위자 사용 A3C
- **다중 작업 학습:** 여러 작업 정책 학습
- **교육 학습:** 점진적 작업 난이도
- **메타 학습:** 정책 빠른 적응 학습

## 결론

정책 경사법은 기울기 상승을 사용하여 정책을 직접 최적화합니다. REINFORCE가 접근법을 개척했고 기준선이 분산을 줄였습니다. PPO는 정책 절단을 통해 실무적이고 안정적인 학습을 도입했습니다. TRPO는 이론적 기초를 제공했습니다. 이 방법들은 연속 제어의 표준이 되었으며 최신 접근법에도 불구하고 경쟁력을 유지합니다. 단순성, 유연성, 이론적 기반이 계속된 두드러짐을 설명합니다.
